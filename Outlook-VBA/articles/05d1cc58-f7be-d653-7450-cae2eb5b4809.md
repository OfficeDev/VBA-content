
# OlkInfoBar.Click Event (Outlook)

 **Last modified:** July 28, 2015

Occurs when the control is clicked, either through user action or programmatically.

## Syntax

 _expression_. **Click**

 _expression_A variable that represents an  **OlkInfoBar** object.
