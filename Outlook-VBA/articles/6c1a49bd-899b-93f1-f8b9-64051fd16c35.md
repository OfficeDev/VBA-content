
# SharingItem.ReplyRecipients Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  ** [Recipients](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)**collection that represents all the reply recipient objects for the  ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**. Read-only.

## Syntax

 _expression_. **ReplyRecipients**

 _expression_A variable that represents a  **SharingItem** object.
