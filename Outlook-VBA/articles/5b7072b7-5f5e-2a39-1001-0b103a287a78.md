
# ViewFields.Item Method (Outlook)

 **Last modified:** July 28, 2015

Returns a  ** [ViewField](997319f0-7ff3-a712-8484-2e442965e187.md)** object from the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **ViewFields** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The value can be a one-based integer that indexes an  **ViewField** object in the ** [ViewFields](2516faed-ed11-6cb3-ce9c-b6afa788e909.md)** collection, a string that matches the ** [ViewXMLSchemaName](69490353-b470-6092-0b8e-b0f1c1549f7a.md)** property value of an **ViewField** object in the collection, or a field name as displayed in the **Field Chooser**.|

### Return Value

A  **ViewField** object that represents the specified object.
