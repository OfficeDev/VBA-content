
# OlkTextBox.DoubleClick Event (Outlook)

 **Last modified:** July 28, 2015

Occurs when the user double-clicks inside the control.

## Syntax

 _expression_. **DoubleClick**

 _expression_A variable that represents an  **OlkTextBox** object.
