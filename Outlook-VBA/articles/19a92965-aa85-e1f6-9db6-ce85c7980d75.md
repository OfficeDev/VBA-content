
# CalendarView.MonthShowEndTime Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Boolean** value that indicates whether end times for Outlook items are displayed in the ** [CalendarView](37e078b9-9fc6-5894-b043-06d7257666a8.md)** object. Read/write.

## Syntax

 _expression_. **MonthShowEndTime**

 _expression_A variable that represents a  **CalendarView** object.


## Remarks

The value of this property is applicable only when the  ** [CalendarViewMode](144e46ed-984f-fac0-fad3-0ff5ac9f2996.md)** property of the **CalendarView** object is set to **olCalendarViewMonth**.
