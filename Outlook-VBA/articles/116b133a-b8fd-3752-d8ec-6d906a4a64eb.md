
# SharingItem.Body Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** representing the clear-text body of the ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**. Read/write.

## Syntax

 _expression_. **Body**

 _expression_A variable that represents a  **SharingItem** object.
