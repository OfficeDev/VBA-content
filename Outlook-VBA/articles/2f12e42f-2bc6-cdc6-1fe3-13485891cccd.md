
# SharingItem.ReceivedByName Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** representing the display name of the true recipient for the ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**. Read-only.

## Syntax

 _expression_. **ReceivedByName**

 _expression_A variable that represents a  **SharingItem** object.


## Remarks

This property corresponds to the MAPI property  **PidTagReceivedByName**.
