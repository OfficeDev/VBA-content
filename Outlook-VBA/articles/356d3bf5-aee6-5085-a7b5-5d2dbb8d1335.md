
# OlkListBox.MatchEntry Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets an  ** [olMatchEntry](b4c8aa72-747a-df06-4b92-5f54461164a3.md)** constant that indicates how the control searches its list as the user types. Read/write.

## Syntax

 _expression_. **MatchEntry**

 _expression_A variable that represents an  **OlkListBox** object.


## Remarks

The default value is  **olMatchEntryComplete**.
