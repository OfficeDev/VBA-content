
# NavigationGroup.GroupType Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [OlGroupType](2a5ee820-41fa-91fc-2ce0-46d97fc4bf11.md)** constant that indicates the group type for the ** [NavigationGroup](a96eb2b1-af1f-71b2-6a0b-dcb5078beb1f.md)** object. Read-only.

## Syntax

 _expression_. **GroupType**

 _expression_A variable that represents a  **NavigationGroup** object.
