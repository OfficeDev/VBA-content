
# TaskRequestItem.BeforeAutoSave Event (Outlook)

 **Last modified:** July 28, 2015

Occurs before the item is automatically saved by Outlook.

## Syntax

 _expression_. **BeforeAutoSave**( **_Cancel_**)

 _expression_A variable that represents a  **TaskRequestItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Cancel|Required| **Boolean**|Set to  **True** to cancel the operation; otherwise, set to **False** to allow the ** [TaskRequestItem](2908a28a-634c-e786-aa53-f3e32038b727.md)** to be saved.|
