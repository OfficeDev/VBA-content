
# OutlookBarGroups.Item Method (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [OutlookBarGroup](4ccc4213-5a57-7a8b-4ce5-869a096bd096.md)** object from the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents an  **OutlookBarGroups** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|Either the index number of the object, or a value used to match the default property of an object in the collection.|

### Return Value

An  **OutlookBarGroup** object that represents the specified object.
