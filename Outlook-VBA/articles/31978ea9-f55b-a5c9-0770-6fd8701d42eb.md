
# OrderFields.Count Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **Long** value indicating the count of ** [OrderField](4ae32270-bde9-3178-bca3-f8d145779d3d.md)** objects in the specified collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents an  **OrderFields** object.
