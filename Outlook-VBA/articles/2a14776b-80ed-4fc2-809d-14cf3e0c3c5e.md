
# UserProperties Properties (Outlook)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](cf34337d-7087-7a71-e13b-9f97beb605ca.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](be47a8e7-a5cb-2b9b-6fec-2e1090329f6b.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [Count](6e316d8a-68b5-f25a-c3d2-4d72a054b027.md)|Returns a  **Long** indicating the count of objects in the specified collection. Read-only.|
| [Parent](35c01dd0-bec0-ece8-59fd-80daf1989e98.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](0cd76318-80c6-4cfc-3aca-32e385ff6b88.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
