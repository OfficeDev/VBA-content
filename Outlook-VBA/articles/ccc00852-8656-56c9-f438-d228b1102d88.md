
# SendRuleAction Members (Outlook)
Represents an action that sends a message to one or more recipients.

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [ActionType](07b46194-32b4-f04f-d18e-d4b7f3db8f07.md)|Returns a constant,  **olRuleActionRedirect**, from the  ** [OlRuleActionType](d6a39ac2-00e7-73e7-3890-ea658211eae9.md)** enumeration that indicates the type of action that is taken by the rule. Read-only.|
| [Application](1587a84f-77e4-765f-9e30-fe8c3ef23798.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](9bcbaf99-6c4e-b706-299a-33750d55dba1.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [Enabled](c046cb54-b275-b903-2f9c-dc9a106cdc8a.md)|Returns or sets a  **Boolean** that determines if the rule action is enabled. Read/write.|
| [Parent](67c8a668-95b2-c70a-ae14-2d01363e5b2d.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Recipients](5a4a9a2f-4e3c-a4b3-e29c-99c6e8e7a061.md)|Returns a  ** [Recipients](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)**collection that represents the recipient list for the send action. Read-only.|
| [Session](0d0b9289-0381-fe88-d4e7-1d0197ce6d6b.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
