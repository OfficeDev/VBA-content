
# CalendarView.StartField Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** value that represents the name of the property that starts the time duration for Outlook items displayed in the ** [CalendarView](37e078b9-9fc6-5894-b043-06d7257666a8.md)** object. Read/write.

## Syntax

 _expression_. **StartField**

 _expression_A variable that represents a  **CalendarView** object.


## Remarks

The values of the  **StartField** and ** [EndField](311994db-ef43-e49c-6f0e-9b346d0bb3ca.md)** properties indicate which Outlook item properties are used by the **CalendarView** object to represent the duration of an Outlook item. Both custom and built-in properties can be specified, but only date/time properties are allowed.
