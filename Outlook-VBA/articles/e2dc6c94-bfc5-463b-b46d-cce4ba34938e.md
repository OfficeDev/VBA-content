
# Views Properties (Outlook)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](06912bad-7d59-ea9b-0491-463386474d99.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](2ffe97b5-8ccb-8d13-b35f-0cc0399fa719.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [Count](cbdd4b27-6383-0df9-2b8a-715fc40f1eb6.md)|Returns a  **Long** indicating the count of objects in the specified collection. Read-only.|
| [Parent](3ada0c2f-3719-f7a9-641c-7d0d7174ca37.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](677d7b97-b138-3506-7b45-26d091f9ba6e.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
