
# Row.Class Property (Outlook)

 **Last modified:** July 28, 2015

Returns a constant in the  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** enumeration indicating the class of the ** [Row](06db3fa4-1649-48bf-3b86-ffdf99a47305.md)** object. Read-only.

## Syntax

 _expression_. **Class**

 _expression_A variable that represents a  **Row** object.
