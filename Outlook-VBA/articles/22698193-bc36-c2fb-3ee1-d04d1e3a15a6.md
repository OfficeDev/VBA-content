
# TaskItem.ReminderPlaySound Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Boolean** value that is **True** if the reminder should play a sound when it occurs for this item. Read/write.

## Syntax

 _expression_. **ReminderPlaySound**

 _expression_A variable that represents a  **TaskItem** object.


## Remarks

The  **ReminderPlaySound** property must be set in order to validate the ** [ReminderSoundFile](29bfa689-08b6-f963-9ecb-3744b1032062.md)**property.
