
# OlkTimeZoneControl.Exit Event (Outlook)

 **Last modified:** July 28, 2015

Occurs just after the focus passes from this control to another control on the same form.

## Syntax

 _expression_. **Exit**( **_Cancel_**)

 _expression_A variable that represents an  **OlkTimeZoneControl** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Cancel|Required| **Boolean**| **False** when the event occurs. If the event procedure sets this argument to **True**, the exit operation is not completed and the focus remains on this control|
