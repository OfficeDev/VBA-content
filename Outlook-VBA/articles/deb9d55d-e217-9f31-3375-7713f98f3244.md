
# FormNameRuleCondition Members (Outlook)
Represents a rule condition that evaluates whether a form name was used to send or receive an item.

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](23642b0c-2f1e-8559-9dee-71219dae8194.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](75a66c33-a79a-b9f7-ddd6-b987e65cf0a2.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [ConditionType](54019220-07f4-e6b8-8690-f2af6dd307e0.md)|Returns a constant,  **olConditionFormName**, from the  ** [OlRuleConditionType](35c2f965-0f9d-8cc8-2f05-60522268574f.md)** enumeration that indicates the type of rule condition. Read-only.|
| [Enabled](ddf66e35-05d0-4bda-c204-018a5c4b716b.md)|Returns or sets a  **Boolean** that determines if the rule condition is enabled. Read/write.|
| [FormName](993f2ee0-58eb-bed0-5819-11148792b8f0.md)|Returns or sets an  **Object** that represents an array of form identifiers to be evaluated by the rule condition. Read/write.|
| [Parent](0a15d010-cf45-b3b1-59ab-4c6fabf7e9cf.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](ec224a2e-1d45-82d8-0336-9f1f36549b7a.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
