
# Store.StoreID Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** identifying the ** [Store](1eb22fe9-8849-7476-5388-2515b48591b9.md)**. Read-only. 

## Syntax

 _expression_. **StoreID**

 _expression_A variable that represents a  **Store** object.


## Remarks

 **StoreID** is unique to the profile. It is equivalent to the MAPI property **PidTagStoreEntryId**.
