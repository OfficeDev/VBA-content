
# ContactItem.ShowCheckPhoneDialog Method (Outlook)

 **Last modified:** July 28, 2015

Displays the  **Check Phone Number** dialog box for a specified telephone number contained by a ** [ContactItem](8e32093c-a678-f1fd-3f35-c2d8994d166f.md)** object.

## Syntax

 _expression_. **ShowCheckPhoneDialog**( **_PhoneNumber_**)

 _expression_An expression that returns a  **ContactItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|PhoneNumber|Required| ** [OlContactPhoneNumber](93be67bb-9b2d-05f9-edb5-1e9cf0061ccc.md)**|The type of telephone number to be checked.|
