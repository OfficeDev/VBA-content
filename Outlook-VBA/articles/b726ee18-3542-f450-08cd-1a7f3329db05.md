
# OlkTextBox.Scrollbars Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets an  ** [olScrollBars](22832728-5d27-ff6b-9ef2-ed6a8e1c9d5c.md)** constant that specifies whether the control has a vertical scroll bar, horizontal scroll bar, or both. Read/write.

## Syntax

 _expression_. **Scrollbars**

 _expression_A variable that represents an  **OlkTextBox** object.


## Remarks

The default value is  **olScrollBarsNone**.
