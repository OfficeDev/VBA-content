
# TasksModule Properties (Outlook)

## Properties



|**Name**|**Description**|

| [Class](2b6d9f8c-c189-c8a8-7c2e-a37ce150389b.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|

| [NavigationModuleType](6ca2b925-e526-08d2-8c87-fc3d041bec1a.md)|Returns an  ** [OlNavigationModuleType](2140a094-6bee-aba1-03cd-71fa2c55842e.md)** constant that represents the navigation module type for the object. Read-only.|

| [Session](947b6795-21db-e2fb-b76b-43dc90520403.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
| [Visible](d04827e6-7e12-7acd-24e0-b72c21082683.md)|Returns or sets a  **Boolean** value that determines whether the ** [TasksModule](fc6ae6c9-6b13-b5f2-9506-c3dbbe709df6.md)** object is displayed in the Navigation Pane. Read/write.|
