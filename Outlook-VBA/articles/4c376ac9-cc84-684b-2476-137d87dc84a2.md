
# OlkOptionButton.Caption Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** that appears on an object to identify or describe it. Read/write.

## Syntax

 _expression_. **Caption**

 _expression_A variable that represents an  **OlkOptionButton** object.


## Remarks

 The default value is based on the control's name. The position of the caption relative to the check box is determined by the ** [Alignment](0d6d5ac4-d084-ea7d-80e6-854315f94b90.md)** property.
