
# UserProperties.Item Method (Outlook)

 **Last modified:** July 28, 2015

Returns a  ** [UserProperty](c94f642f-4368-d775-a79f-ce6c39bfe1fd.md)** object from the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents an  **UserProperties** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|Either the one-based index number of the object, or a value used to match the default property of an object in the collection.|

### Return Value

A  **UserProperty** object that represents the specified object.
