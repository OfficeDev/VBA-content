
# TaskItem.Status Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets an  ** [OlTaskStatus](99ff6947-b53d-9cbc-cbe7-726de6ed93ec.md)** constant specifying the status for the task. Corresponds to the **Status** field of a ** [TaskItem](5df8cfa5-5460-a5a1-a130-ba5bca1a0091.md)**. Read/write.

## Syntax

 _expression_. **Status**

 _expression_A variable that represents a  **TaskItem** object.
