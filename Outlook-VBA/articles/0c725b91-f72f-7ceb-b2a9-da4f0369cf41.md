
# MailItem.BeforeAutoSave Event (Outlook)

 **Last modified:** July 28, 2015

Occurs before the item is automatically saved by Outlook.

## Syntax

 _expression_. **BeforeAutoSave**( **_Cancel_**, )

 _expression_A variable that represents a  **MailItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Cancel|Required| **Boolean**|Set to  **True** to cancel the operation; otherwise, set to **False** to allow the ** [MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** to be saved.|
