
# OutlookBarShortcuts Properties (Outlook)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](e9178112-f2f6-f1fd-e586-89906d3a0130.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|

| [Parent](3981b828-1286-2bbf-4edb-fd297f9540ce.md)|Returns the parent  **Object** of the specified object. Read-only.|
