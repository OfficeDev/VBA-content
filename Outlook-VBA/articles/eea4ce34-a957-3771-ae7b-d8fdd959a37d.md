
# ExchangeUser.Class Property (Outlook)

 **Last modified:** July 28, 2015

Returns a constant in the  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** enumeration indicating the class of the ** [ExchangeUser](6ec117d1-7fdb-aa36-b567-1242f8238df0.md)** object. Read-only.

## Syntax

 _expression_. **Class**

 _expression_A variable that represents an  **ExchangeUser** object.
