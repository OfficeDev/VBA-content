
# OlkTimeControl.Change Event (Outlook)

 **Last modified:** July 28, 2015

Occurs when the  ** [Value](e0cbbfd7-1cbd-2e28-f89e-b9081672a86d.md)** property changes.

## Syntax

 _expression_. **Change**

 _expression_A variable that represents an  **OlkTimeControl** object.
