
# Categories.Count Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **Long** value indicating the count of ** [Category](143ef095-54b0-cbe2-e356-632029061ac2.md)** objects in the specified collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **Categories** object.
