
# ContactItem.LastNameAndFirstName Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** representing the concatenated last name and first name for the contact. Read-only.

## Syntax

 _expression_. **LastNameAndFirstName**

 _expression_A variable that represents a  **ContactItem** object.


## Remarks

This property is parsed from the  ** [FirstName](403b5e5a-037b-cf21-efc2-2bd2a80c3789.md)**and  ** [LastName](430682f6-a230-887b-404b-a71989121fa2.md)**properties for the contact, which are themselves parsed from the  ** [FullName](3036dc57-31fb-45ad-f51e-49336206581d.md)**property.
