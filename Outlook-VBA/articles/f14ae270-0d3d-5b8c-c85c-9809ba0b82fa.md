
# ContactItem.AutoResolvedWinner Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **Boolean** that determines if the item is a winner of an automatic conflict resolution. Read-only.

## Syntax

 _expression_. **AutoResolvedWinner**

 _expression_A variable that represents a  **ContactItem** object.


## Remarks

A value of  **False** does not necessarily indicate that the item is a loser of an automatic conflict resolution. The item could be in conflict with another item.

If an item has  ** [Conflicts.Count](4a7445ff-8628-50d6-f4c0-ada85f3b3f5c.md)** of its ** [ContactItem.Conflicts](c51d7028-40d5-4d67-7bc6-8715bfa89f24.md)** property greater than zero and if its **AutoResolvedWinner** property is **True**, it is a winner of an automatic conflict resolution. On the other hand, if the item is in conflict and has its  **AutoResolvedWinner** property as **False**, it is a loser in an automatic conflict resolution.
