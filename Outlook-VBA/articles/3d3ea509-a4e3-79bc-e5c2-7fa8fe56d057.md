
# SharingItem.SenderEmailType Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** that represents the type of entry for the e-mail address of the sender of the ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**, such as 'SMTP' for Internet address, 'EX' for a Microsoft Exchange server address, and so on. Read-only.

## Syntax

 _expression_. **SenderEmailType**

 _expression_A variable that represents a  **SharingItem** object.
