
# OlkTimeControl.Text Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** that is the text displayed in the control. Read/write.

## Syntax

 _expression_. **Text**

 _expression_A variable that represents an  **OlkTimeControl** object.


## Remarks

The default value of the  **Text** property is the empty string.

The  **Text** property provides the same value as the ** [Value](e0cbbfd7-1cbd-2e28-f89e-b9081672a86d.md)** property, but the value is strongly typed as a **String**.
