
# DistListItem.ReminderOverrideDefault Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Boolean** value that is **True** if the reminder overrides the default reminder behavior for the item. Read/write.

## Syntax

 _expression_. **ReminderOverrideDefault**

 _expression_A variable that represents a  **DistListItem** object.


## Remarks

You must set the  **ReminderOverrideDefault** property to validate the ** [ReminderPlaySound](449c0c37-b0f7-ff42-1ba4-6eabd990e4db.md)**and the  ** [ReminderSoundFile](3d0d0543-97f5-cecd-a838-c36b5c35b8aa.md)**properties.
