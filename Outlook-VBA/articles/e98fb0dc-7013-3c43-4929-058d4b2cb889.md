
# SharingItem.Actions Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [Actions](b0903aa4-9b75-5311-d0a5-5ff4a5e29c79.md)**collection that represents all the available actions for the  ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**. Read-only.

## Syntax

 _expression_. **Actions**

 _expression_A variable that represents a  **SharingItem** object.
