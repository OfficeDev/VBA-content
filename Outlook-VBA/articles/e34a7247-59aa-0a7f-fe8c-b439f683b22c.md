
# CalendarModule.Visible Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Boolean** value that determines whether the ** [CalendarModule](9203024d-9cef-75e0-600f-f3899e24761a.md)** object is displayed in the Navigation Pane. Read/write.

## Syntax

 _expression_. **Visible**

 _expression_A variable that represents a  **CalendarModule** object.
