
# ContactItem.IsMarkedAsTask Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **Boolean** value that indicates whether the ** [ContactItem](8e32093c-a678-f1fd-3f35-c2d8994d166f.md)** is marked as a task. Read-only.

## Syntax

 _expression_. **IsMarkedAsTask**

 _expression_An expression that returns a  **ContactItem** object.
