
# PostItem.ReminderOverrideDefault Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Boolean** value that is **True** if the reminder overrides the default reminder behavior for the item. Read/write.

## Syntax

 _expression_. **ReminderOverrideDefault**

 _expression_A variable that represents a  **PostItem** object.


## Remarks

You must set the  **ReminderOverrideDefault** property to validate the ** [ReminderPlaySound](9e441d70-4478-80fa-cdfa-c71ebec8e2ec.md)**and the  ** [ReminderSoundFile](9292a962-e7f9-75e0-20a0-716daf7d677f.md)**properties.
