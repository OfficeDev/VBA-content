
# AppointmentItem.GetOrganizer Method (Outlook)

 **Last modified:** July 28, 2015

Obtains the  ** [AddressEntry](d4a0a85e-8bab-bc56-57bc-d70c3c570c8e.md)** object that contains information from the Address Book about the organizer of the ** [AppointmentItem](204a409d-654e-27aa-643a-8344c631b82d.md)**.

## Syntax

 _expression_. **GetOrganizer**

 _expression_A variable that represents an  **AppointmentItem** object.


### Return Value

An  **AddressEntry** object that represents the organizer of the **AppointmentItem**.
