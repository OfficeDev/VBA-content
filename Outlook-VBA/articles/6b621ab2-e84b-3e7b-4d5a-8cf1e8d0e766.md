
# ContactsModule.NavigationModuleType Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [olNavigationModuleType](2140a094-6bee-aba1-03cd-71fa2c55842e.md)** constant that represents the navigation module type for the object. Read-only.

## Syntax

 _expression_. **NavigationModuleType**

 _expression_An expression that returns a  **ContactsModule** object.


## Remarks

This property returns  **olModuleContacts** for the ** [ContactsModule](fb183bd5-c72f-b38f-97e3-209a2a463d24.md)** object.
