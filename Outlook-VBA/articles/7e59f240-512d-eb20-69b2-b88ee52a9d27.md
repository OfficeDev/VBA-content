
# Account.AccountType Property (Outlook)

 **Last modified:** July 28, 2015

Returns a constant in the  ** [OlAccountType](8aeafc50-3f97-8d28-7fd9-a9d8e1eafc4c.md)** enumeration that indicates the type of the ** [Account](f624438c-4e45-2822-18b6-bfe8074a33c0.md)**. Read-only.

## Syntax

 _expression_. **AccountType**

 _expression_A variable that represents an  **Account** object.
