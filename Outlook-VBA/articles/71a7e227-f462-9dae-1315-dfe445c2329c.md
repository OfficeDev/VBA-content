
# ContactItem.Email1DisplayName Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** representing the display name of the first e-mail address for the contact. Read/write.

## Syntax

 _expression_. **Email1DisplayName**

 _expression_A variable that represents a  **ContactItem** object.


## Remarks

This property is set to the value of the  ** [FullName](3036dc57-31fb-45ad-f51e-49336206581d.md)**property by default.
