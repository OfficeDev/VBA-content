
# SharingItem.PropertyChange Event (Outlook)

 **Last modified:** July 28, 2015

Occurs when an explicit built-in property (for example,  ** [Subject](f774ec0f-226a-f2f8-6865-4457b26fdacc.md)**) of an instance of the parent object is changed. 

## Syntax

 _expression_. **PropertyChange**( **_Name_**)

 _expression_An expression that returns a  **SharingItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Name|Required| **String**|The name of the property that was changed.|

## Remarks

The property name is passed to the event so that you can determine which property was changed.
