
# Accounts.Count Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **Long** indicating the count of objects in the specified collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents an  **Accounts** object.
