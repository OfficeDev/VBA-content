
# AssignToCategoryRuleAction Members (Outlook)
Represents an action that assigns categories to a message.

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [ActionType](bef50a28-967e-7336-ef0b-2e8edb843c0d.md)|Returns a constant,  **olRuleActionAssignToCategory**, from the  ** [OlRuleActionType](d6a39ac2-00e7-73e7-3890-ea658211eae9.md)** enumeration that indicates the type of action that is taken by the rule. Read-only.|
| [Application](ab5707ec-911c-21e2-6f26-6a0347c2e39a.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Categories](92e849e3-4d5a-a11b-3c32-6214a15a90df.md)|Returns or sets an array of strings representing the categories assigned to the message. Read/write.|
| [Class](64b3b8d7-6cd9-267a-f933-0594455a7bb9.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [Enabled](c6f4558d-fb2a-b732-cfeb-a30f447f0989.md)|Returns or sets a  **Boolean** that determines if the rule action is enabled. Read/write.|
| [Parent](66428062-94de-795a-09b3-c4c9d57fe029.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](4ee91dde-9f5d-101f-f259-98192e45a76d.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
