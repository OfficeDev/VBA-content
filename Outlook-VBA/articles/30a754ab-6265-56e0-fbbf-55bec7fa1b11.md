
# AddressEntry.Name Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** value that represents the display name for the object. Read/write.

## Syntax

 _expression_. **Name**

 _expression_A variable that represents an  **AddressEntry** object.


## Remarks

It is necessary for the  **Name** property to be set before calling the ** [AddressEntry.Details](85457da6-c97a-387d-6c7e-40eb005b25aa.md)**method.
