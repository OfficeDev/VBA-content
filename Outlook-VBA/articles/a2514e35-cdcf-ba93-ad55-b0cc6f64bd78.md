
# TaskItem.Close Event (Outlook)

 **Last modified:** July 28, 2015

Occurs when the inspector associated with an item (which is an instance of the parent object) is being closed.

## Syntax

 _expression_. **Close**( **_Cancel_**)

 _expression_A variable that represents a  **TaskItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Cancel|Required| **Boolean**|(Not used in VBScript).  **False** when the event occurs. If the event procedure sets this argument to **True**, the close operation is not completed and the inspector is left open.|

## Remarks

In Microsoft Visual Basic Scripting Edition (VBScript), if you set the return value of this function to  **False**, the close operation isn't completed and the inspector is left open. 

If you use the  ** [Close](7682f0c8-d132-2bd6-94e8-6e45fcc00867.md)** method to fire this event, it can only be canceled if the **Close** method uses the **olPromptForSave** argument.
