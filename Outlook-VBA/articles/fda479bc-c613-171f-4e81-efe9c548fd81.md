
# OlkDateControl.Text Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** that is the text displayed in the control. Read/write.

## Syntax

 _expression_. **Text**

 _expression_A variable that represents an  **OlkDateControl** object.


## Remarks

The default value of the  **Text** property is the empty string.

The  **Text** property provides the same value as the ** [Value](df2c96d4-42d4-fd33-a55b-2162f65069b7.md)** property, but the value is strongly typed as a **String**.
