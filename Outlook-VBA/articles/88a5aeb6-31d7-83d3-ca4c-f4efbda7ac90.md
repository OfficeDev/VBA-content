
# OlkTimeControl.DropDown Method (Outlook)

 **Last modified:** July 28, 2015

Expands the time drop-down list of the time control.

## Syntax

 _expression_. **DropDown**

 _expression_A variable that represents an  **OlkTimeControl** object.
