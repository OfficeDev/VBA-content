
# NavigationFolder.Class Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.

## Syntax

 _expression_. **Class**

 _expression_A variable that represents a  **NavigationFolder** object.


## Remarks

This property returns  **olNavigationFolder** for the ** [NavigationFolder](c8d7aabb-58ba-df5e-ccdc-06f73db7726c.md)** object.
