
# Table.Parent Property (Outlook)

 **Last modified:** July 28, 2015

Returns the parent  **Object** of the ** [Table](0affaafd-93fe-227a-acee-e09a86cadc20.md)** object. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **Table** object.
