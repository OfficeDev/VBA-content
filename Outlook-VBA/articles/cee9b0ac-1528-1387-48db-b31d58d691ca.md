
# PostItem.SenderName Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** indicating the display name of the sender for the Outlook item. Read-only.

## Syntax

 _expression_. **SenderName**

 _expression_A variable that represents a  **PostItem** object.


## Remarks

This property corresponds to the MAPI property  **PidTagSenderName**.

If you wish to retrieve the fully qualified e-mail address of the sender, use the  ** [SenderEmailAddress](a84bd603-7fce-25ae-668e-083e11e9937e.md)** property.
