
# ImportanceRuleCondition Members (Outlook)
Represents a rule condition that evaluates the importance of a message.

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](b671ee51-9b28-e408-02fe-bf9fa0da7dfb.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](888893ab-a032-3426-9675-29ca01356500.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [ConditionType](c4fd234b-7a7a-dfb6-9b09-2840f254d713.md)|Returns a constant,  **olConditionImportance**, from the  ** [OlRuleConditionType](35c2f965-0f9d-8cc8-2f05-60522268574f.md)** enumeration that indicates the type of rule condition. Read-only.|
| [Enabled](a082587d-d191-1446-6f8b-8604bf9372f5.md)|Returns or sets a  **Boolean** that determines if the rule condition is enabled. Read/write.|
| [Importance](8755d0c6-0016-c682-d746-5fcb48fa4d1a.md)|Returns or sets an  ** [OlImportance](71e04f9a-fab6-153f-b046-11f7ec50e8e4.md)** constant indicating the relative level of importance for the message. Read/write.|
| [Parent](792062c2-9d60-9958-a1e0-dae39d180ba3.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](521d650f-8724-e8cb-6d20-1e7d730bf419.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
