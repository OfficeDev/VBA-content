
# MeetingItem.Size Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **Long** indicating the size (in bytes) of the Outlook item. Read-only.

## Syntax

 _expression_. **Size**

 _expression_A variable that represents a  **MeetingItem** object.
