
# DistListItem.IsMarkedAsTask Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **Boolean** value that indicates whether the ** [DistListItem](027c3986-abff-d9b1-ecc2-26d60805e952.md)** is marked as a task. Read-only.

## Syntax

 _expression_. **IsMarkedAsTask**

 _expression_An expression that returns a  **DistListItem** object.
