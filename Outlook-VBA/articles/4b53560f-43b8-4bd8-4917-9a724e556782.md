
# OlkTimeZoneControl.AppointmentTimeField Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets an  ** [OlAppointmentTimeField](630706ba-c98a-e6de-73ce-c7071e0d1db2.md)** constant that specifies the time field on the appointment that the control binds against. Read/write.

## Syntax

 _expression_. **AppointmentTimeField**

 _expression_A variable that represents an  **OlkTimeZoneControl** object.


## Remarks

This control does not support standard data binding.
