
# AccountSelector.Class Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant that indicates the class of the object. Read-only.

## Syntax

 _expression_. **Class**

 _expression_A variable that represents an  ** [AccountSelector](846f176e-5680-a214-7624-75f3a524c989.md)** object.
