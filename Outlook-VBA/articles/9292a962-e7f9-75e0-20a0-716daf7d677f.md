
# PostItem.ReminderSoundFile Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** indicating the path and file name of the sound file to play when the reminder occurs for the Outlook item. Read/write.

## Syntax

 _expression_. **ReminderSoundFile**

 _expression_A variable that represents a  **PostItem** object.


## Remarks

This property is only valid if the  ** [ReminderOverrideDefault](ee8e8003-7aac-563d-992e-f3cd09901318.md)**and  ** [ReminderPlaySound](9e441d70-4478-80fa-cdfa-c71ebec8e2ec.md)**properties are set to  **True**.
