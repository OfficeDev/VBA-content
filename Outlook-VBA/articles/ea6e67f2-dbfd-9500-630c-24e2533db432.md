
# SharingItem.Forward Method (Outlook)

 **Last modified:** July 28, 2015

Executes the  **Forward** action for an item and returns the resulting copy as a ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** object.

## Syntax

 _expression_. **Forward**

 _expression_A variable that represents a  **SharingItem** object.


### Return Value

A  **SharingItem** object that represents the new sharing invitation.


## Remarks

An error occurs if this method is called on sharing invitations using an Exchange sharing context or on sharing requests.
