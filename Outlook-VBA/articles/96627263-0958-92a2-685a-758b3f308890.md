
# NotesModule.Name Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** value that represents the localized display name for the ** [NotesModule](cdbdde08-0773-a78d-3809-a3811975bcc1.md)** object. Read-only.

## Syntax

 _expression_. **Name**

 _expression_A variable that represents a  **NotesModule** object.
