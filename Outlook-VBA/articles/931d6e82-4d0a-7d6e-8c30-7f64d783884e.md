
# ContactItem.FullNameAndCompany Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** representing the full name and company of the contact by concatenating the values of the ** [FullName](3036dc57-31fb-45ad-f51e-49336206581d.md)**and  ** [CompanyName](076cd6f7-7faa-ab1c-254c-3307c40520ee.md)**properties. Read-only.

## Syntax

 _expression_. **FullNameAndCompany**

 _expression_A variable that represents a  **ContactItem** object.
