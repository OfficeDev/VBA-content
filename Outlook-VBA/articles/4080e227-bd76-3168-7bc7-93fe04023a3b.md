
# NameSpace.DefaultStore Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  ** [Store](1eb22fe9-8849-7476-5388-2515b48591b9.md)** object representing the default Store for the profile. Read-only.

## Syntax

 _expression_. **DefaultStore**

 _expression_A variable that represents a  **NameSpace** object.


## Remarks

Returns an error if there is no default store for the current profile.
