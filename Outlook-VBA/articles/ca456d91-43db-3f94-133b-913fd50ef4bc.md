
# MeetingItem.Forward Method (Outlook)

 **Last modified:** July 28, 2015

Executes the  **Forward** action for an item and returns the resulting copy as a ** [MeetingItem](b75730f5-b395-3d66-5acd-b64fd8fcd78f.md)** object.

## Syntax

 _expression_. **Forward**

 _expression_A variable that represents a  **MeetingItem** object.


### Return Value

A  **MeetingItem** object that represents the new meeting item.
