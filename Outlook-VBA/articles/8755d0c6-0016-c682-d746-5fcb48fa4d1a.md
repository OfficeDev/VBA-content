
# ImportanceRuleCondition.Importance Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets an  ** [OlImportance](71e04f9a-fab6-153f-b046-11f7ec50e8e4.md)** constant indicating the relative level of importance for the message. Read/write.

## Syntax

 _expression_. **Importance**

 _expression_A variable that represents an  **ImportanceRuleCondition** object.


## Remarks

This property corresponds to the MAPI property  **PidTagImportance**.
