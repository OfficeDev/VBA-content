
# AutoFormatRules.Save Method (Outlook)

 **Last modified:** July 28, 2015

Save all changes to  ** [AutoFormatRule](6d295c41-17f9-8e67-4595-4330fd3cec99.md)** objects within the ** [AutoFormatRules](74514b71-964c-f17b-4df6-e1a5c5ed2b52.md)** collection.

## Syntax

 _expression_. **Save**

 _expression_A variable that represents an  **AutoFormatRules** object.


## Remarks

Any changes made to the  **AutoFormatRules** collection, such as adding or removing **AutoFormatRule** objects, or to an **AutoFormatRule** object within the collection, are persisted only if this method is called.
