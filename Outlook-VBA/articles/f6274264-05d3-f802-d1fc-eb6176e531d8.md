
# SharingItem.Sensitivity Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets an  ** [OlSensitivity](611d23ca-40ee-17e9-2560-99c5508f6e29.md)** constant indicating the sensitivity for the ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**. Read/write.

## Syntax

 _expression_. **Sensitivity**

 _expression_A variable that represents a  **SharingItem** object.


## Remarks

This property corresponds to the MAPI property  **PidTagSensitivity**.
