
# OutlookBarStorage.Parent Property (Outlook)

 **Last modified:** July 28, 2015

Returns the parent  **Object** of the specified object. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents an  **OutlookBarStorage** object.
