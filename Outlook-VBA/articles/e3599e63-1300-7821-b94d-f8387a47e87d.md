
# AppointmentItem.ReminderSoundFile Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** indicating the path and file name of the sound file to play when the reminder occurs for the Outlook item. Read/write.

## Syntax

 _expression_. **ReminderSoundFile**

 _expression_A variable that represents an  **AppointmentItem** object.


## Remarks

This property is only valid if the  ** [ReminderOverrideDefault](08c20608-6065-1e4a-8c89-8aa35c682c77.md)**and  ** [ReminderPlaySound](4020684b-c89d-7371-75e0-4f3dfe01bec3.md)**properties are set to  **True**.
