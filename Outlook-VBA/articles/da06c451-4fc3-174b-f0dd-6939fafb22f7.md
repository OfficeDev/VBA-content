
# OlkOptionButton.Change Event (Outlook)

 **Last modified:** July 28, 2015

Occurs when the  ** [Value](5199d3f8-c90c-a66a-0487-78ec7ee66bcc.md)** property changes.

## Syntax

 _expression_. **Change**

 _expression_A variable that represents an  **OlkOptionButton** object.
