
# PeopleView.Reset Method (Outlook)
Resets this built-in Outlook view to its original settings.

 **Last modified:** July 28, 2015


## Version information

Version Added: Outlook 2013 


## Syntax

 _expression_. **Reset**

 _expression_A variable that represents a  **PeopleView** object.
