
# MeetingItem.SenderName Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** indicating the display name of the sender for the Outlook item. Read-only.

## Syntax

 _expression_. **SenderName**

 _expression_A variable that represents a  **MeetingItem** object.


## Remarks

This property corresponds to the MAPI property  **PidTagSenderName**.

If you wish to retrieve the fully qualified e-mail address of the sender, use the  ** [SenderEmailAddress](b318c074-4897-d99d-2b7c-870b4ab083e9.md)** property.
