
# MailItem.ReminderOverrideDefault Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Boolean** value that is **True** if the reminder overrides the default reminder behavior for the item. Read/write.

## Syntax

 _expression_. **ReminderOverrideDefault**

 _expression_A variable that represents a  **MailItem** object.


## Remarks

You must set the  **ReminderOverrideDefault** property to validate the ** [ReminderPlaySound](7fd10182-445f-2aa6-db9f-2534d66fe0ea.md)**and the  ** [ReminderSoundFile](11c5ae79-1ce0-5890-1ba1-5a39a88ecc6b.md)**properties.
