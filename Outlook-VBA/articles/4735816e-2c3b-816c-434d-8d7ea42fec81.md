
# AppointmentItem.StartInStartTimeZone Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Date** value that represents the start date and time of the appointment expressed in the ** [AppointmentItem.StartTimeZone](3259fa91-5f6c-b899-9bfc-2ac669911271.md)**. Read/write.

## Syntax

 _expression_. **StartInStartTimeZone**

 _expression_A variable that represents an  **AppointmentItem** object.


## Remarks

This is the value displayed as  **Start time** in the appointment inspector user interface.
