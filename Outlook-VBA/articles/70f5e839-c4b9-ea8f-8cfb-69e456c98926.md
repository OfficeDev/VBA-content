
# PlaySoundRuleAction Members (Outlook)
Represents an action that plays a .wav file sound.

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [ActionType](f3b2ec1d-9b8b-64b8-cc02-9d1aec8fd764.md)|Returns a constant,  **olRuleActionPlaySound**, from the  ** [OlRuleActionType](d6a39ac2-00e7-73e7-3890-ea658211eae9.md)** enumeration that indicates the type of action that is taken by the rule. Read-only.|
| [Application](36f34b4e-e9da-1c48-012b-0664536189a8.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](74c4ac63-fcf9-12dc-c910-e41b80b417f2.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [Enabled](7a8b222e-a9db-f38f-8f8b-a834ff46c39a.md)|Returns or sets a  **Boolean** that determines if the rule action is enabled. Read/write.|
| [FilePath](8f27c644-aab4-4e9e-b563-0db7242cd86f.md)|Returns or sets a  **String** that represents the full file path to a sound file (.wav). Read/write.|
| [Parent](cac44be6-e790-6863-4462-cf9b1382bbb0.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](8d3e9f6e-848d-9879-61a8-7662858674d4.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
