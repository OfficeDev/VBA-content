
# CategoryRuleCondition Properties (Outlook)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](4db7d71e-c1b3-7320-183a-17efbbf115ff.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Categories](7662a095-43e9-7668-f6f7-d0701b87b28c.md)|Returns or sets an array of  **String** elements that represents the categories evaluated by the rule condition. Read/write.|
| [Class](49742e81-83d6-afad-21bc-a22e501a1359.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [ConditionType](155e8920-479e-fed7-833f-5aaf3eb12d2a.md)|Returns a constant,  **olConditionCategory**, from the  ** [OlRuleConditionType](35c2f965-0f9d-8cc8-2f05-60522268574f.md)** enumeration that indicates the type of rule condition. Read-only.|
| [Enabled](027949cf-d5a9-b6a8-3edf-ae00cb97d6e6.md)|Returns or sets a  **Boolean** that determines if the rule condition is enabled. Read/write.|
| [Parent](bcfb86b5-7904-6cc0-044b-da3ba4e551bd.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](ee8824ce-0cc8-1e32-1878-721f5e7a81be.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
