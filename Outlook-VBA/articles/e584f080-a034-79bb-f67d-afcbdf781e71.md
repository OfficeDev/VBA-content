
# JournalModule.Class Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.

## Syntax

 _expression_. **Class**

 _expression_A variable that represents a  **JournalModule** object.


## Remarks

This property returns  **olJournalModule** for the ** [JournalModule](5a696d10-8a10-c01d-cf65-f8a65718f120.md)** object.
