
# OlkControl.VerticalLayout Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets an  ** [OlVerticalLayout](4b94eee9-65a7-c2ad-323d-f321a6a716af.md)** constant that specifies any alignment in the vertical direction of this control with other selected controls on the form, or whether the control is adjusted in the vertical direction automatically when the form's length is changed. Read/write.

## Syntax

 _expression_. **VerticalLayout**

 _expression_A variable that represents an  **OlkControl** object.
