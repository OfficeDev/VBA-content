
# JournalItem.Reply Method (Outlook)

 **Last modified:** July 28, 2015

Creates a reply, pre-addressed to the original sender, from the original message.

## Syntax

 _expression_. **Reply**

 _expression_A variable that represents a  **JournalItem** object.


### Return Value

A  ** [MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** object that represents the reply.
