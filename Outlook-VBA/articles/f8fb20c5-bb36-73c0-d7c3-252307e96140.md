
# Reminder.Item Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  **Object** corresponding to the specified Outlook item. Read-only.

## Syntax

 _expression_. **Item**

 _expression_A variable that represents a  **Reminder** object.
