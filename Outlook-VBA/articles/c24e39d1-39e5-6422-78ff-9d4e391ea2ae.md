
# AppointmentItem.BeforeAutoSave Event (Outlook)

 **Last modified:** July 28, 2015

Occurs before the item is automatically saved by Outlook.

## Syntax

 _expression_. **BeforeAutoSave**( **_Cancel_**, )

 _expression_A variable that represents an  **AppointmentItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Cancel|Required| **Boolean**|Set to  **True** to cancel the operation; otherwise, set to **False** to allow the ** [AppointmentItem](204a409d-654e-27aa-643a-8344c631b82d.md)** to be saved.|
