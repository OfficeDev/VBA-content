
# NavigationFolder.DisplayName Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** representing the display name of the ** [NavigationFolder](c8d7aabb-58ba-df5e-ccdc-06f73db7726c.md)** object. Read-only.

## Syntax

 _expression_. **DisplayName**

 _expression_A variable that represents a  **NavigationFolder** object.


## Remarks

Use  **NavigationFolder.DisplayName**, instead of  ** [Folder.Name](ec03a345-8c06-f234-e1e9-ecdc54495ed2.md)**, to get the name of a shared folder.
