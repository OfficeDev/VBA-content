
# SharingItem.Attachments Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [Attachments](4cc96a5f-a822-8ad5-6f61-e996bee8ba22.md)**object that represents all the attachments for the specified  ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**. Read-only.

## Syntax

 _expression_. **Attachments**

 _expression_A variable that represents a  **SharingItem** object.
