
# OlkListBox.Value Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Variant** that represents the content selected in the list displayed by the control. Read/write.

## Syntax

 _expression_. **Value**

 _expression_A variable that represents an  **OlkListBox** object.


## Remarks

The value is the same as the value of  ** [Text](5bd07fce-bf12-eec5-b9d9-6d888296c30d.md)**, except that this property is  **Variant** and **Text** is strongly typed as **String**.
