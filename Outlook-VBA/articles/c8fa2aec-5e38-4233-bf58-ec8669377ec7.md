
# Views.Item Method (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [View](41c8d149-9912-1685-4c8b-3c849cc6f1ed.md)** object from the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **Views** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|Either the index number of the object, or a value used to match the default property of an object in the collection.|

### Return Value

A  **View** object that represents the specified object.
