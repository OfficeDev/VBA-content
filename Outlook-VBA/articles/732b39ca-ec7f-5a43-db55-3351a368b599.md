
# TableView.MultiLine Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets an  ** [OlMultiLine](9af600cc-7a46-64d2-7e78-958a8aafbe66.md)** constant that determines how multiple lines are displayed in the ** [TableView](026e27f8-1655-060d-e8cc-87eaaf4f1510.md)** object. Read/write.

## Syntax

 _expression_. **Multiline**

 _expression_A variable that represents a  **TableView** object.


## Remarks

If the value of the  ** [AutomaticColumnSizing](b599fec1-a6a4-fc12-0d95-bae9491c40d1.md)** property is set to **False** or if the value of the ** [AllowInCellEditing](dc6fa249-405f-6262-b085-53da15ba3d72.md)** property is set to **True**, the value of this property is automatically set to  **olAlwaysSingleLine**.
