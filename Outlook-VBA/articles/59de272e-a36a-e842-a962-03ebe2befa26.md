
# MeetingItem.BeforeAutoSave Event (Outlook)

 **Last modified:** July 28, 2015

Occurs before the item is automatically saved by Outlook.

## Syntax

 _expression_. **BeforeAutoSave**( **_Cancel_**)

 _expression_A variable that represents a  **MeetingItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Cancel|Required| **Boolean**|Set to  **True** to cancel the operation; otherwise, set to **False** to allow the ** [MeetingItem](b75730f5-b395-3d66-5acd-b64fd8fcd78f.md)** to be saved.|
