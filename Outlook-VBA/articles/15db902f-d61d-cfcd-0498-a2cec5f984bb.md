
# SharingItem.Send Event (Outlook)

 **Last modified:** July 28, 2015

Occurs when the user selects the  **Send** action for an item, or when the ** [Send](54f92175-0e99-f96a-56de-5fc66d97d80f.md)** method is called for the item, which is an instance of the parent object.

## Syntax

 _expression_. **Send**( **_Cancel_**)

 _expression_An expression that returns a  **SharingItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Cancel|Required| **Boolean**|(Not used in VBScript).  **False** when the event occurs. If the event procedure sets this argument to **True**, the send operation is not completed and the inspector is left open.|

## Remarks

In Microsoft Visual Basic Scripting Edition (VBScript), if you set the return value of this function to  **False**, the item is not sent.
