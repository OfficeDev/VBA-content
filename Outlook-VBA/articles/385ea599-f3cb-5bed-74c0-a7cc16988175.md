
# OlkComboBox.Text Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** that is the text displayed in the control. Read/write.

## Syntax

 _expression_. **Text**

 _expression_A variable that represents an  **OlkComboBox** object.


## Remarks

The default value of the  **Text** property is the empty string.

The  **Text** property provides the same value as the ** [Value](742dd2a3-d3ef-46f9-4aca-5ebe8af17356.md)** property, but the value is strongly typed as a **String**.
