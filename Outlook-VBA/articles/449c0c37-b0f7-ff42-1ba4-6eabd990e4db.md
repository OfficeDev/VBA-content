
# DistListItem.ReminderPlaySound Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Boolean** value that is **True** if the reminder should play a sound when it occurs for this item. Read/write.

## Syntax

 _expression_. **ReminderPlaySound**

 _expression_A variable that represents a  **DistListItem** object.


## Remarks

The  **ReminderPlaySound** property must be set in order to validate the ** [ReminderSoundFile](3d0d0543-97f5-cecd-a838-c36b5c35b8aa.md)**property.
