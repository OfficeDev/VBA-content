
# ContactItem.ReminderSoundFile Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** indicating the path and file name of the sound file to play when the reminder occurs for the Outlook item. Read/write.

## Syntax

 _expression_. **ReminderSoundFile**

 _expression_A variable that represents a  **ContactItem** object.


## Remarks

This property is only valid if the  ** [ReminderOverrideDefault](08e77dff-b325-c565-746a-e47e4d66ed77.md)**and  ** [ReminderPlaySound](a9941154-6c65-57c7-1dab-6d6a59620d92.md)**properties are set to  **True**.
