
# MailModule Properties (Outlook)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](60a85a6b-67bc-c90a-a4a8-8f484d6cb0d6.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](f991b831-21a2-45c4-40ac-075eb5c90604.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [Name](6bb6f48d-c3cb-57dc-076c-67b1a2883926.md)|Returns a  **String** value that represents the localized display name for the ** [MailModule](df20efe5-be5c-952d-c6b7-20c20a83fda0.md)** object. Read-only.|
| [NavigationGroups](23d7891a-41a0-679b-4654-0bfdcf9df2b4.md)|Returns a  ** [NavigationGroups](07206203-36a9-7467-3a89-24fa2a7c2b1f.md)** object that contains the set of navigation groups associated with the ** [MailModule](df20efe5-be5c-952d-c6b7-20c20a83fda0.md)** object. Read-only.|
| [NavigationModuleType](d73c9fba-b768-4a40-2a18-9128e226b42d.md)|Returns an  ** [olNavigationModuleType](2140a094-6bee-aba1-03cd-71fa2c55842e.md)** constant that represents the navigation module type for the object. Read-only.|
| [Parent](9507f813-87f0-3a15-5340-311e0f9374af.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Position](46cfff8e-07ac-f929-94be-c7b39980daa8.md)|Returns or sets a  **Long** value that represents the ordinal position of the ** [MailModule](df20efe5-be5c-952d-c6b7-20c20a83fda0.md)** object when it is displayed in the Navigation Pane. Read/write.|
| [Session](6b4405e4-c7b8-9837-a494-889e2e17d7ef.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
| [Visible](6e1d07fe-1551-e5db-13b6-76d07c965a99.md)|Returns or sets a  **Boolean** value that determines whether the ** [MailModule](df20efe5-be5c-952d-c6b7-20c20a83fda0.md)** object is displayed in the Navigation Pane. Read/write.|
