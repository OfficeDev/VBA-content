
# AddressLists Properties (Outlook)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](fe332bc6-efc1-c286-8aed-064ed47b3720.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](5e2a57fd-fa31-099a-457b-69259ae280c3.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [Count](aca2191f-0b86-2d4a-96b8-f5d7d61c03d3.md)|Returns a  **Long** indicating the count of objects in the specified collection. Read-only.|
| [Parent](725c5835-2b57-b9ac-452d-1a0ca7dc3752.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](60b4307f-92c7-abed-5bc7-2a190cddd4ca.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
