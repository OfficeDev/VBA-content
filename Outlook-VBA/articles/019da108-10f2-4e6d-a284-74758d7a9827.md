
# OutlookBarStorage Properties (Outlook)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](ad9a37cb-de9c-662e-1dcd-2197ebd239d2.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](d4f7819c-8d8f-23a2-829e-117cad81cb74.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [Groups](9b324d3d-3ab6-1e24-962f-19812b6b8ed0.md)|Returns an  ** [OutlookBarGroups](bb5fef46-b15a-51c3-0adf-f94e9da6c921.md)** object representing the set of groups in the **Shortcuts** pane. Read-only.|
| [Parent](d8f82919-4a68-4092-f37b-87d63b0dd57a.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](f3ba6302-aca2-f8ba-3a82-ae35f6b5b609.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
