
# TaskItem.DelegationState Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [OlTaskDelegationState](c6374297-d9ec-dbd0-f6b3-5b8c95127f33.md)** constant indicating the delegation state of the task. Read-only.

## Syntax

 _expression_. **DelegationState**

 _expression_A variable that represents a  **TaskItem** object.
