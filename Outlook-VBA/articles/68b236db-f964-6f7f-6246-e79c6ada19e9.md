
# NameSpace.AddressLists Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [AddressLists](b8c5ce75-3030-0179-45bb-f44fe6628074.md)**collection representing a collection of the address lists available for this session. Read-only.

## Syntax

 _expression_. **AddressLists**

 _expression_A variable that represents a  **NameSpace** object.


## Remarks

The  **AddressLists** collection represents the root of the address book hierarchy for the current session. A particular ** [AddressList](84611afe-48b1-185b-df4b-0f004e7436ff.md)**object represents one of the available address books. The type of access you obtain depends on the access permissions granted to you by each individual address book provider.
