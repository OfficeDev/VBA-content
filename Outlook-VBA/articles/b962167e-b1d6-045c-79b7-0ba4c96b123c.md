
# RuleConditions.Body Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  ** [TextRuleCondition](87e9ca00-7577-02c2-fb6f-a5dc2054ad8b.md)** object with a ** [TextRuleCondition.ConditionType](2dbc7979-deae-fbb8-9def-8c906657024a.md)** of **olConditionBody**. Read-only.

## Syntax

 _expression_. **Body**

 _expression_A variable that represents a  **RuleConditions** object.


## Remarks

Use the returned  **TextRuleCondition** object when enumerating the rule conditions or exception conditions of an existing rule, or when creating a new rule that specifies the condition or exception condition that certain text is in the message body.

This property of the  ** [RuleConditions](e8e9a05a-b36b-add2-b294-8cdc5a97e119.md)** collection always returns a **TextRuleCondition** object regardless of whether the rule associated with this **RuleConditions** collection has defined such a rule condition. If the rule has defined and enabled such a rule condition, then ** [TextRuleCondition.Enabled](7027c22b-08fa-d1b0-f664-8c4a26722cbb.md)** will be **True**.
