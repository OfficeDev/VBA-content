
# TaskItem.ReminderSoundFile Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** indicating the path and file name of the sound file to play when the reminder occurs for the Outlook item. Read/write.

## Syntax

 _expression_. **ReminderSoundFile**

 _expression_A variable that represents a  **TaskItem** object.


## Remarks

This property is only valid if the  ** [ReminderOverrideDefault](3a11ee36-3418-422e-0783-e39bf92ded6f.md)**and  ** [ReminderPlaySound](22698193-bc36-c2fb-3ee1-d04d1e3a15a6.md)**properties are set to  **True**.
