
# MailModule.NavigationModuleType Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [olNavigationModuleType](2140a094-6bee-aba1-03cd-71fa2c55842e.md)** constant that represents the navigation module type for the object. Read-only.

## Syntax

 _expression_. **NavigationModuleType**

 _expression_An expression that returns a  **MailModule** object.


## Remarks

This property returns  **olModuleMail** for the ** [MailModule](df20efe5-be5c-952d-c6b7-20c20a83fda0.md)** object.
