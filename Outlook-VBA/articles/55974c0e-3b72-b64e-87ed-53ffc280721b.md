
# SharingItem.ReminderSoundFile Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** indicating the path and file name of the sound file to play when the reminder occurs for the Outlook item. Read/write.

## Syntax

 _expression_. **ReminderSoundFile**

 _expression_A variable that represents a  **SharingItem** object.


## Remarks

This property is only valid if the  ** [ReminderOverrideDefault](811f7f85-5ddd-26ec-a654-e98eca51c946.md)**and  ** [ReminderPlaySound](c7588cea-a416-8eb1-8f5e-e9c4cc5ad84a.md)**properties are set to  **true**.
