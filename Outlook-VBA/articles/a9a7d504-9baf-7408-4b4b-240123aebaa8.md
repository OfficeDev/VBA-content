
# SharingItem.To Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a semicolon-delimited  **String** list of display names for the **To** recipients for the ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**. Read/write.

## Syntax

 _expression_. **To**

 _expression_A variable that represents a  **SharingItem** object.


## Remarks

This property contains the display names only. The  **To** property corresponds to the MAPI property **PidTagDisplayTo**. The  ** [Recipients](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)**collection should be used to modify this property.
