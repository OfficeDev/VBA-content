
# SharingItem.ReplyAll Method (Outlook)

 **Last modified:** July 28, 2015

Creates a reply to all original recipients from the original  ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**.

## Syntax

 _expression_. **ReplyAll**

 _expression_A variable that represents a  **SharingItem** object.


### Return Value

A  ** [MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** object that represents the reply.
