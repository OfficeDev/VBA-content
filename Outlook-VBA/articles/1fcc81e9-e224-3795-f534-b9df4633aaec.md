
# IconView.SortFields Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [OrderFields](e115fb80-352d-fd2e-c1c3-d266776fe122.md)** object that represents the set of fields by which the items displayed in the ** [IconView](dc2efa6c-4752-f713-f77e-378036f358dc.md)** object are ordered. Read-only.

## Syntax

 _expression_. **SortFields**

 _expression_A variable that represents an  **IconView** object.
