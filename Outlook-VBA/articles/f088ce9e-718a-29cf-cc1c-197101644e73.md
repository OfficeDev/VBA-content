
# SelectNamesDialog.Parent Property (Outlook)

 **Last modified:** July 28, 2015

Returns the parent object of the  ** [SelectNamesDialog](1522736a-3cad-9f1c-4da9-b52a3a01731c.md)** object. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **SelectNamesDialog** object.
