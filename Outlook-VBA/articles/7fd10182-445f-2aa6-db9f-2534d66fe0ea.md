
# MailItem.ReminderPlaySound Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Boolean** value that is **True** if the reminder should play a sound when it occurs for this item. Read/write.

## Syntax

 _expression_. **ReminderPlaySound**

 _expression_A variable that represents a  **MailItem** object.


## Remarks

The  **ReminderPlaySound** property must be set in order to validate the ** [ReminderSoundFile](11c5ae79-1ce0-5890-1ba1-5a39a88ecc6b.md)**property.
