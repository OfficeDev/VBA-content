
# MailItem.ReminderSoundFile Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** indicating the path and file name of the sound file to play when the reminder occurs for the Outlook item. Read/write.

## Syntax

 _expression_. **ReminderSoundFile**

 _expression_A variable that represents a  **MailItem** object.


## Remarks

This property is only valid if the  ** [ReminderOverrideDefault](78aaca38-6de7-9bc1-6539-74d7b03bfd54.md)**and  ** [ReminderPlaySound](7fd10182-445f-2aa6-db9f-2534d66fe0ea.md)**properties are set to  **True**.
