
# OutlookBarShortcut Members (Outlook)
Represents a shortcut in a group in the  **Shortcuts** pane.

 **Last modified:** July 28, 2015


## Methods



|**Name**|**Description**|
|:-----|:-----|
| [SetIcon](d54a60b5-e667-e030-0724-d61be3ad3745.md)|Sets the icon for the specified shortcut on the  **Shortcuts** pane.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](7ad7628e-4435-654a-7162-34b7989e32b5.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](23360c23-9ca0-566c-cf63-2cb2bd2eaa20.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [Name](403a1755-ca83-b6e6-db95-55dc12d05ec5.md)|Returns or sets a  **String** value that represents the display name for the object. Read/write.|
| [Parent](cddbca1d-70ee-1b9c-718d-6f4df4ca5c94.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](aee32453-1650-1d28-10ae-a125fa4c4394.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
| [Target](990671c0-bfc5-6b09-26a1-1cdf9d0e143b.md)|Returns a  **Variant** indicating the target of the specified shortcut in a **Shortcuts** pane group. Read-only.|
