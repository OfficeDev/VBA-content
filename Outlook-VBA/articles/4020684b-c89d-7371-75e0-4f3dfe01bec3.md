
# AppointmentItem.ReminderPlaySound Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Boolean** value that is **True** if the reminder should play a sound when it occurs for this item. Read/write.

## Syntax

 _expression_. **ReminderPlaySound**

 _expression_A variable that represents an  **AppointmentItem** object.


## Remarks

The  **ReminderPlaySound** property must be set in order to validate the ** [ReminderSoundFile](e3599e63-1300-7821-b94d-f8387a47e87d.md)**property.
