
# AppointmentItem.RecurrenceState Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [OlRecurrenceState](4b9be4ca-3094-8f0f-6406-978635795aaa.md)** constant indicating the recurrence property of the specified object. Read-only.

## Syntax

 _expression_. **RecurrenceState**

 _expression_A variable that represents an  **AppointmentItem** object.
