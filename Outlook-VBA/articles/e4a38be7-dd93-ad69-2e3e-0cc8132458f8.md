
# FormRegion.Inspector Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [Inspector](d7384756-669c-0549-1032-c3b864187994.md)** object representing the Inspector that hosts the form region. Read-only.

## Syntax

 _expression_. **Inspector**

 _expression_A variable that represents a  **FormRegion** object.


## Remarks

Returns  **Null** ( **Nothing** in Visual Basic) if the form region is displayed in the Reading Pane.
