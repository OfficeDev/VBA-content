
# BusinessCardView.SortFields Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [OrderFields](e115fb80-352d-fd2e-c1c3-d266776fe122.md)** collection object that represents all the fields by which the view is sorted. Read-only.

## Syntax

 _expression_. **SortFields**

 _expression_An expression that returns a  **BusinessCardView** object.
