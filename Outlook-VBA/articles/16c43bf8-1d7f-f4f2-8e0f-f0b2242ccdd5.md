
# JournalItem.Duration Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Long** indicating the duration (in minutes) of the ** [JournalItem](6e850295-39f9-47b8-e866-9622e9958c69.md)** Read/write.

## Syntax

 _expression_. **Duration**

 _expression_A variable that represents a  **JournalItem** object.
