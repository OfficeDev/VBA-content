
# DistListItem.TaskStartDate Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Date** value that represents the start date of the task for this ** [DistListItem](027c3986-abff-d9b1-ecc2-26d60805e952.md)** object. Read/write.

## Syntax

 _expression_. **TaskStartDate**

 _expression_An expression that returns a  **DistListItem** object.


## Remarks

This property returns  **Null** ( **Nothing** in Visual Basic) if the ** [IsMarkedAsTask](6dc846e0-c6a6-2151-13e7-f305772b5ae8.md)** property is set to **False**.
