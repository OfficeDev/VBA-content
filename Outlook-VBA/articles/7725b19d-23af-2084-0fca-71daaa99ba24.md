
# SharingItem.SenderName Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** indicating the display name of the sender for the ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**. Read-only.

## Syntax

 _expression_. **SenderName**

 _expression_A variable that represents a  **SharingItem** object.


## Remarks

This property corresponds to the MAPI property  **PidTagSenderName**.

If you wish to retrieve the fully qualified e-mail address of the sender, use the  ** [SenderEmailAddress](9cbde5cf-cd9a-b35d-eee9-4ed8b617f444.md)** property.
