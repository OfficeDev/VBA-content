
# OlkTimeZoneControl.SelectedTimeZoneIndex Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets an index into the  ** [Application.TimeZones](920e55d1-9914-fa74-101a-921083328d23.md)** collection that determines the selected time zone. Read/write.

## Syntax

 _expression_. **SelectedTimeZoneIndex**

 _expression_A variable that represents an  **OlkTimeZoneControl** object.


## Remarks

 **SelectedTimeZoneIndex** is 1-based, and a value of zero (0) means that no time zone has been selected.
