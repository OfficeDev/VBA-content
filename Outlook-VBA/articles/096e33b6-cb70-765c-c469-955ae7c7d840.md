
# AppointmentItem.PropertyAccessor Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  ** [PropertyAccessor](2fc91e13-703c-3ec9-9066-ffee7144306c.md)** object that supports creating, getting, setting, and deleting properties of the parent ** [AppointmentItem](204a409d-654e-27aa-643a-8344c631b82d.md)** object. Read-only.

## Syntax

 _expression_. **PropertyAccessor**

 _expression_A variable that represents an  **AppointmentItem** object.
