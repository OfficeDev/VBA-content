
# SyncObject.Name Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** value that represents the display name for the object. Read-only.

## Syntax

 _expression_. **Name**

 _expression_A variable that represents a  **SyncObject** object.
