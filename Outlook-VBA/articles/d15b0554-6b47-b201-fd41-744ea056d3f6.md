
# AddressRuleCondition Members (Outlook)
Represents a rule condition that evaluates whether the address for the recipient or sender of the message is contained in the address specified in  ** [AddressRuleCondition.Address](de4186ec-0741-8ff6-7789-af0a46c470e0.md)**.

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Address](de4186ec-0741-8ff6-7789-af0a46c470e0.md)|Returns or sets an array of  **String** elements to evaluate the address rule condition. Read/write.|
| [Application](bc908e8a-83eb-03e7-5b98-9dc0918a67a6.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](566eb9a5-2b7a-1833-f803-60a750fda257.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [ConditionType](8b531745-1a4d-d903-5c7d-465b9fd8cbf3.md)|Returns a constant from the  ** [OlRuleConditionType](35c2f965-0f9d-8cc8-2f05-60522268574f.md)** enumeration that indicates the type of rule condition. Read-only.|
| [Enabled](170cd84c-4733-0801-c411-34736e2e1a06.md)|Returns or sets a  **Boolean** that determines if the rule condition is enabled. Read/write.|
| [Parent](8943ab05-a3c7-6ee2-c2c1-f97315a08ac0.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](c5134be6-7ce4-dc65-8bde-9c725ef3ba8c.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
