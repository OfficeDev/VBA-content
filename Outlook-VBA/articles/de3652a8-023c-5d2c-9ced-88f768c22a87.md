
# ExchangeUser.Type Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **String** representing the type of entry for the ** [ExchangeUser](6ec117d1-7fdb-aa36-b567-1242f8238df0.md)**. Read/write.

## Syntax

 _expression_. **Type**

 _expression_A variable that represents an  **ExchangeUser** object.


## Remarks

The  **ExchangeUser** object is derived from the ** [AddressEntry](d4a0a85e-8bab-bc56-57bc-d70c3c570c8e.md)** object. It inherits the **Type** property from the **AddressEntry** object. In the case of **ExchangeUser**, the type is always the string "EX".
