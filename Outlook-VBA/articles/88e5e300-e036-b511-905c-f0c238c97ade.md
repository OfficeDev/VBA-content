
# TaskItem.Read Event (Outlook)

 **Last modified:** July 28, 2015

Occurs when an instance of the parent object is opened for editing by the user. 

## Syntax

 _expression_. **Read**

 _expression_A variable that represents a  **TaskItem** object.


## Remarks

The  **Read** event differs from the ** [Open](001d2598-58e1-86d9-b893-31a79ac2a0a0.md)** event in that **Read** occurs whenever the user selects the item in a view that supports in-cell editing as well as when the item is being opened in an ** [Inspector](d7384756-669c-0549-1032-c3b864187994.md)**.
