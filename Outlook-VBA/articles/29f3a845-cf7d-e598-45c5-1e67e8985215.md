
# AppointmentItem.Move Method (Outlook)

 **Last modified:** July 28, 2015

Moves a Microsoft Outlook item to a new folder.

## Syntax

 _expression_. **Move**( **_DestFldr_**)

 _expression_A variable that represents an  **AppointmentItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|DestFldr|Required| ** [Folder](3cf6cda8-6d70-666e-2643-9d9c5b9cacfc.md)**|The destination folder.|

### Return Value

An  **Object** value that represents the item which has been moved to the designated folder.
