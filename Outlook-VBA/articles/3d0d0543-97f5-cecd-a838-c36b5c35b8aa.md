
# DistListItem.ReminderSoundFile Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** indicating the path and file name of the sound file to play when the reminder occurs for the Outlook item. Read/write.

## Syntax

 _expression_. **ReminderSoundFile**

 _expression_A variable that represents a  **DistListItem** object.


## Remarks

This property is only valid if the  ** [ReminderOverrideDefault](652e4cf9-9183-0940-4376-2a1ba4337bb5.md)**and  ** [ReminderPlaySound](449c0c37-b0f7-ff42-1ba4-6eabd990e4db.md)**properties are set to  **True**.
