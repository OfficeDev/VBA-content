
# TaskItem.GetRecurrencePattern Method (Outlook)

 **Last modified:** July 28, 2015

Returns a  ** [RecurrencePattern](36c098f7-59fb-879a-5173-ed0260d13fa4.md)** object that represents the recurrence attributes of a task. If there is no existing recurrence pattern, a new empty **RecurrencePattern** object is returned.

## Syntax

 _expression_. **GetRecurrencePattern**

 _expression_A variable that represents a  **TaskItem** object.


### Return Value

A  **RecurrencePattern** that represents the recurrence attributes of the task.
