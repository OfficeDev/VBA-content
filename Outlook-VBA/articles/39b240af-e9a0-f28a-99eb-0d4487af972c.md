
# MoveOrCopyRuleAction Members (Outlook)
Represents an action that moves or copies a message.

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [ActionType](204bef7d-a19a-abd1-d494-23c33aa9f145.md)|Returns a constant,  **olRuleActionMoveToFolder**, from the  ** [OlRuleActionType](d6a39ac2-00e7-73e7-3890-ea658211eae9.md)** enumeration that indicates the type of action that is taken by the rule. Read-only.|
| [Application](9b2b2206-9544-54b3-772b-90a0ce5da846.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](103d5283-1613-7a49-0fcf-dd9e153ed18e.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [Enabled](795374af-a8de-b771-97df-3d9e82949af0.md)|Returns or sets a  **Boolean** that determines if the rule action is enabled. Read/write.|
| [Folder](d9588bab-c863-428f-0bbe-289f0981ff0f.md)|Returns or sets a  ** [Folder](3cf6cda8-6d70-666e-2643-9d9c5b9cacfc.md)** object that represents the folder to which the rule moves or copies the message. Read/write.|
| [Parent](07231a6d-d64c-81a4-1ffa-73b3c97b6e32.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](a77c9ccf-6f8d-92de-f6d4-1f3d7e92c810.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
