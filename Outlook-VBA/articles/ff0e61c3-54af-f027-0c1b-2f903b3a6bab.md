
# SharingItem.RecipientReassignmentProhibited Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **Boolean** that indicates **true** if the recipient cannot forward the specified ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)**. Read/write.

## Syntax

 _expression_. **RecipientReassignmentProhibited**

 _expression_A variable that represents a  **SharingItem** object.
