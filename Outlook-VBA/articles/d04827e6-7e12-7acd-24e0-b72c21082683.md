
# TasksModule.Visible Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Boolean** value that determines whether the ** [TasksModule](fc6ae6c9-6b13-b5f2-9506-c3dbbe709df6.md)** object is displayed in the Navigation Pane. Read/write.

## Syntax

 _expression_. **Visible**

 _expression_A variable that represents a  **TasksModule** object.
