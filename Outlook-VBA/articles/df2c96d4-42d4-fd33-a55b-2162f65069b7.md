
# OlkDateControl.Value Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **Variant** that represents the content of the control. Read/write.

## Syntax

 _expression_. **Value**

 _expression_A variable that represents an  **OlkDateControl** object.


## Remarks

The value is the same as the value of  ** [Text](fda479bc-c613-171f-4e81-efe9c548fd81.md)**, except that this property is  **Variant** and **Text** is strongly typed as **String**.
