
# OlkSenderPhoto.PreferredHeight Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  **Long** that specifies the application-preferred height of the sender picture. Read-only.

## Syntax

 _expression_. **PreferredHeight**

 _expression_A variable that represents an  **OlkSenderPhoto** object.


## Remarks

The value is expressed in units of pixels. The default value is 96.

 You can use this property and the ** [PreferredWidth](8546b80a-a191-bfce-2e24-6bd74f8e2a8b.md)** property to simulate Outlook's behavior to render, resize, and align the sender picture in a standard form.
