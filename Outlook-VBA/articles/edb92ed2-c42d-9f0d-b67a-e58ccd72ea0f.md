
# ContactItem.Suffix Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** indicating the name suffix (such as Jr., III, or Ph.D.) for the specified contact. Read/write.

## Syntax

 _expression_. **Suffix**

 _expression_A variable that represents a  **ContactItem** object.


## Remarks

The  ** [LastName](430682f6-a230-887b-404b-a71989121fa2.md)**,  ** [FirstName](403b5e5a-037b-cf21-efc2-2bd2a80c3789.md)**,  ** [MiddleName](07e0c9b1-1093-2f8a-3b89-ba8570b2bdf5.md)**, and  **Suffix** properties are parsed from the ** [FullName](3036dc57-31fb-45ad-f51e-49336206581d.md)**property.
