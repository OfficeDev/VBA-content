
# PostItem.Post Method (Outlook)

 **Last modified:** July 28, 2015

Sends (posts) the  ** [PostItem](de44065d-4e93-315a-279f-7b92f09c0465.md)**object.

## Syntax

 _expression_. **Post**

 _expression_A variable that represents a  **PostItem** object.


## Remarks

 The **Post** method, which is analogous to the ** [Send](78c85013-523e-447b-c47d-2da0705f1fe0.md)**method for the  ** [MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)**object, is used to save the post to the target public folder instead of mailing it.
