
# TextRuleCondition Members (Outlook)
Represents a rule condition that the part of the message, which can be the body, header, or subject, as specified by  ** [TextRuleCondition.ConditionType](2dbc7979-deae-fbb8-9def-8c906657024a.md)**, contains the words specified in  ** [TextRuleCondition.Text](615f47e9-2c43-a473-33f6-46765ccd3903.md)**.

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](4e6d0b1b-4d7a-fd92-f8a1-07c1dc21ddb3.md)|Returns an  ** [Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)**object that represents the parent Outlook application for the object. Read-only.|
| [Class](3c60e11c-7ddc-13a5-59b6-894e641f9238.md)|Returns an  ** [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
| [ConditionType](2dbc7979-deae-fbb8-9def-8c906657024a.md)|Returns a constant from the  ** [OlRuleConditionType](35c2f965-0f9d-8cc8-2f05-60522268574f.md)** enumeration that indicates the type of rule condition. Read-only.|
| [Enabled](7027c22b-08fa-d1b0-f664-8c4a26722cbb.md)|Returns or sets a  **Boolean** that determines if the rule condition is enabled. Read/write.|
| [Parent](55be70c5-a4d9-3359-8bf6-f71ae8f75a7d.md)|Returns the parent  **Object** of the specified object. Read-only.|
| [Session](29422538-9045-66b5-44a1-b226870dc307.md)|Returns the  ** [NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)**object for the current session. Read-only.|
| [Text](615f47e9-2c43-a473-33f6-46765ccd3903.md)|Returns or sets an array of  **String** elements that represents the text to be evaluated by the rule condition. Read/write.|
