
# CardView.SortFields Property (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [OrderFields](e115fb80-352d-fd2e-c1c3-d266776fe122.md)** object that represents the set of fields by which the items displayed in the ** [CardView](cdac229b-f2b6-9ecb-e1a7-b53509426570.md)** object are ordered. Read-only.

## Syntax

 _expression_. **SortFields**

 _expression_A variable that represents a  **CardView** object.
