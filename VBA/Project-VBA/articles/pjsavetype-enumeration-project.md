# PjSaveType Enumeration (Project)

Contains constants that specify save behavior.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**pjDoNotSave**|0|Do not save.|
|**pjPromptSave**|2|Prompt the user before saving.|
|**pjSave**|1|Save.|
