# ChildDrivers.Count Property (Project)

Gets the number of items in the  **[ChildDrivers](childdrivers-object-project.md)** collection. Read-only **Long**.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **ChildDrivers** object.


## Remarks

If  **TotalDetectedCount** is greater than 5 then count is 0.

Use of the  **Count** property in most collection objects is similar. For an example, see the **[Assignments.Count](assignments-count-property-project.md)** property.
