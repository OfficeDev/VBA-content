# Assignment.Cost4 Property (Project)

Gets or sets the value of the  **Cost4** roll-down custom field for the assignment. Read/write **Variant**.


## Syntax

 _expression_. **Cost4**

 _expression_ A variable that represents an **Assignment** object.
