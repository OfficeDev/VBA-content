# Assignment.Cost1 Property (Project)

Gets or sets the value of the  **Cost1** roll-down custom field for the assignment. Read/write **Variant**.


## Syntax

 _expression_. **Cost1**

 _expression_ A variable that represents an **Assignment** object.
