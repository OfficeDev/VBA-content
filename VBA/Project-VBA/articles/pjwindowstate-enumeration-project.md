# PjWindowState Enumeration (Project)

Contains constants that specify the window state.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**pjMaximized**|3|Maximized.|
|**pjMinimized**|2|Minimized.|
|**pjNormal**|1|Normal.|
