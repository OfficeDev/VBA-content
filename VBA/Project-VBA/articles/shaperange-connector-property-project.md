# ShapeRange.Connector Property (Project)
Gets a value that indicates whether the shape range is a connector. Read-only  **[MsoTriState](http://msdn.microsoft.com/en-us/library/office/ff860737%28v=office.15%29)**

## Syntax

 _expression_. **Connector**

 _expression_ A variable that represents a **ShapeRange** object.


## Property value

 **MSOTRISTATE**
