# Assignment.Cost6 Property (Project)

Gets or sets the value of the  **Cost6** roll-down custom field for the assignment. Read/write **Variant**.


## Syntax

 _expression_. **Cost6**

 _expression_ A variable that represents an **Assignment** object.
