# Application.FormatPaste Method (Project)

Pastes formats (copied with the  **FormatCopy** method) into the selected cells.


## Syntax

 _expression_. **FormatPaste**

 _expression_ A variable that represents an **Application** object.


### Return Value

 **Boolean**
