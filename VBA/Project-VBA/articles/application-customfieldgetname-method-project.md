# Application.CustomFieldGetName Method (Project)

Returns the friendly name for a custom field.


## Syntax

 _expression_. **CustomFieldGetName**( ** _FieldID_** )

 _expression_ A variable that represents an **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _FieldID_|Required|**Long**|The custom field. Can be one of the  **[PjCustomField](pjcustomfield-enumeration-project.md)** constants.|

### Return Value

 **String**
