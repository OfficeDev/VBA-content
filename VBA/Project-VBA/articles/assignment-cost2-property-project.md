# Assignment.Cost2 Property (Project)

Gets or sets the value of the  **Cost2** roll-down custom field for the assignment. Read/write **Variant**.


## Syntax

 _expression_. **Cost2**

 _expression_ A variable that represents an **Assignment** object.
