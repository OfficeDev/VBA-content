# Application.DetailStylesRemoveAll Method (Project)

Removes all timescale data fields from a usage view.


## Syntax

 _expression_. **DetailStylesRemoveAll**

 _expression_ A variable that represents an **Application** object.


### Return Value

 **Boolean**
