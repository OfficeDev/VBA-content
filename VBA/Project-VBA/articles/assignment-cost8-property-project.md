# Assignment.Cost8 Property (Project)

Gets or sets the value of the  **Cost8** roll-down custom field for the assignment. Read/write **Variant**.


## Syntax

 _expression_. **Cost8**

 _expression_ A variable that represents an **Assignment** object.
