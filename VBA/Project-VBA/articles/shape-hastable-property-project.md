# Shape.HasTable Property (Project)
Gets a value that indicates whether the shape contains a table. Read-only  **[MsoTriState](http://msdn.microsoft.com/en-us/library/office/ff860737%28v=office.15%29)**.

## Syntax

 _expression_. **HasTable**

 _expression_ A variable that represents a **Shape** object.


## Property value

 **MSOTRISTATE**
