# PjLevelOrder Enumeration (Project)

Contains constants that specify the leveling order.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**pjLevelID**|0|Level ID.|
|**pjLevelPriority**|2|Level priority.|
|**pjLevelStandard**|1|Level standard.|
