# PjItemType Enumeration (Project)

Contains constants that specify the type of an item.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**pjOtherItem**|2|Other item.|
|**pjResourceItem**|1|Resource item.|
|**pjTaskItem**|0|Task item.|
