# Assignment.Cost7 Property (Project)

Gets or sets the value of the  **Cost7** roll-down custom field for the assignment. Read/write **Variant**.


## Syntax

 _expression_. **Cost7**

 _expression_ A variable that represents an **Assignment** object.
