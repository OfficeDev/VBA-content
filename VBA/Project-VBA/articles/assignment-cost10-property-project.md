# Assignment.Cost10 Property (Project)

Gets or sets the value of the  **Cost10** roll-down custom field for the assignment. Read/write **Variant**.


## Syntax

 _expression_. **Cost10**

 _expression_ A variable that represents an **Assignment** object.
