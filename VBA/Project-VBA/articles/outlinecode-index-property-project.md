# OutlineCode.Index Property (Project)

Gets the index of an  **OutlineCode** object in the containing **OutlineCodes** collection. Read-only **Long**.


## Syntax

 _expression_. **Index**

 _expression_ A variable that represents an **OutlineCode** object.


## Remarks

The  **Index** properties of other objects are used in similar ways. For an example, see the **[Index](project-index-property-project.md)** property of the **Project** object.
