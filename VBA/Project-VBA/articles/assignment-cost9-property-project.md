# Assignment.Cost9 Property (Project)

Gets or sets the value of the  **Cost9** roll-down custom field for the assignment. Read/write **Variant**.


## Syntax

 _expression_. **Cost9**

 _expression_ A variable that represents an **Assignment** object.
