# PjPlacement Enumeration (Project)

Contains constants that specify the placement of a symbol.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**pjAfter**|1|After.|
|**pjAfterWithSpace**|3|After with a space.|
|**pjBefore**|0|Before.|
|**pjBeforeWithSpace**|2|Before with a space.|
