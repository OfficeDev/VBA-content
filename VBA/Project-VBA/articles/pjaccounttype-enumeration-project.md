# PjAccountType Enumeration (Project)

Contains constants that specify the Project Server account type.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**pjManagerAccount**|1|Manager account.|
|**pjResourceAccount**|0|Resource account.|
