# Project.Path Property (Project)

Gets the path of the open project. Read-only  **String**.


## Syntax

 _expression_. **Path**

 _expression_ A variable that represents a **Project** object.


## Remarks

The  **Path** property returns an empty string ("") for an unsaved project.
