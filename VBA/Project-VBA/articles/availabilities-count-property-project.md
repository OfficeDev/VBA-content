# Availabilities.Count Property (Project)

Gets the number of items in the  **Availabilities** collection. Read-only **Long**.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents an **Availabilities** object.


## Remarks

Use of the  **Count** property in most collection objects is similar. For an example, see the **[Assignments.Count](assignments-count-property-project.md)** property.
