# Views.Count Property (Project)

Gets the number of items in the  **Views** collection. Read-only **Long**. .


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **Views** object.


## Remarks

Use of the  **Count** property in most collection objects is similar. For an example, see the **[Assignments.Count](assignments-count-property-project.md)** property.
