# Image.HorizontalAnchor Property (Access)

Gets or sets an  **[AcHorizontalAnchor](achorizontalanchor-enumeration-access.md)** constant that indicates how the image is anchored horizontally within its layout. Read/write.


## Syntax

 _expression_. **HorizontalAnchor**

 _expression_ A variable that represents an **Image** object.


## Remarks

The default value is  **acHorizontalAnchorLeft**.
