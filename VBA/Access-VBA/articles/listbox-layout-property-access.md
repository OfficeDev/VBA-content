# ListBox.Layout Property (Access)

Returns the type of layout for the specified list box. Read-only  **[AcLayoutType](aclayouttype-enumeration-access.md)**.


## Syntax

 _expression_. **Layout**

 _expression_ A variable that represents a **ListBox** object.
