# Failure to execute the HTTP request. (Error 3677)

  

**Applies to:** Access 2013 | Access 2016

Failed attempting to send a request to the internet server. The request could have been one of the following:



- An attempt to open and read a file across the internet.
    
- A request to delete a file across the internet.
    
- An attempt to start the MSTRAI40.exe syncronization process on the internet server.
