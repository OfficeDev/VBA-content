# CommandButton.VerticalAnchor Property (Access)

Gets or sets an [AcVerticalAnchor](acverticalanchor-enumeration-access.md) constant that indicates how the specified button is anchored vertically within its layout. Read/write.


## Syntax

 _expression_. **VerticalAnchor**

 _expression_ A variable that represents a **CommandButton** object.


## Remarks

The default value is  **acVerticalAnchorTop**.
