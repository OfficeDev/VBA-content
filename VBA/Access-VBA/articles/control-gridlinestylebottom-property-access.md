# Control.GridlineStyleBottom Property (Access)

Gets or sets the bottom gridline style of the specified control. Read/write  **Byte**.


## Syntax

 _expression_. **GridlineStyleBottom**

 _expression_ A variable that represents a **Control** object.


## Remarks

The  **BottomGridlineStyle** property uses the following settings.



|**Setting**|**Gridline type**|
|:-----|:-----|
|0 (default)|Transparent|
|1|Solid|
|2|Dashes|
|3|Short Dashes|
|4|Dots|
|5|Sparse Dots|
|6|Dash Dot|
|7|Dash Dot Dot|
