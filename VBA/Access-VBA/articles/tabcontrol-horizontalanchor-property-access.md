# TabControl.HorizontalAnchor Property (Access)

Gets or sets an  **[AcHorizontalAnchor](achorizontalanchor-enumeration-access.md)** constant that indicates how the tab control is anchored horizontally within its layout. Read/write.


## Syntax

 _expression_. **HorizontalAnchor**

 _expression_ A variable that represents a **TabControl** object.


## Remarks

The default value is  **acHorizontalAnchorLeft**.
