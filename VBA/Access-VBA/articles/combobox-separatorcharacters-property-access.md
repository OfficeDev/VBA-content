# ComboBox.SeparatorCharacters Property (Access)

Gets or sets the separator displayed between values when the combo box is bound to a multi-valued field. Read/write [AcSeparatorCharacters](acseparatorcharacters-enumeration-access.md).


## Syntax

 _expression_. **SeparatorCharacters**

 _expression_ A variable that represents a **ComboBox** object.
