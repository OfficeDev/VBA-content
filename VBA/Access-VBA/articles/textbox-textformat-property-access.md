# TextBox.TextFormat Property (Access)

Gets or sets whether rich text is displayed in the specified text box. Read/write [AcTextFormat](actextformat-enumeration-access.md).


## Syntax

 _expression_. **TextFormat**

 _expression_ A variable that represents a **TextBox** object.
