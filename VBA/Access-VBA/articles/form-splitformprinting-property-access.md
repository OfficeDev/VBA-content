# Form.SplitFormPrinting Property (Access)

Gets or sets whether the contents of the form or the datasheet are printed when printing a form displayed in Split Form view. Read/write [AcSplitFormPrinting](acsplitformprinting-enumeration-access.md).


## Syntax

 _expression_. **SplitFormPrinting**

 _expression_ A variable that represents a **Form** object.
