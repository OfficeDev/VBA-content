# ContainerProperties.ObjectType Property (Visio)

Returns  **visObjTypeContainerProperties** , the type of this object. Read-only.


## Syntax

 _expression_ . **ObjectType**

 _expression_ An expression that returns a **[ContainerProperties](containerproperties-object-visio.md)** object.


### Return Value

 **visObjTypeContainerProperties**


## Remarks

Constants that represent object types are prefixed with  **visObjType** and are declared by the Visio type library in the **[VisObjectTypes](visobjecttypes-enumeration-visio.md)** enumeration.
