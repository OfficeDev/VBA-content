# Hyperlink.Copy Method (Visio)

Copies a hyperlink to the Clipboard.


## Syntax

 _expression_ . **Copy**

 _expression_ A variable that represents a **Hyperlink** object.


### Return Value

Nothing


## Remarks

To make a copy without using the Clipboard, use the  **Duplicate** method.
