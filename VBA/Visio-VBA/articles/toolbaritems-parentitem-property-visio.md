# ToolbarItems.ParentItem Property (Visio)

Returns the parent object of a hierarchical toolbar. Read-only.


## Syntax

 _expression_ . **ParentItem**

 _expression_ A variable that represents a **ToolbarItems** object.


### Return Value

ToolbarItem
