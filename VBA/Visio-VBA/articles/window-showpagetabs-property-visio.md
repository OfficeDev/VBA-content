# Window.ShowPageTabs Property (Visio)

Determines whether page tab controls are shown in the drawing window. Read/write.


## Syntax

 _expression_ . **ShowPageTabs**

 _expression_ A variable that represents a **Window** object.


### Return Value

Boolean
