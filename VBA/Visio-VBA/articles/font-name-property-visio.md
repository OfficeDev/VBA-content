# Font.Name Property (Visio)

Specifies the name of an object. Read-only.


## Syntax

 _expression_ . **Name**

 _expression_ A variable that represents a **Font** object.


### Return Value

String


## Remarks

You can get, but not set, the  **Name** property of an **Addon** object or a **Font** object.
