# ApplicationSettings.CenterSelectionOnZoom Property (Visio)

Determines whether when the user zooms in, the selection appears in the center of the window. Read/write.


## Syntax

 _expression_ . **CenterSelectionOnZoom**

 _expression_ A variable that represents an **ApplicationSettings** object.


### Return Value

Boolean


## Remarks

Setting the  **CenterSelectionOnZoom** property is equivalent to selecting **Center selection on zoom** under **Editing options** on the **Advanced** tab in the **Visio Options** dialog box (click the **File** tab, and then click **Options**).
