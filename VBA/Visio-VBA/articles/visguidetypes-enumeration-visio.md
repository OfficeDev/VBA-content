# VisGuideTypes Enumeration (Visio)

Guide type codes to be passed to the  **Page.AddGuide** method.



|**Constant**|**Value**|**Description**|
|:-----|:-----|:-----|
| **visHorz**|2|Vertical guide|
| **visPoint**|1|Guide point|
| **visVert**|3|Horizontal guide|
