# CoauthMergeEvent.DownloadDocument Property (Visio)

Returns a [Document](document-object-visio.md) object that represents a document that includes changes from all users other than the current user. Read-only.


## Syntax

 _expression_ . **DownloadDocument**

 _expression_ A variable that represents a **CoauthMergeEvent** object.


## Property value

 **IVDOCUMENT**
