# Characters.Copy Method (Visio)

Copies a text range to the Clipboard.


## Syntax

 _expression_ . **Copy**

 _expression_ A variable that represents a **Characters** object.


### Return Value

Nothing


## Remarks

To make a copy without using the Clipboard, use the  **Duplicate** method.
