# ValidationRuleSets.ObjectType Property (Visio)

Returns  **visObjTypeValidationRuleSets** , the type of this object. Read-only.


## Syntax

 _expression_ . **ObjectType**

 _expression_ A variable that represents a **[ValidationRuleSets](validationrulesets-object-visio.md)** object.


### Return Value

 **visObjTypeValidationRuleSets**


## Remarks

Constants that represent object types are prefixed with  **visObjType** and are declared by the Visio type library in the **[VisObjectTypes](visobjecttypes-enumeration-visio.md)** enumeration.
