# Validation.Document Property (Visio)

Returns the  **[Document](document-object-visio.md)** object that is associated with an object. Read-only.


## Syntax

 _expression_ . **Document**

 _expression_ A variable that represents a **[Validation](validation-object-visio.md)** object.


### Return Value

 **Document**
