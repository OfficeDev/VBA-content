# DisplayUnitLabel.Shadow Property (Excel)

Returns or sets a  **Boolean** value that determines if the object has a shadow.


## Syntax

 _expression_ . **Shadow**

 _expression_ A variable that represents a **DisplayUnitLabel** object.
