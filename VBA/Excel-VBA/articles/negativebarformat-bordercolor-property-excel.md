# NegativeBarFormat.BorderColor Property (Excel)

 Returns a **[FormatColor](formatcolor-object-excel.md)** object that you can use to specify the border color for negative data bars. Read-only


## Syntax

 _expression_ . **BorderColor**

 _expression_ A variable that represents a **[NegativeBarFormat](negativebarformat-object-excel.md)** object.


### Return Value

 **FormatColor**
