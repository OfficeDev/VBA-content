# XlDisplayDrawingObjects Enumeration (Excel)

Specifies how shapes are displayed.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlDisplayShapes**|-4104|Show all shapes.|
| **xlHide**|3|Hide all shapes.|
| **xlPlaceholders**|2|Show only placeholders.|
