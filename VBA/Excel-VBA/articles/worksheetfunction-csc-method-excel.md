# WorksheetFunction.Csc Method (Excel)

Returns the cosecant of an angle.


## Syntax

 _expression_ . **Csc**_(Arg1)_

 _expression_ A variable that represents a[WorksheetFunction Object (Excel)](worksheetfunction-object-excel.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Required|DOUBLE|The angle in radians for which you want the cosecant.|

### Return value

 **DOUBLE**
