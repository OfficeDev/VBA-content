# FileExportConverter.Parent Property (Excel)

Returns an  **Object** that represents the parent object of the specified **[FileExportConverter](fileexportconverter-object-excel.md)** object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **FileExportConverter** object.
