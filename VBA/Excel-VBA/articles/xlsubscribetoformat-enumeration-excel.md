# XlSubscribeToFormat Enumeration (Excel)

Specifies the format to use when subscribing to a published edition.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlSubscribeToPicture**|-4147|Picture|
| **xlSubscribeToText**|-4158|Text|
