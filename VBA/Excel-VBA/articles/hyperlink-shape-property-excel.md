# Hyperlink.Shape Property (Excel)

Returns a  **[Shape](shape-object-excel.md)** object that represents the shape attached to the specified hyperlink.


## Syntax

 _expression_ . **Shape**

 _expression_ An expression that returns a **Hyperlink** object.
