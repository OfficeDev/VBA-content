# SlicerItem.Value Property (Excel)

Returns the value of the slicer item. Read-only.


## Syntax

 _expression_ . **Value**

 _expression_ A variable that represents a **[SlicerItem](sliceritem-object-excel.md)** object.


### Return Value

String
