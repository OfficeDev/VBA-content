# TableObject.Delete Method (Excel)

Deletes the  **TableObject** object.


## Syntax

 _expression_ . **Delete**

 _expression_ A variable that represents a[ModelTable Object (Excel)](modeltable-object-excel.md) object.


### Return value

 **VOID**
