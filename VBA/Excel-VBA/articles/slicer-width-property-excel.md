# Slicer.Width Property (Excel)

Returns or sets the width of the specified slicer, in points. Read/write.


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[Slicer](slicer-object-excel.md)** object.


### Return Value

Double
