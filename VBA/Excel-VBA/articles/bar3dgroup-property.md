# Bar3DGroup Property

Returns a  **[ChartGroup](chartgroup-object.md)** object that represents the specified bar chart group on a 3-D chart. Read-only.


## Example

This example sets the space between bar clusters in the 3-D bar chart group to be 50 percent of the bar width.


```
myChart.BarGroup3DGroup.GapWidth = 50
```
