# SparklineGroup.Item Property (Excel)

Returns a  **[Sparkline](sparkline-object-excel.md)** object. Read-only


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ A variable that represents a **[SparklineGroup](sparklinegroup-object-excel.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Variant**|Index value of the sparkline.|

### Return Value

Sparkline
