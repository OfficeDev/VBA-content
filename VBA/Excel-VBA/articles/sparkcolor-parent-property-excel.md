# SparkColor.Parent Property (Excel)

Returns the parent  **[SparklineGroup](sparklinegroup-object-excel.md)** object for the specified object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[SparkColor](sparkcolor-object-excel.md)** object.
