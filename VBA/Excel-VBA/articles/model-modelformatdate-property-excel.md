# Model.ModelFormatDate Property (Excel)

Returns a [ModelFormatDate](modelformatdate-object-excel.md) object that represents formatting of type date in the data model. Read-only .


## Syntax

 _expression_ . **ModelFormatDate**

 _expression_ A variable that represents a **Model** object.
