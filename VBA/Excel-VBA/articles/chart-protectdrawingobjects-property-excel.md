# Chart.ProtectDrawingObjects Property (Excel)

 **True** if shapes are protected. To turn on shape protection, use the **[Protect](chart-protect-method-excel.md)** method with the _DrawingObjects_ argument set to **True** . Read-only **Boolean** .


## Syntax

 _expression_ . **ProtectDrawingObjects**

 _expression_ A variable that represents a **Chart** object.
