# XlReferenceStyle Enumeration (Excel)

Specifies the reference style.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlA1**|1|Default. Use  **xlA1** to return an A1-style reference.|
| **xlR1C1**|-4150|Use  **xlR1C1** to return an R1C1-style reference.|
