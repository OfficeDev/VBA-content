# ChartTitle.Format Property (Excel)

Returns the  **[ChartFormat](chartformat-object-excel.md)** object. Read-only.


## Syntax

 _expression_ . **Format**

 _expression_ A variable that represents a **ChartTitle** object.


## Remarks

The  **[ChartFormat](chartformat-object-excel.md)** object contains the line, fill, effect, and text formatting for the chart area.
