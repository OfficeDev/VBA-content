# SlicerItem.Name Property (Excel)

Returns the name of the slicer item. Read-only.


## Syntax

 _expression_ . **Name**

 _expression_ A variable that represents a **[SlicerItem](sliceritem-object-excel.md)** object.


### Return Value

String
