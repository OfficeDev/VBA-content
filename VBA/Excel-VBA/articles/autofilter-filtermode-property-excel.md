# AutoFilter.FilterMode Property (Excel)

Returns  **True** if the worksheet is in the AutoFilter filter mode. Read-only **Boolean** .


## Syntax

 _expression_ . **FilterMode**

 _expression_ A variable that represents an **AutoFilter** object.
