# SparklineGroup.Type Property (Excel)

Gets or sets the type of sparkline for the group. Read/write.


## Syntax

 _expression_ . **Type**

 _expression_ A variable that represents a **[SparklineGroup](sparklinegroup-object-excel.md)** object.


### Return Value

XlSparkType
