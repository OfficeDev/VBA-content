# ModelTableNameChanges.Item Method (Excel)

Returns a single object from the  **ModelTableNameChanges** collection.


## Syntax

 _expression_ . **Item**_(Index)_

 _expression_ A variable that represents a[ModelTableNameChanges Object (Excel)](modeltablenamechanges-object-excel.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required|VARIANT|The index number for the object.|

### Return value

 **MODELTABLENAMECHANGE**
