# TableObject.Refresh Method (Excel)

This method updates the  **TableObject** object.


## Syntax

 _expression_ . **Refresh**

 _expression_ A variable that represents a[ModelTable Object (Excel)](modeltable-object-excel.md) object.


### Return value

 **BOOL**
