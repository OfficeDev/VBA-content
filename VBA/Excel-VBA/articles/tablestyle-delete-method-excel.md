# TableStyle.Delete Method (Excel)

Deletes the  **[TableStyle](tablestyle-object-excel.md)** object.


## Syntax

 _expression_ . **Delete**

 _expression_ A variable that represents a **TableStyle** object.


### Return Value

Nothing
