# LegendKey.PictureType Property (Excel)

Returns or sets a  **[XlChartPictureType](xlchartpicturetype-enumeration-excel.md)** value that represents the way pictures are displayed on a legend key.


## Syntax

 _expression_ . **PictureType**

 _expression_ A variable that represents a **LegendKey** object.
