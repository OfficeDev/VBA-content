# WorksheetFunction.Arabic Method (Excel)

Converts a Roman numeral to an Arabic numeral.


## Syntax

 _expression_ . **Arabic**_(Arg1)_

 _expression_ A variable that represents a[WorksheetFunction Object (Excel)](worksheetfunction-object-excel.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Required|STRING|The Roman numeral you want to convert.|

### Return value

 **DOUBLE**
