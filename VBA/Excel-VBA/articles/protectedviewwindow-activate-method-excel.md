# ProtectedViewWindow.Activate Method (Excel)

Brings the  **Protected View** window to the front of the z-order.


## Syntax

 _expression_ . **Activate**

 _expression_ A variable that represents a **[ProtectedViewWindow](protectedviewwindow-object-excel.md)** object.


### Return Value

Nothing
