# WorksheetFunction.Gamma Method (Excel)

Returns the Gamma function value.


## Syntax

 _expression_ . **Gamma**_(Arg1)_

 _expression_ A variable that represents a[WorksheetFunction Object (Excel)](worksheetfunction-object-excel.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Required|DOUBLE|The value for which you want to calculate Gamma.|

### Return value

 **DOUBLE**
