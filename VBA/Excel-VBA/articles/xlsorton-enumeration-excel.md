# XlSortOn Enumeration (Excel)

Specifies the parameter on which the data should be sorted.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **SortOnCellColor**|1|Cell color.|
| **SortOnFontColor**|2|Font color.|
| **SortOnIcon**|3|Icon.|
| **SortOnValues**|0|Values.|
