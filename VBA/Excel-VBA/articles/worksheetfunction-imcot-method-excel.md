# WorksheetFunction.ImCot Method (Excel)

Returns the cotangent of a complex number.


## Syntax

 _expression_ . **ImCot**_(Arg1)_

 _expression_ A variable that represents a[WorksheetFunction Object (Excel)](worksheetfunction-object-excel.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Required|STRING|Complex number for which you want the cotangent.|

### Return value

 **STRING**
