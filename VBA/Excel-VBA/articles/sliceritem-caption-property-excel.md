# SlicerItem.Caption Property (Excel)

Returns the label text for the slicer item. Read-only.


## Syntax

 _expression_ . **Caption**

 _expression_ A variable that represents a **[SlicerItem](sliceritem-object-excel.md)** object.


### Return Value

String
