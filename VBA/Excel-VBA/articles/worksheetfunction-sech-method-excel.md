# WorksheetFunction.Sech Method (Excel)

Returns the hyperbolic secant of an angle.


## Syntax

 _expression_ . **Sech**_(Arg1)_

 _expression_ A variable that represents a[WorksheetFunction Object (Excel)](worksheetfunction-object-excel.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Required|DOUBLE|Angle in radians for which you want the hyperbolic secant.|

### Return value

 **DOUBLE**
