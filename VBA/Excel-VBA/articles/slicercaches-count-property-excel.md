# SlicerCaches.Count Property (Excel)

Returns the number of objects in the collection.


## Syntax

 _expression_ . **Count**

 _expression_ A variable that returns a **[SlicerCaches](slicercaches-object-excel.md)** collection.


### Return Value

 **Long**
