# WorksheetFunction.Phi Method (Excel)

Returns the value of the density function for a standard normal distribution.


## Syntax

 _expression_ . **Phi**_(Arg1)_

 _expression_ A variable that represents a[WorksheetFunction Object (Excel)](worksheetfunction-object-excel.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Required|DOUBLE|Number for which you want the density of the standard normal distribution.|

### Return value

 **DOUBLE**
