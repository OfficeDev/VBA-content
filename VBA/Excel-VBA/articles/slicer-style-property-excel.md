# Slicer.Style Property (Excel)

Returns or sets the style currently applied to the specified slicer. Read/write.


## Syntax

 _expression_ . **Style**

 _expression_ A variable that represents a **[Slicer](slicer-object-excel.md)** object.


### Return Value

Variant
