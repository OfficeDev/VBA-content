# Update Method

Updates the specified embedded object in the host file.

 _expression_. **Update**

 _expression_ Required. An expression that returns an **[Application](application-object.md)** object.


## Example

This example updates the application.


```vb
Sub UseUpdate() 
 
 Application.Update 
 
End Sub
```
