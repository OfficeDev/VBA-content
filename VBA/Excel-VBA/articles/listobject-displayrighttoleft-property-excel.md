# ListObject.DisplayRightToLeft Property (Excel)

 **True** if the specified **[ListObject](listobject-object-excel.md)** is displayed from right to left instead of from left to right. **False** if the object is displayed from left to right. Read-only **Boolean** .


## Syntax

 _expression_ . **DisplayRightToLeft**

 _expression_ A variable that represents a **ListObject** object.
