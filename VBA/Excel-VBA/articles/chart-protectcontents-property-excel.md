# Chart.ProtectContents Property (Excel)

 **True** if the contents of the sheet are protected. For a chart, this protects the entire chart. To turn on content protection, use the **[Protect](chart-protect-method-excel.md)** method with the _Contents_ argument set to **True** . Read-only **Boolean** .


## Syntax

 _expression_ . **ProtectContents**

 _expression_ A variable that represents a **Chart** object.
