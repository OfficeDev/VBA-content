# SparklineGroups.ClearGroups Method (Excel)

Clears the selected sparkline groups.


## Syntax

 _expression_ . **ClearGroups**

 _expression_ A variable that represents a **[SparklineGroups](sparklinegroups-object-excel.md)** object.


### Return Value

Nothing
