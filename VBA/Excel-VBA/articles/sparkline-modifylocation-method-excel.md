# Sparkline.ModifyLocation Method (Excel)

Modifies the location of a single sparkline.


## Syntax

 _expression_ . **ModifyLocation**( **_Range_** )

 _expression_ A variable that represents a **[Sparkline](sparkline-object-excel.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Range_|Required| **[Range](range-object-excel.md)**|The cell that contains the sparkline.|

### Return Value

Nothing
