# XlSaveAction Enumeration (Excel)

Specifies during file close if the file will be saved.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlDoNotSaveChanges**|2|Changes will not be saved.|
| **xlSaveChanges**|1|Changes will be saved.|
