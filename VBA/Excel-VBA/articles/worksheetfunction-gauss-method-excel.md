# WorksheetFunction.Gauss Method (Excel)

Returns 0.5 less than the standard normal cumulative distribution.


## Syntax

 _expression_ . **Gauss**_(Arg1)_

 _expression_ A variable that represents a[WorksheetFunction Object (Excel)](worksheetfunction-object-excel.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Required|DOUBLE|The value for which you want the distribution.|

### Return value

 **DOUBLE**
