# Chart.Shapes Property (Excel)

Returns a  **[Shapes](shapes-object-excel.md)** collection that represents all the shapes on the chart sheet. Read-only.


## Syntax

 _expression_ . **Shapes**

 _expression_ An expression that returns a **Chart** object.
