# Model.ModelFormatCurrency Property (Excel)

Returns a [ModelFormatCurrency](modelformatcurrency-object-excel.md) object that represents formatting of type currency in the data model. Read-only .


## Syntax

 _expression_ . **ModelFormatCurrency**

 _expression_ A variable that represents a **Model** object.
