# ModelRelationship.Delete Method (Excel)

Deletes the relationship.


## Syntax

 _expression_ . **Delete**

 _expression_ A variable that represents a[ModelRelationship Object (Excel)](modelrelationship-object-excel.md) object.


### Return value

 **VOID**
