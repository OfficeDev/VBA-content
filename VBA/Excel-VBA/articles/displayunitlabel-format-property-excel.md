# DisplayUnitLabel.Format Property (Excel)

Returns the  **[ChartFormat](chartformat-object-excel.md)** object. Read-only.


## Syntax

 _expression_ . **Format**

 _expression_ A variable that represents a **DisplayUnitLabel** object.
