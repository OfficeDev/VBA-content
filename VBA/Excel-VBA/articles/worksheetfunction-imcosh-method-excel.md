# WorksheetFunction.ImCosh Method (Excel)

Returns the hyperbolic cosine of a complex number.


## Syntax

 _expression_ . **ImCosh**_(Arg1)_

 _expression_ A variable that represents a[WorksheetFunction Object (Excel)](worksheetfunction-object-excel.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Required|STRING|Complex number for which you want the hyperbolic cosine.|

### Return value

 **STRING**
