# OMathLimLow.ToLimUpp Method (Word)

Converts an equation from the lower limit to the upper limit. .


## Syntax

 _expression_ . **ToLimUpp**

 _expression_ An expression that returns a **OMathLimLow** object.


### Return Value

OMathFunction
