# ShadowFormat.Size Property (Word)

Returns or sets a  **Single** that represents the width of the shadow. Read/write.


## Syntax

 _expression_ . **Size**

 _expression_ An expression that returns a **[ShadowFormat](shadowformat-object-word.md)** object.
