# TaskPanes.Count Property (Word)

Returns a  **Long** that represents the number of task panes in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[TaskPanes](taskpanes-object-word.md)** collection.
