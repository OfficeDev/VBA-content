# WdOrientation Enumeration (Word)

Specifies a page layout orientation.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **wdOrientLandscape**|1|Landscape orientation.|
| **wdOrientPortrait**|0|Portrait orientation.|
