# TableOfAuthoritiesCategory.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TableOfAuthoritiesCategory** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TableOfAuthoritiesCategory](tableofauthoritiescategory-object-word.md)** object.
