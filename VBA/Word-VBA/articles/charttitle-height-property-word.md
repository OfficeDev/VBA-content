# ChartTitle.Height Property (Word)

Returns the height, in points, of the object. Read-only.


## Syntax

 _expression_ . **Height**

 _expression_ A variable that represents a **[ChartTitle](charttitle-object-word.md)** object.


### Return Value

Double
