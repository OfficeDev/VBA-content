# CheckBox.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **CheckBox** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[CheckBox](checkbox-object-word.md)** object.
