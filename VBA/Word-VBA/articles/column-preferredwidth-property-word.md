# Column.PreferredWidth Property (Word)

Returns or sets the preferred width (in points or as a percentage of the window width) for the specified column. Read/write  **Single** .


## Syntax

 _expression_**PreferredWidth**

 _expression_ Required. An expression that returns a **[Column](column-object-word.md)** object.


## Remarks

If the  **[PreferredWidthType](column-preferredwidthtype-property-word.md)** property is set to **wdPreferredWidthPoints** , the **PreferredWidth** property returns or sets the width in points. If the **PreferredWidthType** property is set to **wdPreferredWidthPercent** , the **PreferredWidth** property returns or sets the width as a percentage of the window width.
