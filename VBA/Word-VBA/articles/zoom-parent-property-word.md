# Zoom.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Zoom** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Zoom](zoom-object-word.md)** object.
