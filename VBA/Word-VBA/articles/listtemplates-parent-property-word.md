# ListTemplates.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ListTemplates** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ListTemplates](listtemplates-object-word.md)** collection.
