# TextRetrievalMode.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TextRetrievalMode** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TextRetrievalMode](textretrievalmode-object-word.md)** object.
