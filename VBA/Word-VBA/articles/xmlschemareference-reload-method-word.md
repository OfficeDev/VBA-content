# XMLSchemaReference.Reload Method (Word)

Reloads the XML schemas that are referenced in a document.


## Syntax

 _expression_ . **Reload**

 _expression_ An expression that returns an **[XMLSchemaReference](xmlschemareference-object-word.md)** object.


### Return Value

Nothing
