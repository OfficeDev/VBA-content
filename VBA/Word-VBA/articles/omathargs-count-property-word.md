# OMathArgs.Count Property (Word)

Returns the number of items in the  **OMathArgs** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[OMathArgs](omathargs-object-word.md)** object.
