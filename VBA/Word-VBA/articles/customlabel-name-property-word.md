# CustomLabel.Name Property (Word)

Returns or sets the name of the specified object. Read/write  **CustomLabel** .


## Syntax

 _expression_ . **Name**

 _expression_ Required. A variable that represents a **[CustomLabel](customlabel-object-word.md)** object.
