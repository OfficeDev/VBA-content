# Shapes.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Shapes** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Shapes](shapes-object-word.md)** collection.
