# CoAuthLocks.Application Property (Word)

Returns an [Application](application-object-word.md) object that represents the Microsoft Word application. Read-only.


## Syntax

 _expression_ . **Application**

 _expression_ An expression that returns a **[CoAuthLocks](coauthlocks-object-word.md)** object.
