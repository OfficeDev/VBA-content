# Options.PasteFormatBetweenStyledDocuments Property (Word)

Returns or sets a  **[WdPasteOptions](wdpasteoptions-enumeration-word.md)** constant that represents how text is pasted when text is copied from a document that uses styles. Read/write.


## Syntax

 _expression_ . **PasteFormatBetweenStyledDocuments**

 _expression_ An expression that returns an **Options** object.


## Remarks

Corresponds to the  **Pasting between two documents (with styles)** option in the **Advanced** tab of the **Word Options** dialog box.
