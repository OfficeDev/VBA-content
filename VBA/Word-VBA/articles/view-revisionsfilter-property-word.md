# View.RevisionsFilter Property (Word)

Returns an instance of a  **RevisionsFilter** object. Read-only.


## Syntax

 _expression_ . **RevisionsFilter**

 _expression_ A variable that represents a **View** object.


## Property value

 **REVISIONSFILTER**
