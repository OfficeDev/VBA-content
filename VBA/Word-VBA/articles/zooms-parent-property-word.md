# Zooms.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Zooms** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Zooms](zooms-object-word.md)** collection.
