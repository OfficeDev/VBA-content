# Document.ViewGuides Property (Publisher)

Returns  **True** if guides are visible in the specified publication. Read/write **Boolean**.


## Syntax

 _expression_. **ViewGuides**

 _expression_A variable that represents a  **Document** object.


### Return Value

Boolean
