# PageSizes.Parent Property (Publisher)

Returns the parent of the  **[PageSizes](pagesizes-object-publisher.md)** collection. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **PageSizes** collection.

Object
