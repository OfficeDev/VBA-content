# Printer.Parent Property (Publisher)

Returns the parent of the  **[Printer](printer-object-publisher.md)** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **Printer** object.


### Return Value

Object
