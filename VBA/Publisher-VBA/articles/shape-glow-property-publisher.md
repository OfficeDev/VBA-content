# Shape.Glow Property (Publisher)

Returns a  [GlowFormat](glowformat-object-publisher.md) object that represents the glow formatting for a shape. Read-only.


## Syntax

 _expression_. **Glow**

 _expression_A variable that represents a  **Shape** object.


## Return value

 **GLOWFORMAT**
