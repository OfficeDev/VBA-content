# PageSizes.Count Property (Publisher)

Returns the number of  **PageSize** objects in the **PageSizes** collection of the active document. Read-only.


## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **PageSizes** collection.


### Return Value

Long
