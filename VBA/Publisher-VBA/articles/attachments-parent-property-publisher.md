# Attachments.Parent Property (Publisher)

Returns the parent of the  **[Attachments](attachments-object-publisher.md)** collection. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_A variable that represents an  **Attachments** object.


### Return Value

Object
