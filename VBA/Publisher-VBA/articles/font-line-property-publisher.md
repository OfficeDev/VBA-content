# Font.Line Property (Publisher)

Returns a  [LineFormat](lineformat-object-publisher.md) object that specifies the formatting for a line. Read/write.


## Syntax

 _expression_. **Line**

 _expression_A variable that represents a  **Font** object.


## Return value

 **LINEFORMAT**
