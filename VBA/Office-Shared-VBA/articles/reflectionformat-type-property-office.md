# ReflectionFormat.Type Property (Office)

Gets or sets the type of the  **ReflectionFormat** object. Read/write.


## Syntax

 _expression_. **Type**

 _expression_ An expression that returns a **ReflectionFormat** object.


### Return Value

MsoReflectionType
