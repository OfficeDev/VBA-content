# CommandBarPopup.Id Property (Office)

Gets the ID for a built-in  **CommandBarPopup** control. Read-only.


## 


 **Note**  The use of CommandBars in some Microsoft Office applications has been superseded by the new ribbon component of the Microsoft Office Fluent user interface. For more information, search Help for the keyword "ribbon."


## Syntax

 _expression_. **Id**

 _expression_ A variable that represents a **CommandBarPopup** object.


### Return Value

Integer


## Remarks

A popup's ID determines the built-in action for that object.
