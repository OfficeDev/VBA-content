# CustomXMLPrefixMapping.Prefix Property (Office)

Gets the prefix for a  **CustomXMLPrefixMapping** object. Read-only.


## Syntax

 _expression_. **Prefix**

 _expression_ An expression that returns a **CustomXMLPrefixMapping** object.


### Return Value

String
