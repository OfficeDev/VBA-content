# LanguageSettings.Parent Property (Office)

Gets the  **Parent** object for the **ODSOFilter** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **LanguageSettings** object.


### Return Value

Object
