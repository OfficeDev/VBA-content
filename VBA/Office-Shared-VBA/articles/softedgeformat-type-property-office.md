# SoftEdgeFormat.Type Property (Office)

Gets or sets the type of the  **SoftEdgeFormat** object. Read/write.


## Syntax

 _expression_. **Type**

 _expression_ An expression that returns a **SoftEdgeFormat** object.


### Return Value

MsoSoftEdgeType
