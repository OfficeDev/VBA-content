# SharedWorkspaceFile.Delete Method (Office)

Deletes the current  **SharedWorkspaceFile** object.


 **Note**  Beginning with Microsoft Office 2010, this object or member has been deprecated and should not be used.


## Syntax

 _expression_. **Delete**

 _expression_ Required. A variable that represents a **[SharedWorkspaceFile](sharedworkspacefile-object-office.md)** object.
