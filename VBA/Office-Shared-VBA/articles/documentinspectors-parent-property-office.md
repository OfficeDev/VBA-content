# DocumentInspectors.Parent Property (Office)

Gets an object that represents the parent of a  **DocumentInspectors** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **DocumentInspectors** object.


### Return Value

Object
