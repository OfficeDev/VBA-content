# CustomXMLPrefixMappings.Parent Property (Office)

Gets the  **Parent** object for the **CustomXMLPrefixMappings** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **CustomXMLPrefixMappings** object.


### Return Value

Object
