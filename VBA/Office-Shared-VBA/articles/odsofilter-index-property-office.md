# ODSOFilter.Index Property (Office)

Gets a  **Long** representing the index number for an **ODSOFilter** object in the collection. Read-only.


## Syntax

 _expression_. **Index**

 _expression_ A variable that represents an **ODSOFilter** object.


### Return Value

Long
