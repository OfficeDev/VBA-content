# SearchFolders.Remove Method (Office)

Removes the specified object from the collection.


## Syntax

 _expression_. **Remove**( **_Index_** )

 _expression_ Required. A variable that represents a **[SearchFolders](searchfolders-object-office.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required|**Long**|The index number of the folder to be removed.|
