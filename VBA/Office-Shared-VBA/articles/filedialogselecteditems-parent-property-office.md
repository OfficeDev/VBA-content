# FileDialogSelectedItems.Parent Property (Office)

Gets the  **Parent** object for the **FileDialogSelectedItems** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **FileDialogSelectedItems** object.


### Return Value

Object
