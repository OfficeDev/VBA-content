# SearchFolders.Count Property (Office)

Gets a  **Long** indicating the number of items in the **SearchFolders** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **SearchFolders** object.


### Return Value

Long
