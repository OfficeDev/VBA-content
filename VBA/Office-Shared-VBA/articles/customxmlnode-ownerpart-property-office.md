# CustomXMLNode.OwnerPart Property (Office)

Gets the object representing the part associated with this node. Read-only.


## Syntax

 _expression_. **OwnerPart**

 _expression_ An expression that returns a **CustomXMLNode** object.


### Return Value

CustomXMLPart
