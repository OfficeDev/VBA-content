# FileDialogFilters.Delete Method (Office)

Removes a file dialog filter.


## Syntax

 _expression_. **Delete**( **_filter_** )

 _expression_ Required. A variable that represents a **[FileDialogFilters](filedialogfilters-object-office.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _filter_|Optional|**Variant**|The filter to be removed.|
