# CustomTaskPane.Visible Property (Office)

 **True** if the specified **CustomTaskPane** object is visible. Read/write.


## Syntax

 _expression_. **Visible**

 _expression_ An expression that returns a **CustomTaskPane** object.


### Return Value

Boolean
