# CustomXMLParts.Count Property (Office)

Gets a  **Long** indicating the number of items in the **CustomXMLParts** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ An expression that returns a **CustomXMLParts** object.


### Return Value

Long
