# CommandBarComboBox.Caption Property (Office)

Gets or sets the caption text for a command bar control. Read/write.


## 


 **Note**  The use of CommandBars in some Microsoft Office applications has been superseded by the new ribbon component of the Microsoft Office Fluent user interface. For more information, search Help for the keyword "ribbon."


## Syntax

 _expression_. **Caption**

 _expression_ A variable that represents a **CommandBarComboBox** object.


### Return Value

String


 **Note**  


 **Note**  A control's caption is also displayed as its default ScreenTip.
