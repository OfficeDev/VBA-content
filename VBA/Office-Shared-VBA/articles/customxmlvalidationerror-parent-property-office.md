# CustomXMLValidationError.Parent Property (Office)

Gets the  **Parent** object for the **CustomXMLValidationError** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **CustomXMLValidationError** object.


### Return Value

Object
