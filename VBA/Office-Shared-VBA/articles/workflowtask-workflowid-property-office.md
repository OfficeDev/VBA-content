# WorkflowTask.WorkflowID Property (Office)

Gets the ID of the workflow associated with a workflow task. Read-only.


## Syntax

 _expression_. **WorkflowID**

 _expression_ An expression that returns a **WorkflowTask** object.


### Return Value

String
