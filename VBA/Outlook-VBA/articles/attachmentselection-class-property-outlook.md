# AttachmentSelection.Class Property (Outlook)

Returns an  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** constant indicating the object's class. Read-only.


## Syntax

 _expression_ . **Class**

 _expression_ A variable that represents an **AttachmentSelection** object.
