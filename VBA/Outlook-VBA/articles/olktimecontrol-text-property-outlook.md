# OlkTimeControl.Text Property (Outlook)

Returns or sets a  **String** that is the text displayed in the control. Read/write.


## Syntax

 _expression_ . **Text**

 _expression_ A variable that represents an **OlkTimeControl** object.


## Remarks

The default value of the  **Text** property is the empty string.

The  **Text** property provides the same value as the **[Value](olktimecontrol-value-property-outlook.md)** property, but the value is strongly typed as a **String** .
