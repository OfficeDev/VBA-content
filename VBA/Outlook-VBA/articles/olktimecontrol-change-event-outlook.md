# OlkTimeControl.Change Event (Outlook)

Occurs when the  **[Value](olktimecontrol-value-property-outlook.md)** property changes.


## Syntax

 _expression_ . **Change**

 _expression_ A variable that represents an **OlkTimeControl** object.
