# MeetingItem.NoAging Property (Outlook)

Returns or sets a  **Boolean** value that is **True** to not age the Outlook item. Read/write.


## Syntax

 _expression_ . **NoAging**

 _expression_ A variable that represents a **MeetingItem** object.
