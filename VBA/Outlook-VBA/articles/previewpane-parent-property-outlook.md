# PreviewPane.Parent Property (Outlook)

Returns the parent property for the [PreviewPane](previewpane-object-outlook.md) Object. Read only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **PreviewPane** object.
