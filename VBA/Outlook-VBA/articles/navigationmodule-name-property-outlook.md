# NavigationModule.Name Property (Outlook)

Returns a  **String** value that represents the localized display name for the **[NavigationModule](navigationmodule-object-outlook.md)** object. Read-only.


## Syntax

 _expression_ . **Name**

 _expression_ An expression that returns a **NavigationModule** object.
