# MailItem.ReminderOverrideDefault Property (Outlook)

Returns or sets a  **Boolean** value that is **True** if the reminder overrides the default reminder behavior for the item. Read/write.


## Syntax

 _expression_ . **ReminderOverrideDefault**

 _expression_ A variable that represents a **MailItem** object.


## Remarks

You must set the  **ReminderOverrideDefault** property to validate the **[ReminderPlaySound](mailitem-reminderplaysound-property-outlook.md)** and the **[ReminderSoundFile](mailitem-remindersoundfile-property-outlook.md)** properties.
