# Application.DisplayGuides Property (PowerPoint)

Gets or sets whether drawing guides are displayed in the application. 


## Syntax

 _expression_. **DisplayGuides**

 _expression_ A variable that represents a **Application** object.


### Return Value

MsoTriState


## Property value

 **&lt;unknown type&gt;**
