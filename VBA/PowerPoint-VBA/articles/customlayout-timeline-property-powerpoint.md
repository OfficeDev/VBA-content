# CustomLayout.TimeLine Property (PowerPoint)

Returns a  **[TimeLine](timeline-object-powerpoint.md)** object that represents the animation timeline for the slide associated with the specified custom layout. Read-only.


## Syntax

 _expression_. **TimeLine**

 _expression_ A variable that represents a **CustomLayout** object.


### Return Value

TimeLine
