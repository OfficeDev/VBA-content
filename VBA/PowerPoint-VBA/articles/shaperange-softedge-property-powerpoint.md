# ShapeRange.SoftEdge Property (PowerPoint)

Returns the soft edge format for the specified range of shapes. Read-only.


## Syntax

 _expression_. **SoftEdge**

 _expression_ A variable that represents a **ShapeRange** object.


### Return Value

MsoSoftEdgeType
