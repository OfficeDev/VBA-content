# Font.Italic Property (PowerPoint)

 **True** if the font or range is formatted as italic. Read/write **Long**.


## Syntax

 _expression_. **Italic**

 _expression_ A variable that represents a **Font** object.


## Property value

 **&lt;unknown type&gt;**
