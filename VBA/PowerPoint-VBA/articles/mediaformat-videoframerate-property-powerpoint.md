# MediaFormat.VideoFrameRate Property (PowerPoint)

Returns the video frame rate per second of the media. Read-only.


## Syntax

 _expression_. **VideoFrameRate**

 _expression_ An expression that returns a **MediaFormat** object.


### Return Value

Long
