# DataLabel.Caption Property (PowerPoint)

Returns or sets the data label text. Read/write  **String**.


## Syntax

 _expression_. **Caption**

 _expression_ A variable that represents a **[DataLabel](datalabel-object-powerpoint.md)** object.
