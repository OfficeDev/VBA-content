# Presentation.Broadcast Property (PowerPoint)

Returns the  **Broadcast** object of the current **Presentation** object. Read-only.


## Syntax

 _expression_. **Broadcast**

 _expression_ A variable that represents a **Presentation** object.


### Return Value

 **Broadcast**
