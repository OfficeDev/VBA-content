# LegendKey.Delete Method (PowerPoint)

Deletes the object.


## Syntax

 _expression_. **Delete**

 _expression_ A variable that represents a **[LegendKey](legendkey-object-powerpoint.md)** object.


## Remarks

Deleting a  **LegendKey** object deletes the entire series.
