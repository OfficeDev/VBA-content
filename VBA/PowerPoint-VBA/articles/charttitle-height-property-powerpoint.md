# ChartTitle.Height Property (PowerPoint)

Returns or sets a  **Double** value that represents the height, in points, of the object. Read-only.


## Syntax

 _expression_. **Height**

 _expression_ A variable that represents a **ChartTitle** object.


### Return Value

Double
