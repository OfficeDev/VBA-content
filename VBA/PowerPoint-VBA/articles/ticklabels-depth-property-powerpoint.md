# TickLabels.Depth Property (PowerPoint)

Returns the number of levels of category tick labels. Read-only  **Long**.


## Syntax

 _expression_. **Depth**

 _expression_ A variable that represents a **[TickLabels](ticklabels-object-powerpoint.md)** object.
