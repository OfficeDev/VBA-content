# CategoryCollection.Count Property (PowerPoint)

Returns a  **Long** that represents the number of categories in the collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **CategoryCollection** object.


## Property value

 **INT32**
