# DataTable.Border Property (PowerPoint)

Returns the border of the object. Read-only  **[ChartBorder](chartborder-object-powerpoint.md)**.


## Syntax

 _expression_. **Border**

 _expression_ A variable that represents a **[DataTable](datatable-object-powerpoint.md)** object.
