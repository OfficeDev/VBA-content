# ResampleMediaTask.SampleWidth Property (PowerPoint)

Returns the resolution width of the sample. Read-only.


## Syntax

 _expression_. **SampleWidth**

 _expression_ A variable that represents a **ResampleMediaTask** object.


### Return Value

Long
