# HeaderFooter.Text Property (PowerPoint)

Returns or sets a  **String** that represents the text contained in the specified object. Read/write.


## Syntax

 _expression_. **Text**

 _expression_ A variable that represents a **HeaderFooter** object.


### Return Value

String
