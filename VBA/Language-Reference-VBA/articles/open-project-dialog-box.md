# Open Project Dialog Box


![Open project dialog box](images/va24w61_ZA01201774.gif)



Lets you locate and open a standalone project.

## Dialog Box Options

 **Open**

Displays the selected file.

 **Cancel**

Closes the dialog box without opening a new project.
