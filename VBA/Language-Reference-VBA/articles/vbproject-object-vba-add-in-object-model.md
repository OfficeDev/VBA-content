# VBProject Object (VBA Add-In Object Model)



Represents a [project](vbe-glossary.md).
 **Remarks**
Use the  **VBProject** object to set[properties](vbe-glossary.md) for the project, to access the **VBComponents** collection, and to access the **References** collection.
