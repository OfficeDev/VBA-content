
# WorkflowTemplate.Id Property (Office)

 **Last modified:** July 28, 2015

Gets the ID of a template used to create a workflow instance. Read-only.

## Syntax

 _expression_. **Id**

 _expression_An expression that returns a  **WorkflowTemplate** object.


### Return Value

String
