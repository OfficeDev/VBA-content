
# BulletFormat2.Type Property (Office)

 **Last modified:** July 28, 2015

Gets or sets a constant that represents the type of bullet. Read/write.

## Syntax

 _expression_. **Type**

 _expression_An expression that returns a  **BulletFormat2** object.


### Return Value

MsoBulletType
