
# FileTypes.Item Property (Office)

 **Last modified:** July 28, 2015

Gets a value that indicates which file type will be searched for. Read-only.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [FileTypes](5e8b5240-5ebd-704d-72e6-1f4ad951dfdc.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The index number of the object to be returned.|
