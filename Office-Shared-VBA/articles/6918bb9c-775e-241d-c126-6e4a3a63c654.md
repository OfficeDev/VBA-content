
# SignatureSet.Count Property (Office)

 **Last modified:** July 28, 2015

Gets a  **Long** indicating the number of items in the **SignatureSet** object. Read-only.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **SignatureSet** object.


### Return Value

Long
