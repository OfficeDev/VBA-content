
# CommandBarControls.Item Property (Office)

 **Last modified:** July 28, 2015

Gets a  **CommandBarControl** object from the **CommandBarControls** collection. Read-only.

## 


 **Note**  The use of CommandBars in some Microsoft Office applications has been superseded by the new ribbon component of the Microsoft Office Fluent user interface. For more information, search Help for the keyword "ribbon."


## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [CommandBarControls](7ccae243-2870-95c2-1e08-140a3e638fe6.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the object to be returned.|
