
# TextRange2.Text Property (Office)

 **Last modified:** July 28, 2015

Gets or sets a  **String** value that represents the text in a text range. Read/write.

## Syntax

 _expression_. **Text**

 _expression_An expression that returns a  **TextRange2** object.


### Return Value

String
