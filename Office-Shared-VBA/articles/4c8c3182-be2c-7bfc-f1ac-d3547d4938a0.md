
# ODSOFilters.Parent Property (Office)

 **Last modified:** July 28, 2015

Gets the  **Parent** object for the **ODSOFilters** object. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents an  **ODSOFilters** object.


### Return Value

Object
