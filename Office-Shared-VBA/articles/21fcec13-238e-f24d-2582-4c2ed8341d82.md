
# ServerPolicy.Item Property (Office)

 **Last modified:** July 28, 2015

Gets a  **PolicyItem** object from the **ServerPolicy** collection. Read-only.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns a  **ServerPolicy** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the  **PolicyItem** object to be returned.|

### Return Value

PolicyItem
