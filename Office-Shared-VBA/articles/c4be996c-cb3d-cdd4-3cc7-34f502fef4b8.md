
# PickerResult.Fields Property (Office)

 **Last modified:** July 28, 2015

Represent field definitions of SubItems in a PickerFields collection. Read-only

## Syntax

 _expression_. **Fields**

 _expression_An expression that returns a  **PickerResult** object.
