
# SearchFolders.Remove Method (Office)

 **Last modified:** July 28, 2015

Removes the specified object from the collection.

## Syntax

 _expression_. **Remove**( **_Index_**)

 _expression_Required. A variable that represents a  ** [SearchFolders](5958cafc-880e-ee9f-b2f5-be463bfe5232.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The index number of the folder to be removed.|
