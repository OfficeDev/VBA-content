
# ThemeFonts.Count Property (Office)

 **Last modified:** July 28, 2015

Gets a  **Long** indicating the number of items in the **ThemeFonts** collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_An expression that returns a  **ThemeFonts** object.


### Return Value

Integer
