
# Font2.Line Property (Office)

 **Last modified:** July 28, 2015

Gets a value specifiying the format of a line. Read-only.

## Syntax

 _expression_. **Line**

 _expression_An expression that returns a  **Font2** object.


### Return Value

LineFormat
