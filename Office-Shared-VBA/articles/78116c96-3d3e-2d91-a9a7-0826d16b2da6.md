
# DocumentInspectors.Count Property (Office)

 **Last modified:** July 28, 2015

Gets the number of items in the  **DocumentInspectors** object. Read-only.

## Syntax

 _expression_. **Count**

 _expression_An expression that returns a  **DocumentInspectors** object.


### Return Value

Integer
