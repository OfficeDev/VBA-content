
# SmartArtColors.Item Method (Office)

 **Last modified:** July 28, 2015

Retrieves a  **SmartArtColor** object at the specified index or with the specified unique Id.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns a  **SmartArtColors** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|Specifies either an integer representing the index or a string representing the Id of the Smart Art color.|

### Return Value

SmartArtColor
