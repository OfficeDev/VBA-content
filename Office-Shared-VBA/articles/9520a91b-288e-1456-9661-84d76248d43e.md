
# SignatureSetup.Id Property (Office)

 **Last modified:** July 28, 2015

Gets the ID of the signature provider for a document. Read-only.

## Syntax

 _expression_. **Id**

 _expression_An expression that returns a  **SignatureSetup** object.


### Return Value

String
