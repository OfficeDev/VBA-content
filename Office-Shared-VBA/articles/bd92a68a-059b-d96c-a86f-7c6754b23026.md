
# PictureEffects.Item Property (Office)

 **Last modified:** July 28, 2015

Retrieves a  **PictureEffect** object at the specified index. Read-only

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns a  **PictureEffects** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Integer**|Specifies an integer representing the indexed location of the  **PictureEffect** object.|
