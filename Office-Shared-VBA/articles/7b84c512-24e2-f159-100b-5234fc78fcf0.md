
# CommandBarComboBox.Delete Method (Office)

 **Last modified:** July 28, 2015

Deletes  **CommandBarCombo** control object from its collection.

## Syntax

 _expression_. **Delete**( **_Temporary_**)

 _expression_Required. A variable that represents a  ** [CommandBarComboBox](fcfe6bde-dea0-f1f1-ad30-d0e28f97dd07.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Temporary|Optional| **Variant**| **True** to delete the control for the current session. The application will display the control again in the next session.|
