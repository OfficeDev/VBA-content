
# SharedWorkspace.Delete Method (Office)

 **Last modified:** July 28, 2015

Deletes the current shared workspace and all data within it.

 **Note**  Beginning with Microsoft Office 2010, this object or member has been deprecated and should not be used.


## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents a  ** [SharedWorkspace](7512f0ff-382d-d344-9424-aa10549d14f9.md)** object.
