
# COMAddIns.Count Property (Office)

 **Last modified:** July 28, 2015

Gets a count of the number of COM add-ins in the host application. Read-only.

## Syntax

 _expression_. **Count**

 _expression_An expression that returns a  **COMAddIns** object.


### Return Value

Long
