
# Permission.Count Property (Office)

 **Last modified:** July 28, 2015

Gets a  **Long** indicating the number of items in the **Permission** object. Read-only.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **Permission** object.


### Return Value

Long
