
# ParagraphFormat2.Bullet Property (Office)

 **Last modified:** July 28, 2015

Gets a  **BulletFormat2** object for the paragraph. Read-only.

## Syntax

 _expression_. **Bullet**

 _expression_An expression that returns a  **ParagraphFormat2** object.
