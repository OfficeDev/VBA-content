
# DocumentLibraryVersions.Item Property (Office)

 **Last modified:** July 28, 2015

Gets a  **DocumentLibraryVersion** object from the **DocumentLibraryVersions** collection. Read-only.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [DocumentLibraryVersions](075c0315-fade-6d45-9ab9-6c798f6f09ac.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The index number of the DocumentLibraryVersion returned.|
