
# FileDialogFilters.Parent Property (Office)

 **Last modified:** July 28, 2015

Gets the  **Parent** object for the **FileDialogFilters** object. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **FileDialogFilters** object.


### Return Value

Object
