
# ODSOColumn.Name Property (Office)

 **Last modified:** July 28, 2015

Gets the name of a data field in a mail merge data source. Read-only.

## Syntax

 _expression_. **Name**

 _expression_A variable that represents an  **ODSOColumn** object.


### Return Value

String
