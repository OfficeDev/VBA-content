
# CustomXMLPrefixMappings.Item Property (Office)

 **Last modified:** July 28, 2015

Gets a  **CustomXMLPrefixMapping** object from the **CustomXMLPrefixMappings** collection. Read-only.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns a  **CustomXMLPrefixMappings** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the  **CustomXMLPrefixMapping** object to be returned.|

### Return Value

CustomXMLPrefixMapping
