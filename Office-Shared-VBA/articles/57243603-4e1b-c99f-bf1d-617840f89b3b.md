
# UserPermission.Parent Property (Office)

 **Last modified:** July 28, 2015

Gets the  **Parent** object for the **UserPermission** object. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **UserPermission** object.


### Return Value

Object
