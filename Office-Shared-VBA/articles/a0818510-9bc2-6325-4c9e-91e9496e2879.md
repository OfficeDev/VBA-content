
# OfficeTheme.Parent Property (Office)

 **Last modified:** July 28, 2015

Gets the  **Parent** object for the **OfficeTheme** object. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_An expression that returns a  **OfficeTheme** object.


### Return Value

Object
