
# CommandBarControls.Count Property (Office)

 **Last modified:** July 28, 2015

Gets a count of the numbers of controls on a command bar. Read-only.

## Syntax

 _expression_. **Count**

 _expression_An expression that returns a  **CommandBarControls** object.


### Return Value

Integer
