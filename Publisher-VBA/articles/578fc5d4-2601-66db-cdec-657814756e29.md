
# Printer.IsActivePrinter Property (Publisher)

 **Last modified:** July 28, 2015

Gets or sets the printer represented by the parent  ** [Printer](46f8c6a2-4cf1-bb6a-1214-a751440870f2.md)** object as the active printer. Read/write.

## Syntax

 _expression_. **IsActivePrinter**

 _expression_A variable that represents a  **Printer** object.


### Return Value

Boolean
