
# Shape.Glow Property (Publisher)

 **Last modified:** July 28, 2015

Returns a  [GlowFormat](62fd5b2a-f199-588e-c15e-ea27c5b59699.md) object that represents the glow formatting for a shape. Read-only.

## Syntax

 _expression_. **Glow**

 _expression_A variable that represents a  **Shape** object.


## Return value

 **GLOWFORMAT**
