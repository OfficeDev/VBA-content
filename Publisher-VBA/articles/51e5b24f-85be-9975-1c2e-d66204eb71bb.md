
# BorderArt Members (Publisher)
Represents an available type of BorderArt. BorderArt is picture borders that can be applied to text boxes, picture frames, or rectangles. The  **BorderArt** object is a member of the ** [BorderArts](0fc016f6-154e-3591-34b3-e094bbad9d16.md)** collection. The **BorderArts** collection contains all BorderArt available for use in the specified publication.

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](ecdd7a8a-9f3b-9cd3-9454-648e0be6f42e.md)|Used without an object qualifier, this property returns an  ** [Application](acfc7efb-e6a5-a89a-3aee-3cb4af2f3508.md)**object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [Name](71e68f08-6bf1-7f4d-5da2-9f4ce4f63021.md)|Returns a  **String** value indicating the name of the specified object. Read-only.|
| [Parent](cf2ab5cd-ed49-a9cf-decd-6319613668bf.md)|Returns an object that represents the parent object of the specified object. For example, for a  ** [TextFrame](95e88f5a-b3dc-272e-7c1d-5282c97ae11e.md)** object, returns a ** [Shape](666cb7f0-62a8-f419-9838-007ef29506ee.md)** object representing the parent shape of the text frame. Read-only.|
