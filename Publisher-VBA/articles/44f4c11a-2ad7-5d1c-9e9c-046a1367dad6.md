
# Attachments.Count Property (Publisher)

 **Last modified:** July 28, 2015

Returns the number of  ** [Attachment](d617bdf6-b0ba-be0d-0f72-f729010636c1.md)** objects in the ** [Attachments](61957961-8c75-992f-159c-51412ed309ea.md)** collection of an ** [EmailMergeEnvelope](555dd80e-bac2-96dd-4256-ad1b8006da0f.md)** object. Read-only.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents an  **Attachments** collection.


### Return Value

Long
