
# Field Members (Publisher)
Represents a field. The  **Field** object is a member of the ** [Fields](fd7c95d9-bc34-95ee-180d-b99f3629eb33.md)**collection. The  **Fields** collection represents the fields in a selection, range, or publication.

 **Last modified:** July 28, 2015


## Methods



|**Name**|**Description**|
|:-----|:-----|
| [Unlink](4dfe5c29-eb1e-b071-fd86-6ee222455c4e.md)|Replaces the specified field or  ** [Fields](fd7c95d9-bc34-95ee-180d-b99f3629eb33.md)** collection with with their most recent results.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](3250d161-b88c-8e2e-34e1-65ca42368bd6.md)|Used without an object qualifier, this property returns an  ** [Application](acfc7efb-e6a5-a89a-3aee-3cb4af2f3508.md)**object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [Code](bb2f3b23-dea1-bdfb-90bf-4b4ea09570f6.md)|Returns a  **String** that represents the text displayed when the page view is set to show field codes. Read-only.|
| [Next](a8f0a246-c55e-715e-3f97-a2f08c383e87.md)|Returns a  ** [Field](93da311a-b834-f990-60e9-786d4f6a16f1.md)** object that represents the next field in a text range.|
| [Parent](8ae30944-be11-03a6-61d7-3cbb93758e5c.md)|Returns an object that represents the parent object of the specified object. For example, for a  ** [TextFrame](95e88f5a-b3dc-272e-7c1d-5282c97ae11e.md)** object, returns a ** [Shape](666cb7f0-62a8-f419-9838-007ef29506ee.md)** object representing the parent shape of the text frame. Read-only.|
| [PhoneticGuide](c68dba00-56c0-abba-0be8-5bd1a725f678.md)|Returns a  **PhoneticGuide** object that represents the properties of phonetic text applied to a text range.|
| [Result](213e123e-90a7-32b8-1dcf-37da61a8a7e7.md)|Returns a  **String** that represents the result of the specified field. Read-only.|
| [TextRange](09279cc7-3911-3b8d-51f2-b26494220c68.md)|Returns a  ** [TextRange](566f240b-d2a6-8cb3-9eb7-68328d6c28bd.md)**object that represents the text that is attached to a shape and properties and methods for manipulating the text.|
| [Type](5a875159-e9a6-0fae-5ca9-cd88ea0cda10.md)|Specifies the field type. Read-only.|
