
# ValidationIssues.Document Property (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Returns the  ** [Document](21640062-13a2-a2b2-7c61-7e707671207c.md)** object that is associated with an object. Read-only.


## Syntax

 _expression_. **Document**

 _expression_A variable that represents a  ** [ValidationIssues](13362aa2-7e09-14ed-8aa9-bf2a93edf302.md)** object.


### Return Value

 **Document**
