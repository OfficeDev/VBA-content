
# MovedSelectionEvent.SelectionSource Property (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Returns a  ** [Selection](e5734140-6dbe-7de8-9695-1a22fb4ac628.md)** object that represents the set of selected shapes before they are moved to a sub-process page. Read-only.


## Syntax

 _expression_. **SelectionSource**

 _expression_A variable that represents a  ** [MovedSelectionEvent](e29b0f23-08b6-7c63-9955-4a989c6e6892.md)** object.


### Return Value

 **Selection**
