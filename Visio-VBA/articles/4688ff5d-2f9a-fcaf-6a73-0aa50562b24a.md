
# Master.Type Property (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Returns the type of the  **Master** object. Read-only.


## Syntax

 _expression_. **Type**

 _expression_A variable that represents a  **Master** object.


### Return Value

VisMasterTypes


## Remarks

Type value constants for  **Master** objects (the possible values that the **Type** property of a **Master** object returns) are declared by the Visio type library in ** [VisMasterTypes](02949efd-bcf5-eacf-8ffc-73f1b8c3016d.md)**.
