
# ShapeRange.Height Property (Project)
Gets or sets the height of the shape range, in points. Read-write  **Single**.

 **Last modified:** July 28, 2015


## Syntax

 _expression_. **Height**

 _expression_A variable that represents a  **ShapeRange** object.


## Property value

 **FLOAT**
