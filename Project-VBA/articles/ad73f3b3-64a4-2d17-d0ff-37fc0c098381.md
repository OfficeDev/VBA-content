
# WorkWeek.Index Property (Project)

 **Last modified:** July 28, 2015

Gets the index of a  **WorkWeek** object in the **WorkWeeks** containing object. Read-only **Long**.

## Syntax

 _expression_. **Index**

 _expression_An expression that returns a  **WorkWeek** object.


## Remarks

The  **Index** properties of different objects are used in similar ways. For an example, see the ** [Index](1213f55b-aca0-76ee-2e8a-2442a2c576e1.md)** property of the **Project** object.
