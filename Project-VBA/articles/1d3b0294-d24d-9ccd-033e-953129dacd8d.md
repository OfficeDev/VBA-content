
# Year.Working Property (Project)

 **Last modified:** July 28, 2015

 **True** if any day in the year is a working day. Read/write **Boolean**.

## Syntax

 _expression_. **Working**

 _expression_A variable that represents a  **Year** object.

For an example that uses the  **Month** object, see ** [Working](6fa33218-2cf0-dbe4-af31-514c7c83a047.md)**.
