
# CodeMask.Count Property (Project)

 **Last modified:** July 28, 2015

Gets the number of items in the  **CodeMask** object. Read-only **Long**.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **CodeMask** object.


## Remarks

Use of the  **Count** property in most collection objects is similar. For an example, see the ** [Assignments.Count](de8cee7b-a471-4c3f-2965-de6c3d47dda5.md)** property.
