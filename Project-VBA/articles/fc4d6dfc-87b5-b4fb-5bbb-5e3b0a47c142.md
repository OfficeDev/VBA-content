
# Groups2.Add Method (Project)

 **Last modified:** July 28, 2015

Adds a  ** [Group2](a7a61fa4-e752-006e-a47e-03987b04f01c.md)** object to a **Groups2** collection.

## Syntax

 _expression_. **Add**( **_Name_**,  **_FieldName_**)

 _expression_An expression that returns a  **Groups2** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Name|Required| **String**|The name of the group.|
|FieldName|Required| **String**|The name of the field being grouped by.|

### Return Value

 **Group2**
