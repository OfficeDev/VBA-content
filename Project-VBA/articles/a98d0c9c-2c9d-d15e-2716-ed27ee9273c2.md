
# Subproject.InsertedProjectSummary Property (Project)

 **Last modified:** July 28, 2015

Gets a  ** [Task](bc6bb4a5-95a6-9d1f-3e28-92b9548a544a.md)** object that represents the summary task of the subproject in a master project. Read-only **Task**.

## Syntax

 _expression_. **InsertedProjectSummary**

 _expression_A variable that represents a  **Subproject** object.
