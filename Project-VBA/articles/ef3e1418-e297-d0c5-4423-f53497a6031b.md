
# Exception.Name Property (Project)

 **Last modified:** July 28, 2015

Gets the name of an  **Exception** object. Read/write **String**.

## Syntax

 _expression_. **Name**

 _expression_A variable that represents an  **Exception** object.


## Remarks

 **Name** is the default property of the **Exception** object.

For a code example that uses the  **Task** object, see ** [Name](2df034b0-13bc-f912-abbc-6b97b8c8d5ed.md)**.
