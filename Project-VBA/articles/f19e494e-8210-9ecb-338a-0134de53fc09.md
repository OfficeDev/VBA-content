
# ShapeRange.Width Property (Project)
Gets or sets the width, in points, of the shapes within the range. Read/write  **Long**.

 **Last modified:** July 28, 2015


## Syntax

 _expression_. **Width**

 _expression_A variable that represents a  **ShapeRange** object.


## Property value

 **FLOAT**
