
# Shape.Chart Property (Project)
Gets a  **Chart** object that represents the chart contained in the shape. Read-only **Chart**.

 **Last modified:** July 28, 2015


## Syntax

 _expression_. **Chart**

 _expression_A variable that represents a  **Shape** object.


## Property value

 **CHART**
