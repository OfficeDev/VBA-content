
# Shape.Width Property (Project)
Gets or sets the width, in points, of the shape. Read/write  **Long**.

 **Last modified:** July 28, 2015


## Syntax

 _expression_. **Width**

 _expression_A variable that represents a  **Shape** object.


## Property value

 **FLOAT**
