
# Shape.Rotation Property (Project)
Gets or sets the rotation of the shape, in degrees. Read/write  **Single**.

 **Last modified:** July 28, 2015


## Syntax

 _expression_. **Rotation**

 _expression_A variable that represents a  **Shape** object.


## Property value

 **FLOAT**
