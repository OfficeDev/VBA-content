
# Groups2.Count Property (Project)

 **Last modified:** July 28, 2015

Gets the number of  **Group2** objects in the **Groups2** collection. Read-only **Long**.

## Syntax

 _expression_. **Count**

 _expression_An expression that returns a  **Groups2** object.


## Remarks

Use of the  **Count** property in most collection objects is similar. For an example, see the ** [Assignments.Count](de8cee7b-a471-4c3f-2965-de6c3d47dda5.md)** property.
