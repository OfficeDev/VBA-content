
# Exception.Shift2 Property (Project)

 **Last modified:** July 28, 2015

Gets a  ** [Shift](bf224646-d1c6-bc4a-1cce-a08b2f4e417d.md)** object representing the second work shift in a calendar exception for a day, month, period, weekday, or throughout a year. Read-only **Shift**.

## Syntax

 _expression_. **Shift2**

 _expression_A variable that represents an  **Exception** object.


## Example

The following example schedules a half-day of work on Fridays by creating a shift from 8 A.M. to noon.


```
Sub HalfDayFridays() 

 

 With ActiveProject.Calendar.WeekDays(pjFriday) 

 .Shift1.Start = #8:00:00 AM# 

 .Shift1.Finish = #12:00:00 PM# 

 .Shift2.Clear 

 .Shift3.Clear 

 End With 

 

End Sub
```
