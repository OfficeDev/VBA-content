
# Resources.UniqueID Property (Project)

 **Last modified:** July 28, 2015

Gets a  ** [Resource](eb83ed2f-2415-3f5d-3856-f4451a73a128.md)** object from its containing collection. Read-only **Resource**.

## Syntax

 _expression_. **UniqueID**( **_Index_**)

 _expression_A variable that represents a  **Resources** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The position of the object within its containing collection.|
