
# Tables.Copy Method (Project)

 **Last modified:** July 28, 2015

Makes a copy of a group definition for the  **Tables** collection and returns a reference to the ** [Table](f50f5d2d-a733-c5b0-16d8-e4ee98943321.md)** object.

## Syntax

 _expression_. **Copy**( **_Source_**,  **_NewName_**)

 _expression_A variable that represents a  **Tables** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Source|Required| **String**|The name of the table to copy.|
|NewName|Required| **String**|The name of the new table.|

### Return Value

 **Table**
