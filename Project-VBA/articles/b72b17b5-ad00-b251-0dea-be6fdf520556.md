
# CostRateTable.Name Property (Project)

 **Last modified:** July 28, 2015

Gets the name of a  **CostRateTable** object. Read-only **String**.

## Syntax

 _expression_. **Name**

 _expression_A variable that represents a  **CostRateTable** object.
