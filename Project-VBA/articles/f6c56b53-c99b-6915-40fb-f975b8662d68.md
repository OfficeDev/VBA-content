
# WorkWeek.Start Property (Project)

 **Last modified:** July 28, 2015

Gets or sets the start date of the work week. Read/write  **Variant**.

## Syntax

 _expression_. **Start**

 _expression_An expression that returns a  **WorkWeek** object.
