
# Sources.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Sources** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [Sources](92a74894-1c69-0ff1-749d-baa49b45a174.md)** object.
