
# TaskPane.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **TaskPane** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [TaskPane](57367e56-2de5-37bd-a9ca-f1fcb6b8c465.md)** object.
