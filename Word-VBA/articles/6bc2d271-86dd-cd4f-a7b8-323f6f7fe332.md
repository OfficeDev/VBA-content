
# Chart.SetBackgroundPicture Method (Word)

 **Last modified:** July 28, 2015

Sets the background graphic for a chart.

## Syntax

 _expression_. **SetBackgroundPicture**( **_FileName_**)

 _expression_A variable that represents a  ** [Chart](366a825e-0daf-dbb7-b6f2-e7ce1a5ee2ef.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|FileName|Required| **String**|The name of the file for the graphic.|
