
# SpellingSuggestions.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **SpellingSuggestions** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [SpellingSuggestions](7e0fb008-e43c-c4cb-b7d2-9436d039a070.md)** collection.
