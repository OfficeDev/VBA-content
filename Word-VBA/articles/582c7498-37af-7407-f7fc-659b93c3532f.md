
# InlineShapes.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **InlineShape** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents an  ** [InlineShapes](88c632b2-80de-c96a-8879-a98461b38bd0.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

InlineShape
