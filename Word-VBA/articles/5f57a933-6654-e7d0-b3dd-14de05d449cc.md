
# MappedDataFields.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **MappedDataFields** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [MappedDataFields](d67de1fb-f495-ff4a-f21d-fd165a96232c.md)** collection.
