
# Endnotes.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Endnotes** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [Endnotes](32676579-dd41-e83d-a305-fcc2b7cb4f64.md)** collection.
