
# PageSetup.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **PageSetup** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [PageSetup](1879d601-80ad-4fc0-1a87-92e999b59f88.md)** object.
