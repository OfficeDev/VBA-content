
# FirstLetterExceptions.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **FirstLetterException** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [FirstLetterExceptions](5dc5cc43-a696-d80f-58f9-0f74dfcad0ed.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The individual object to be returned. Can be a  **Long** indicating the ordinal position or a **String** representing the name of the individual object.|

### Return Value

FirstLetterException
