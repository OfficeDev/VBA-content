
# LegendEntry.Font Property (Word)

 **Last modified:** July 28, 2015

Returns the font of the specified object. Read-only  ** [ChartFont](2ca7fb97-fa22-dec1-6978-8ebb6d8aad7c.md)**.

## Syntax

 _expression_. **Font**

 _expression_A variable that represents a  ** [LegendEntry](9f793578-cb9b-faa1-f0a1-ea0f9e90dc6f.md)** object.
