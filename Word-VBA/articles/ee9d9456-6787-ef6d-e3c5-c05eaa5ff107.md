
# ReadabilityStatistic.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ReadabilityStatistic** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ReadabilityStatistic](5e82c44d-fc6d-9586-816b-0c46c4a01f3b.md)** object.
