
# Subdocument.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified subdocument.

## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents a  ** [Subdocument](ed966369-34f6-ef0c-d6d6-4c86baff4793.md)** object.
