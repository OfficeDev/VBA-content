
# HTMLDivision.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **HTMLDivision** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [HTMLDivision](a38918ed-61aa-3fd1-3522-d077f1ff312f.md)** object.
