
# Editor.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Editor** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [Editor](af0c80f5-8c8a-be0e-4475-d3b3b3bacd0d.md)** object.
