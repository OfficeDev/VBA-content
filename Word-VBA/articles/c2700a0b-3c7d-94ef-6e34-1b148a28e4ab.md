
# EmailAuthor.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **EmailAuthor** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [EmailAuthor](2749e018-42e9-7a1a-f18b-8605b38ff0ae.md)** object.
