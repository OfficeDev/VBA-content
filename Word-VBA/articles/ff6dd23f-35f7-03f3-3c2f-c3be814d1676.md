
# AddIns.Count Property (Word)

 **Last modified:** July 28, 2015

Returns the number of  ** [AddIn](5615a8a9-1fd6-04fa-1fee-ec16502bd84a.md)** objects in the **AddIns** collection. Read-only **Long**.

## Syntax

 _expression_. **Count**

 _expression_An expression that returns an  ** [AddIns](acf58e58-d3f6-23cf-677b-4780f7cbc24d.md)** collection.
