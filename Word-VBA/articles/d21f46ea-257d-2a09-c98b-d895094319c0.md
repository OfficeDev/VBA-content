
# Frames.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified Frames collection.

## Syntax

 _expression_. **Delete**

 _expression_An expression that returns a  ** [Frames](d0f526b5-ae1d-ad7a-0da3-5a7b30526b55.md)** object.


### Return Value

Nothing
