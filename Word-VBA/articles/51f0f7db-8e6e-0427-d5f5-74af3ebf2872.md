
# Variables.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of variables in the collection.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Variables](9719d0d4-319d-c710-d243-12a9dee45880.md)** collection.
