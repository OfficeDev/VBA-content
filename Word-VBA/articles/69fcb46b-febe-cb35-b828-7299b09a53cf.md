
# Cells.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Cells** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [Cells](ceaa5b45-518d-d6ea-1ce8-5a34f6e37046.md)** object.
