
# WebOptions.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **WebOptions** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [WebOptions](658ae89d-3f92-067b-1309-7fc90b257111.md)** collection.
