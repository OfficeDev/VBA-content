
# OMathScrSubSup.Sup Property (Word)

 **Last modified:** July 28, 2015

Returns an  ** [OMath](82f2f81b-e2d5-140f-bdcc-8b52b821b24d.md)** object that represents the superscript for a subscript-superscript object. Read-only.

## Syntax

 _expression_. **Sup**

 _expression_An expression that returns an  ** [OMathScrSubSup](e5fbf9cb-461c-3b08-a441-9e91e0745b15.md)** object.
