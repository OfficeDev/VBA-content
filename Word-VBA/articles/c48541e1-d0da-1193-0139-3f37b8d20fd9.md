
# Rows.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified table rows.

## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents a  ** [Rows](cd83d0ef-f743-1886-54de-497017c5f542.md)** collection.
