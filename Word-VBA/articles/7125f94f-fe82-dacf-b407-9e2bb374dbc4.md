
# Shape.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified shape node.

## Syntax

 _expression_. **Delete**( **_Index_**)

 _expression_Required. A variable that represents a  ** [Shape](604029ce-9b2f-9748-5d4e-b458796fa2f0.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The number of the shape node to delete.|
