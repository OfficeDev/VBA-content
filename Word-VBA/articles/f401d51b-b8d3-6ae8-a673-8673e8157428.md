
# Variable.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Variable** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Variable](e6a75f54-6f91-75b4-7ca0-9be302e8dbe0.md)** object.
