
# PageNumbers.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of page numbers in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [PageNumbers](9090f96e-d898-ace6-35fa-f6e59c527ea2.md)** collection.
