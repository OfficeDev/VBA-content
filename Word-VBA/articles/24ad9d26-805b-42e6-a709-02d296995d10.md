
# ErrorBars Properties (Word)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](1a570155-974f-dd5d-98fd-c169f3acbbe1.md)|When used without an object qualifier, returns an  ** [Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)**object that represents the Microsoft Word application. When used with an object qualifier, returns an  **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
| [Border](8ec5857a-c056-be40-8580-e1dd1c2b885b.md)|Returns the border of the object. Read-only  ** [ChartBorder](eea90670-c599-2ec8-5b7b-c946a4bcd638.md)**.|
| [Creator](c8052d99-6816-1345-c1ab-14e5816fcc69.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
| [EndStyle](e0396671-1c83-c844-2ec3-e205ffda6ddf.md)|Returns or sets the end style for the error bars. Read/write  **Long**.|
| [Format](5c897701-361f-c7cc-8552-1ff5799ea90a.md)|Returns the line, fill, and effect formatting for the object. Read-only  ** [ChartFormat](5f6546e8-c2fd-eec5-27a9-f2fd2c058f16.md)**.|
| [Name](1a99484e-e19c-5bd9-af33-eb2fd1f49cee.md)|Returns the name of the object. Read-only  **String**.|
| [Parent](96f2f8bc-cdcd-b5b8-9141-8e13959cec85.md)|Returns the parent for the specified object. Read-only  **Object**.|
