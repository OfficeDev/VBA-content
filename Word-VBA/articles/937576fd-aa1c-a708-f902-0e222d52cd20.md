
# AutoCaptions.Count Property (Word)

 **Last modified:** July 28, 2015

Returns the number of items in the  **AutoCaptions** collection. Read-only **Long**.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  ** [AutoCaptions](da4bd001-8f4c-28c9-4f46-a5a6499000a8.md)** object.
