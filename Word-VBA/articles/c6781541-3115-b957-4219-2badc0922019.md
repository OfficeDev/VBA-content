
# LegendKey.Smooth Property (Word)

 **Last modified:** July 28, 2015

 **True** if curve smoothing is turned on for the legend key. Read/write **Boolean**.

## Syntax

 _expression_. **Smooth**

 _expression_A variable that represents a  ** [LegendKey](07578528-3e73-7898-47dc-296aefb854f0.md)** object.
