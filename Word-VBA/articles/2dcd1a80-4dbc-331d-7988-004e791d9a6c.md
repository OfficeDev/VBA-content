
# Conflicts.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Conflicts** object.

## Syntax

 _expression_. **Parent**

 _expression_An expression that returns a  **Conflicts** object.
