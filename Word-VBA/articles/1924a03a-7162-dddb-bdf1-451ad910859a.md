
# TextColumns.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **TextColumns** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [TextColumns](00b62c93-db7d-00b9-cc84-9a21e427d0cd.md)** collection.
