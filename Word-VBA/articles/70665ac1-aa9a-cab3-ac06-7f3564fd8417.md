
# OMathAcc.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathAcc** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathAcc](9458cc50-5764-8b1a-9939-56bf5118c9ce.md)** object.
