
# Bookmarks Methods (Word)

## Methods



|**Name**|**Description**|
|:-----|:-----|
| [Add](647795da-d7e2-7b6f-c412-5b684ec962a2.md)|Returns a  **Bookmark** object that represents a bookmark added to a range.|
