
# Template.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Template** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Template](47d1d92d-bba9-3f2a-9c71-22ac43159bd3.md)** object.
