
# Documents.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Documents** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Documents](fc4ac973-19c1-703a-5538-f4426b8b7564.md)** collection.
