
# DefaultWebOptions.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **DefaultWebOptions** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [DefaultWebOptions](7459af1e-c495-f84f-929c-f7a611ec49b3.md)** collection.
