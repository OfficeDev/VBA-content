
# OMathBar.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathBar** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathBar](9eebeeb0-c136-1652-767e-188470529193.md)** object.
