
# Window.Top Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the vertical position of the specified document window, in points. Read/write  **Long**.

## Syntax

 _expression_. **Top**

 _expression_Required. A variable that represents a  ** [Window](d92f83f9-ae44-56c0-4584-7a9359253c6d.md)** object.
