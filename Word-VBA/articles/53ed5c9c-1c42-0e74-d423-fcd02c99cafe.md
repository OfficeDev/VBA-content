
# ConditionalStyle.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ConditionalStyle** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ConditionalStyle](2380494e-09e9-8494-a93c-8bbaf621aad1.md)** object.
