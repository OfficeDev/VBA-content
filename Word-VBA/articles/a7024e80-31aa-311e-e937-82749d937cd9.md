
# MailMergeDataSource.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **MailMergeDataSource** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [MailMergeDataSource](f86f7d3c-d7ab-45e8-21e7-fd5a426e0391.md)** object.
