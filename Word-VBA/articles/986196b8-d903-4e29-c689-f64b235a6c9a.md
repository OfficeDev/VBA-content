
# Editor.Name Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the name of the specified object. Read-only  **String**.

## Syntax

 _expression_. **Name**

 _expression_An expression that represents a  ** [Editor](af0c80f5-8c8a-be0e-4475-d3b3b3bacd0d.md)** object.
