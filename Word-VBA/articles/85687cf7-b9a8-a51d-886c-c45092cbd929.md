
# Point.PieSliceLocation Method (Word)

 **Last modified:** July 28, 2015

Returns the vertical or horizontal position of a point on a chart item, in points, from the top or left edge of the object to the top or left edge of the chart area.

## Syntax

 _expression_. **PieSliceLocation**( **_loc_**,  **_Index_**)

 _expression_A variable that represents a  ** [Point](349ea9a3-9e9a-b26f-146f-799d39c3d4a9.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|loc|Required| ** [XlPieSliceLocation](d3a2113f-8779-096c-ce6a-52f29db242b3.md)**|Specifies a horizontal or vertical coordinate.|
|Index|Optional| ** [XlPieSliceIndex](3745028d-34b0-eaff-4437-d5df00b5ce52.md)**|Specifies which pie slice position coordinate to return. The default value is  **xlOuterCenterPoint**.|

### Return Value

Double


## Remarks

This property only applies to pie chart types.
