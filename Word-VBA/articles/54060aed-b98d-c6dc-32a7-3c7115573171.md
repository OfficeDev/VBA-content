
# ShapeRange.Adjustments Property (Word)

 **Last modified:** July 28, 2015

Returns an  ** [Adjustments](ed65525d-2c55-ae2a-ef42-1663b17e5c97.md)** object that contains adjustment values for all the adjustments in the specified **ShapeRange** object that represents an AutoShape or WordArt. Read-only.

## Syntax

 _expression_. **Adjustments**

 _expression_A variable that represents a  ** [ShapeRange](7112acc0-e241-16ef-77bc-101b72d05af0.md)** object.
