
# XMLNodes.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of XML elements in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [XMLNodes](c29850f2-8db2-aef6-57ee-fed1b625616c.md)** collection.
