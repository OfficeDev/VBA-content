
# LetterContent.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **LetterContent** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [LetterContent](62a4e17a-6598-c904-f27d-817c19c04981.md)** object.
