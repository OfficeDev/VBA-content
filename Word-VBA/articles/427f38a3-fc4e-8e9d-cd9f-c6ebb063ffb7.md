
# Shapes.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of shapes in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Shapes](0907eed3-886e-8e73-0e5e-71f4b37ddd5b.md)** collection.
