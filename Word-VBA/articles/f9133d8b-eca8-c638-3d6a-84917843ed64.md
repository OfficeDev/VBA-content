
# EmailOptions.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **EmailOptions** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [EmailOptions](41fefa03-c993-e218-0f92-0cf30c0bfbd4.md)** collection.
