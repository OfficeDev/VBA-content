
# Styles.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of styles in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Styles](bc4688ce-5055-c135-a656-e58e31d8be42.md)** collection.
