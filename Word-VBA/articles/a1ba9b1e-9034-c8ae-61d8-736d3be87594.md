
# HangulHanjaConversionDictionaries.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **Dictionary** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [HangulHanjaConversionDictionaries](b6ed1c54-428b-c160-a2bd-642978660f44.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The individual object to be returned. Can be a  **Long** indicating the ordinal position or a **String** representing the name of the individual object.|

### Return Value

Dictionary
