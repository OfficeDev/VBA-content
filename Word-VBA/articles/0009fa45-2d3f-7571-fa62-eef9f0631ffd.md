
# MappedDataFields.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of mapped data fields in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [MappedDataFields](d67de1fb-f495-ff4a-f21d-fd165a96232c.md)** collection.
