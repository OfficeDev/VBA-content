
# OMathMatRow.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathMatRow** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathMatRow](a6e96db5-9119-51a2-f303-a6c74b5be88d.md)** object.
