
# XMLNamespace.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified XML schema from the list of available XML schemas.

## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents a  ** [XMLNamespace](a50ed533-8961-f433-d501-797e5d678fda.md)** object.
