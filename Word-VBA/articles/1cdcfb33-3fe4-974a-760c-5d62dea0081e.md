
# CoAuthLocks.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  ** [CoAuthLock](3efa12b0-1079-c6df-20c1-a66398161c8e.md)** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns a  ** [CoAuthLocks](589763ed-8463-6988-3817-9c2152506d16.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned.|

### Return Value

 ** [CoAuthLock](3efa12b0-1079-c6df-20c1-a66398161c8e.md)**
