
# OMathMatRows.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathMatRows** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathMatRows](beeb74aa-23ba-b9da-1f24-65c91fb3dc2c.md)** object.
