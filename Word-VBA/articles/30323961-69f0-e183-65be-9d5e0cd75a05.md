
# ThreeDFormat.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ThreeDFormat** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ThreeDFormat](d397e780-a53d-0cc3-7a02-b40397253e91.md)** object.
