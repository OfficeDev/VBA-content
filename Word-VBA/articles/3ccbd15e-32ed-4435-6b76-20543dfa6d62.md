
# Shape.RelativeVerticalSize Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a  ** [WdRelativeVerticalSize](07a507e4-66fb-b383-fe36-aa5ccadbdcba.md)** constant that represents the relative vertical size of a shape. Read/write.

## Syntax

 _expression_. **RelativeVerticalSize**

 _expression_An expression that returns a  ** [Shape](604029ce-9b2f-9748-5d4e-b458796fa2f0.md)** object.


## Remarks

Use this property with the  ** [HeightRelative](24a52ebf-1071-a2e4-8222-9b17d295e653.md)** property.
