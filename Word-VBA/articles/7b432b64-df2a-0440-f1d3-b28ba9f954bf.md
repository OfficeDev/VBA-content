
# KeysBoundTo.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **KeysBoundTo** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [KeysBoundTo](63ed40e5-8223-78d6-c90a-bf6be8a2fbf6.md)** object.
