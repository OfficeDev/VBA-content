
# ShadowFormat.Type Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the shape shadow type. Read/write  **MsoShadowType**.

## Syntax

 _expression_. **Type**

 _expression_Required. A variable that represents a  ** [ShadowFormat](2a179f0b-ec18-c3dd-dd73-51b18f42e0e2.md)** object.
