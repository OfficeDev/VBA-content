
# AutoTextEntries.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **AutoTextEntries** collection.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents an  ** [AutoTextEntries](4e4d92b3-d259-84b7-061f-82065e177c29.md)** collection.
