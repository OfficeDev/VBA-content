
# ShapeNodes.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of shape nodes in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [ShapeNodes](f2e13db2-102f-1a14-fd7a-d179f63e513e.md)** collection.
