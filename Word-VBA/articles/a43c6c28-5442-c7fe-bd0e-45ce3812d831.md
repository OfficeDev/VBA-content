
# OMathMat.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathMat** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathMat](40478b6e-18fe-b7b7-d0bc-def1349db56a.md)** object.
