
# ContentControls.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ContentControls** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [ContentControls](2595eea9-df68-edce-3a51-069cad14bb87.md)** object.
