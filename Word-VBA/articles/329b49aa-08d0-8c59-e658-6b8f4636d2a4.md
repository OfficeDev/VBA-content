
# MailMergeField.Next Property (Word)

 **Last modified:** July 28, 2015

Returns the next mail merge field in the collection of mail merge fields. Read-only.

## Syntax

 _expression_. **Next**

 _expression_A variable that represents a  ** [MailMergeField](8beb6228-079c-008c-10aa-3f8f711fcf5c.md)** object.
