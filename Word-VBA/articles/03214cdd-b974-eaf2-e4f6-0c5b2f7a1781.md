
# XMLNodes.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **XMLNode** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [XMLNodes](c29850f2-8db2-aef6-57ee-fed1b625616c.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

XMLNode
