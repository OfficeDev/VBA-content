
# Point.PictureUnit2 Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the unit for each picture on the chart if the  ** [PictureType](050ce20e-0214-96df-6f9e-e8eef8625b1e.md)** property is set to **xlStackScale** (if not, this property is ignored). Read/write **Double**.

## Syntax

 _expression_. **PictureUnit2**

 _expression_A variable that represents a  ** [Point](349ea9a3-9e9a-b26f-146f-799d39c3d4a9.md)** object.
