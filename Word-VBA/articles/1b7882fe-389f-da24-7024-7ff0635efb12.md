
# XMLMapping.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **XMLMapping** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [XMLMapping](cf76802b-f93d-0f3b-4936-ca357a7d7ff8.md)** object.
