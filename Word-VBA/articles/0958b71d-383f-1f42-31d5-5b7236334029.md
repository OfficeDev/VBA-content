
# FreeformBuilder.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **FreeformBuilder** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [FreeformBuilder](31e89628-4b50-ff72-ce3d-dc7c161dad3e.md)** object.
