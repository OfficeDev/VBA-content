
# CustomLabel.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **CustomLabel** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [CustomLabel](a89ff4e1-ff8a-8a8f-afa2-6071bb49355b.md)** object.
