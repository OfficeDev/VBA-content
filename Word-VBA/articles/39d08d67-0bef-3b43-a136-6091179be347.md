
# CheckBox.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **CheckBox** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [CheckBox](e72b57b7-0328-9e78-94ca-ab7fb3c64afb.md)** object.
