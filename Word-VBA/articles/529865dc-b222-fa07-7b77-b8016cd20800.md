
# Dictionaries.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of dictionaries in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Dictionaries](41f31292-4b3e-0d7b-c857-f6b9a0662e9a.md)** collection.
