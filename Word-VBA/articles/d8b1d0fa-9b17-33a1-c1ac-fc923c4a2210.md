
# Pages.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Pages** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Pages](d51e5c61-5719-c70f-b244-99507889f2dc.md)** collection.
