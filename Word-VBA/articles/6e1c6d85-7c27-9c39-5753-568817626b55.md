
# OMathFrac.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathFrac** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathFrac](31221b8f-9edc-9684-3b4e-867c23cf1c26.md)** object.
