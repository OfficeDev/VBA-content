
# Dialogs.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Dialogs** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Dialogs](8dfa5d8a-bb81-1cdd-853b-3acf9db70aa9.md)** collection.
