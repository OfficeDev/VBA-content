
# CustomLabel.Height Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the height of a specified custom mailing label, in points. Read/write  **Single**.

## Syntax

 _expression_. **Height**

 _expression_An expression that returns a  ** [CustomLabel](a89ff4e1-ff8a-8a8f-afa2-6071bb49355b.md)** object.
