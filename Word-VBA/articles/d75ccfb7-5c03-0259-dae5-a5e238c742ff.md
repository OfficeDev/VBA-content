
# Row.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified table row.

## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents a  ** [Row](38a05858-829a-ea5c-ce63-7f7343bf7b88.md)** object.
