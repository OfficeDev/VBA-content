
# ProofreadingErrors.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **Range** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [ProofreadingErrors](53fb6382-4c08-83f3-1835-ac2633939758.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

Range
