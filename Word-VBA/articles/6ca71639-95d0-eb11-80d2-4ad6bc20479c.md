
# ChartArea.Shadow Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a value that determines whether the object has a shadow. Read/write  **Boolean**.

## Syntax

 _expression_. **Shadow**

 _expression_A variable that represents a  ** [ChartArea](7b3384df-f331-033d-4dfa-ee2ff26111c6.md)** object.
