
# ListParagraphs.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **Paragraph** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [ListParagraphs](759c510b-bca1-0b4b-005c-5a3783dd8e96.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

Paragraph
