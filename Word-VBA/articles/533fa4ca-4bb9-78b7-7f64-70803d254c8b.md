
# ListLevels.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **ListLevel** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [ListLevels](9165c008-c066-8d3e-9254-d9e0ab2ec091.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

ListLevel
