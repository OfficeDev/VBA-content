
# SpellingSuggestion.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **SpellingSuggestion** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [SpellingSuggestion](39598da5-8c76-41f3-dcb6-1e1162b30f28.md)** object.
