
# MailMergeField.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **MailMergeField** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [MailMergeField](8beb6228-079c-008c-10aa-3f8f711fcf5c.md)** object.
