
# Index.Type Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the index type. Read/write  ** [WdIndexType](031ee672-56e4-2c56-7ec1-a3aaa62e094d.md)**.

## Syntax

 _expression_. **Type**

 _expression_Required. A variable that represents an  ** [Index](6a2aab98-485b-01c3-8d9b-9e108b455e22.md)** object.
