
# InlineShape.Width Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the width, in points, of the specified inline shape. Read/write  **Long**.

## Syntax

 _expression_. **Width**

 _expression_A variable that represents an  ** [InlineShape](a8fd110a-4aa7-c4b9-1559-32022787d955.md)** object.
