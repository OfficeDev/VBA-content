
# FontNames.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of font names in the collection. Read-only **Long**.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [FontNames](d3a9a52f-b441-ac63-3e12-25dbf1022f38.md)** collection.
