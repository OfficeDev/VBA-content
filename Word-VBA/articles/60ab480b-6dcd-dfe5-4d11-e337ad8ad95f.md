
# HTMLDivisions.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **HTMLDivisions** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [HTMLDivisions](fe896440-817f-5485-794c-c5e9700cd062.md)** collection.
