
# TablesOfAuthorities.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **TablesOfAuthorities** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [TablesOfAuthorities](c0fd88b1-b737-2811-ec4c-1fc274fc3e20.md)** collection.
