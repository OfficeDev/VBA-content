
# Line.Width Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the width, in points, of a line. Read/write  **Long**.

## Syntax

 _expression_. **Width**

 _expression_A variable that represents a  ** [Line](1fbc9a15-c677-0f79-4311-9e6de6fc1b27.md)** object.
