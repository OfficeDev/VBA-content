
# OMathFunctions.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathFunctions** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathFunctions](163c6af9-5cb9-93bb-f1c2-b9ebeca28f6a.md)** object.
