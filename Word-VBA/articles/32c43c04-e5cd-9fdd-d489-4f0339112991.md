
# XMLNamespace.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **XMLNamespace** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [XMLNamespace](a50ed533-8961-f433-d501-797e5d678fda.md)** object.
