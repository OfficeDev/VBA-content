
# ListLevels.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of list levels in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [ListLevels](9165c008-c066-8d3e-9254-d9e0ab2ec091.md)** collection.
