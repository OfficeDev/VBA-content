
# OtherCorrectionsExceptions.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **OtherCorrectionsException** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents an  ** [OtherCorrectionsExceptions](f72135be-9a82-2c45-1835-8cabb18869de.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The individual object to be returned. Can be a  **Long** indicating the ordinal position or a **String** representing the name of the individual object.|

### Return Value

OtherCorrectionsException
