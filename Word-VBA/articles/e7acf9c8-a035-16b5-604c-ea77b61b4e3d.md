
# Pages.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of pages in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Pages](d51e5c61-5719-c70f-b244-99507889f2dc.md)** collection.
