
# OMathFunctions.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an  **OMathFunction** object that represents the specified item in the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns an  ** [OMathFunctions](163c6af9-5cb9-93bb-f1c2-b9ebeca28f6a.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|Specifies the ordinal position of the object within the collection.|

### Return Value

OMathFunction
