
# Rectangle.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Rectangle** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Rectangle](90ad4f48-2051-38f9-9b2e-a14bd38478be.md)** object.
