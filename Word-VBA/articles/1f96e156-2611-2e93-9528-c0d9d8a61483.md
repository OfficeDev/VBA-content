
# Footnotes.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Footnotes** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Footnotes](d46a0972-2784-4814-d547-30122a35cdc1.md)** collection.
