
# KeysBoundTo.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of keys in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [KeysBoundTo](63ed40e5-8223-78d6-c90a-bf6be8a2fbf6.md)** object.
