
# OMath.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMath** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMath](82f2f81b-e2d5-140f-bdcc-8b52b821b24d.md)** object.
