
# Broadcast.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Broadcast** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **Broadcast** object.


## Property value

 **OBJECT**
