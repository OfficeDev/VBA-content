
# Sections.Last Property (Word)

 **Last modified:** July 28, 2015

Returns the last item in the  **Sections** collection as a **Section** object.

## Syntax

 _expression_. **Last**

 _expression_Required. A variable that represents a  ** [Sections](cf6f77ba-9eee-5614-e697-bc031c4c6dcd.md)** collection.
