
# Document.Select Method (Word)

 **Last modified:** July 28, 2015

Selects the contents of the specified document.

## Syntax

 _expression_. **Select**

 _expression_Required. A variable that represents a  ** [Document](8d83487a-2345-a036-a916-971c9db5b7fb.md)** object.


## Remarks

After using this method, use the  ** [Selection](7b574a91-c33e-ecfd-6783-6b7528b2ed8f.md)** property to work with the selected items. For more information, see [Working with the Selection Object](a1ef7e48-5a0f-d278-4b67-7b96f4e24052.md).
