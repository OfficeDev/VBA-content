
# HeadingStyles.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **HeadingStyles** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [HeadingStyles](be882a12-1264-8f7e-415b-b8bcbf28e703.md)** collection.
