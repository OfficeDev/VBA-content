
# CustomProperties.Item Method (Word)

 **Last modified:** July 28, 2015

Returns a  **CustomProperty** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [CustomProperties](8b4248a1-7e1f-dbbd-37ab-f52a2d1ee505.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The individual object to be returned. Can be a  **Long** indicating the ordinal position or a **String** representing the name of the individual object.|

### Return Value

CustomProperty
