
# Windows.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of windows in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Windows](377b493b-e73c-0132-869c-3876c3beaef7.md)** collection.
