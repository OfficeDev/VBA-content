
# Hyperlinks.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Hyperlinks** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Hyperlinks](25801753-737f-9219-6a14-6531eb2ca699.md)** collection.
