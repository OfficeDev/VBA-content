
# Break.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Break** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Break](771ba998-c22e-3cf0-fab7-af9329793855.md)** object.
