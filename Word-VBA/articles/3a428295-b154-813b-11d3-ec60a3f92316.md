
# OMathScrSup.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathScrSup** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathScrSup](70adf652-d78c-4ca3-604f-57ca109e8109.md)** object.
