
# Fields.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Fields** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Fields](c79065bb-ba29-22fd-a9d7-90bb10550035.md)** collection.
