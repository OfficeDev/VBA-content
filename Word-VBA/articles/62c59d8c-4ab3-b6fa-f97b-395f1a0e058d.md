
# Columns.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of columns in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Columns](7c2d1353-cbc4-a162-83a1-6cac1300266f.md)** collection.
