
# XMLNode.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **XMLNode** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [XMLNode](fe305ba9-7375-ad4f-6036-155add17a9d0.md)** object.
