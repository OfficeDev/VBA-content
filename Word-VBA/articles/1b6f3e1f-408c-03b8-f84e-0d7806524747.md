
# OtherCorrectionsExceptions.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OtherCorrectionsExceptions** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [OtherCorrectionsExceptions](f72135be-9a82-2c45-1835-8cabb18869de.md)** collection.
