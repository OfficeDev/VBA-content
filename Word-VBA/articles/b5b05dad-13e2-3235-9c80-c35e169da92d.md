
# Reviewers.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Reviewers** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Reviewers](ae1bec96-e6dc-39f0-421a-dfeeb95c9049.md)** collection.
