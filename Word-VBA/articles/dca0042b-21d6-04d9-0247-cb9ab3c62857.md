
# SpellingSuggestions.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **SpellingSuggestion** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [SpellingSuggestions](7e0fb008-e43c-c4cb-b7d2-9436d039a070.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

SpellingSuggestion
