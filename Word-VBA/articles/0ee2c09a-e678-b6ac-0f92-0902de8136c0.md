
# Bibliography.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Bibliography** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [Bibliography](2e6c5a98-5c49-2404-e519-ef8bfc4215d2.md)** object.
