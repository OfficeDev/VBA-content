
# CustomLabel.Name Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the name of the specified object. Read/write  **CustomLabel**.

## Syntax

 _expression_. **Name**

 _expression_Required. A variable that represents a  ** [CustomLabel](a89ff4e1-ff8a-8a8f-afa2-6071bb49355b.md)** object.
