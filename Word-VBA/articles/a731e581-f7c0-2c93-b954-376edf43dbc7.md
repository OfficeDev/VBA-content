
# TablesOfAuthorities.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **TableOfAuthorities** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [TablesOfAuthorities](c0fd88b1-b737-2811-ec4c-1fc274fc3e20.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

TableOfAuthorities
