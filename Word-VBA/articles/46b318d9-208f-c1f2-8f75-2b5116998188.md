
# OMathNary.Char Property (Word)

 **Last modified:** July 28, 2015

Returns or sets an  **Integer** that represents a character used as the n-ary operator. Read/write.

## Syntax

 _expression_. **Char**

 _expression_An expression that returns an  ** [OMathNary](a1e89e56-2881-7ec3-915d-6a0dff45eab2.md)** object.
