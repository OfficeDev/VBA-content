
# Document.Bibliography Property (Word)

 **Last modified:** July 28, 2015

Returns a  ** [Bibliography](2e6c5a98-5c49-2404-e519-ef8bfc4215d2.md)** object that represents the bibliography references contained within a document. Read-only.

## Syntax

 _expression_. **Bibliography**

 _expression_An expression that returns a  ** [Document](8d83487a-2345-a036-a916-971c9db5b7fb.md)** object.
