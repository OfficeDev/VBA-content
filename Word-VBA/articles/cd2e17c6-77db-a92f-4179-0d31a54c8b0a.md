
# ProofreadingErrors.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ProofreadingErrors** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ProofreadingErrors](53fb6382-4c08-83f3-1835-ac2633939758.md)** collection.
