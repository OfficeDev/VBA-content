
# CustomProperties.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of items in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [CustomProperties](8b4248a1-7e1f-dbbd-37ab-f52a2d1ee505.md)** collection.
