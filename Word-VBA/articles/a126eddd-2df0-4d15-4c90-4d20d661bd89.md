
# XMLSchemaReferences.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **XMLSchemaReferences** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [XMLSchemaReferences](56bef973-805c-c77a-6d2a-54a39fbd1206.md)** collection.
