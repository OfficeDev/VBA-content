
# FileConverters.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **FileConverters** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [FileConverters](b9b8fc53-1c8e-224d-726a-4edf172ca647.md)** collection.
