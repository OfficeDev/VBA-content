
# ListTemplate.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ListTemplate** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ListTemplate](d5e339f7-5798-305b-a6b0-6b572d9112f4.md)** object.
