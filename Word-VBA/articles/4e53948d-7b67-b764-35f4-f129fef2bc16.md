
# KeyBindings.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of key bindings in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [KeyBindings](d2e38b04-b7e1-b35c-e511-5988d132b074.md)** collection.
