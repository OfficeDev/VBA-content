
# LegendEntries Properties (Word)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](41c01d34-0f89-c898-4b8a-43daf05d9a8d.md)|When used without an object qualifier, returns an  ** [Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)**object that represents the Microsoft Word application. When used with an object qualifier, returns an  **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
| [Count](217d61b8-d48a-fa64-219a-e9c4cde91403.md)|Returns the number of objects in the collection. Read-only  **Long**.|
| [Creator](4153e082-a1df-a118-3db6-4603e991bf9c.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
| [Parent](0c35d00d-a4d6-b521-e9f7-27b4adc3a290.md)|Returns the parent for the specified object. Read-only  **Object**.|
