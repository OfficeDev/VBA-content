
# OMathFunction.Rad Property (Word)

 **Last modified:** July 28, 2015

Returns an  ** [OMathRad](2179cda9-b1dc-9593-c4f9-99496081e191.md)** object that represents the mathematical radical function. Read-only.

## Syntax

 _expression_. **Rad**

 _expression_An expression that returns an  ** [OMathFunction](2347ec7b-5e1a-8039-5fd0-195c08860cb5.md)** object.
