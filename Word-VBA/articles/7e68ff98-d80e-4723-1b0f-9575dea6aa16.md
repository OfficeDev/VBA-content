
# XMLNode.Copy Method (Word)

 **Last modified:** July 28, 2015

Copies the specified XML element, excluding XML markup, to the Clipboard.

## Syntax

 _expression_. **Copy**

 _expression_An expression that returns an  **XMLNode** object.


### Return Value

Nothing
