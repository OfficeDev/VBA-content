
# Gridlines Properties (Word)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](49950209-0edd-e2b2-a2a7-35a6ce9dc0cb.md)|When used without an object qualifier, returns an  ** [Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)**object that represents the Microsoft Word application. When used with an object qualifier, returns an  **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
| [Border](faf4d3ac-c870-c28d-6ab1-aca4b28c0132.md)|Returns the border of the object. Read-only  ** [ChartBorder](eea90670-c599-2ec8-5b7b-c946a4bcd638.md)**.|
| [Creator](e3548aa9-1e88-b5ee-eb4a-417504d7bde5.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
| [Format](d3cca580-e797-f317-4d0d-af75967eedb4.md)|Returns the line, fill, and effect formatting for the object. Read-only  ** [ChartFormat](5f6546e8-c2fd-eec5-27a9-f2fd2c058f16.md)**.|
| [Name](755f93cb-dc5c-f1be-a64b-4b1dc514a5a8.md)|Returns the name of the object. Read-only  **String**.|
| [Parent](9b83ee7f-f5fa-008a-99fb-b714d34da497.md)|Returns the parent for the specified object. Read-only  **Object**.|
