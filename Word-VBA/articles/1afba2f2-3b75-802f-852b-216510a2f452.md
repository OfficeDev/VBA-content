
# Zooms.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Zooms** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Zooms](1a4d5375-ad91-1eb9-77cb-4a6f8dcc3eb8.md)** collection.
