
# Version.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified version.

## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents a  ** [Version](63eeefb0-2d63-75e6-a070-a4a80f243bc4.md)** object.
