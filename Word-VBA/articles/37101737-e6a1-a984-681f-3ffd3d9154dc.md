
# XSLTransform.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **XSLTransform** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [XSLTransform](cccf0383-8b21-0f46-b5b6-9a092599fd76.md)** object.
