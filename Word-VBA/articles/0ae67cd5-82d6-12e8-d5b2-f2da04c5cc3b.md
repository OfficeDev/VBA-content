
# OMathMatCols.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an  **OMathMatCol** object that represents the specified item in the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns an  ** [OMathMatCols](b56ee426-56bd-6588-ebe9-898f4bfbba0c.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|Specifies the ordinal position of the object within the collection.|

### Return Value

OMathMatCol
