
# OMathMatCols.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathMatCols** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathMatCols](b56ee426-56bd-6588-ebe9-898f4bfbba0c.md)** object.
