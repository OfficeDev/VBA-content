
# Frame.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified frame.

## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents a  ** [Frame](d36d3361-9e93-7dd9-b8c9-0ce503e03810.md)** object.
