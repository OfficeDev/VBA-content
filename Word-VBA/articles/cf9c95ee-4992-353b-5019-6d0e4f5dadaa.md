
# Hyperlink.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified hyperlink.

## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents a  ** [Hyperlink](af785a9e-081a-e359-705f-04f490304e2e.md)** object.
