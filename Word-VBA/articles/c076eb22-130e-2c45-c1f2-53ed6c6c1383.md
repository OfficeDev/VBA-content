
# CaptionLabel.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **CaptionLabel** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [CaptionLabel](71c82dfd-6a66-e0f4-e30f-ae453c764864.md)** object.
