
# EmailSignatureEntry.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **EmailSignatureEntry** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [EmailSignatureEntry](a8cf11de-7a46-c609-3cd7-508e9ef91e09.md)** object.
