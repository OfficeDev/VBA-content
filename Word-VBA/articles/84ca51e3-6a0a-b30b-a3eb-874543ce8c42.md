
# CaptionLabels.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **CaptionLabel** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [CaptionLabels](7d18c0d6-6d58-9841-4665-ab13e2e2ad9f.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The individual object to be returned. Can be a  **Long** indicating the ordinal position or a **String** representing the name of the individual object.|

### Return Value

CaptionLabel
