
# Subdocuments.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **Subdocument** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [Subdocuments](8e14fced-19b3-2794-39a3-9e5ec15dd0ad.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

Subdocument
