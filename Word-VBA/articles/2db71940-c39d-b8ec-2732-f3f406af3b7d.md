
# View.MarkupMode Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a  ** [WdRevisionsMode](eb6663f0-a043-2ded-28d1-d54c73fa2813.md)** constant that represents the display mode for tracked changes. Read/write.

## Syntax

 _expression_. **MarkupMode**

 _expression_An expression that returns a  **View** object.
