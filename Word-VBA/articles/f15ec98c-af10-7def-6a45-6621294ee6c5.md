
# Field.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Field** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Field](75139aa4-89f4-2ffb-b964-8dc805b9a32b.md)** object.
