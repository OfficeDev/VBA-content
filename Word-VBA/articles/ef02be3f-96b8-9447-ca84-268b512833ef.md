
# ReadabilityStatistics.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of readability statistics in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [ReadabilityStatistics](eabef73c-f837-435a-cfec-b76082cc0f7e.md)** collection.
