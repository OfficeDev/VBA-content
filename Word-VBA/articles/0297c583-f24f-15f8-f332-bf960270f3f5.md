
# View.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **View** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [View](8bf5b26b-14c0-1985-65b2-3e034360baeb.md)** object.
