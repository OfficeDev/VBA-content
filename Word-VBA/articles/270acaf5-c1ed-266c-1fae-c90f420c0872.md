
# Version.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Version** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Version](63eeefb0-2d63-75e6-a070-a4a80f243bc4.md)** object.
