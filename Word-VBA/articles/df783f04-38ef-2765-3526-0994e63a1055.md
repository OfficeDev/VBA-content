
# XMLNamespaces.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **XMLNamespaces** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [XMLNamespaces](f11a6cc1-f33d-e1ab-870c-aa5857d66797.md)** collection.
