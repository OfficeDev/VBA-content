
# Endnote.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Endnote** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [Endnote](01f29be4-58e7-28f5-5fcb-dae50c33890e.md)** object.
