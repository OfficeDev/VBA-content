
# Chart.Shapes Property (Word)

 **Last modified:** July 28, 2015

Returns a collection that represents all the shapes on the chart sheet. Read-only  ** [Shapes](0907eed3-886e-8e73-0e5e-71f4b37ddd5b.md)**.

## Syntax

 _expression_. **Shapes**

 _expression_A variable that represents a  ** [Chart](366a825e-0daf-dbb7-b6f2-e7ce1a5ee2ef.md)** object.
