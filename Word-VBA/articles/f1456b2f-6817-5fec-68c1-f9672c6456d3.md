
# KeysBoundTo.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **KeyBinding** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [KeysBoundTo](63ed40e5-8223-78d6-c90a-bf6be8a2fbf6.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

KeyBinding
