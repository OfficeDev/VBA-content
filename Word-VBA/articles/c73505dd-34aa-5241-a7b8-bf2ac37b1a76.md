
# List.RemoveNumbers Method (Word)

 **Last modified:** July 28, 2015

Removes numbers or bullets from the specified list.

## Syntax

 _expression_. **RemoveNumbers**( **_NumberType_**)

 _expression_A variable that represents a  ** [List](2c3dae28-447a-af48-2966-e19ae75ab6c2.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|NumberType|Optional| ** [WdNumberType](3ed66985-c6ea-56fc-affb-0f6f0bb30f44.md)**| The type of number to be removed.|
