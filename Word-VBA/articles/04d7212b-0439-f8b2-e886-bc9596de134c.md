
# PageNumber.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified page number.

## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents a  ** [PageNumber](5b58f562-de19-ac9d-0b2c-7696603c1606.md)** object.
