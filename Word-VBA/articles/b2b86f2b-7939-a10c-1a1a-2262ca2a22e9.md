
# EmailSignature.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **EmailSignature** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [EmailSignature](9d641321-d52b-ab9a-4117-6f9e11dedbba.md)** object.
