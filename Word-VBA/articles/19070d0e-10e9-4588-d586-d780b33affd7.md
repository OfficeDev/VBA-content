
# Templates.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Templates** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Templates](de62f768-011a-7446-48c3-1c4512da5f7c.md)** collection.
