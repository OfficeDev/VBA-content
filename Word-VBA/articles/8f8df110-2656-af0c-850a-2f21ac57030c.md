
# Rectangles.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of rectangles in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Rectangles](c1de5e7f-13b1-e35a-d9f1-9a8f1246e2e5.md)** collection.
