
# ListGalleries.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of list galleries in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [ListGalleries](3ae91fbf-fb7c-e96f-fd13-e4e4e9c4f09e.md)** collection.
