
# ShapeRange.RelativeVerticalSize Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a  ** [WdRelativeVerticalSize](07a507e4-66fb-b383-fe36-aa5ccadbdcba.md)** constant that represents the object to which a range of shapes is relative. Read/write.

## Syntax

 _expression_. **RelativeVerticalSize**

 _expression_An expression that returns a  ** [ShapeRange](7112acc0-e241-16ef-77bc-101b72d05af0.md)** object.


## Remarks

Use this property with the  ** [HeightRelative](f0414af1-f09a-475d-5e96-bfe2ceab8901.md)** property.
