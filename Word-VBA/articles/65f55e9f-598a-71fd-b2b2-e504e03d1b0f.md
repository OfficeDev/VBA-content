
# OMathBreaks.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathBreaks** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathBreaks](fa01cd62-b8ad-52bf-f36a-f5d1548d3d1e.md)** object.
