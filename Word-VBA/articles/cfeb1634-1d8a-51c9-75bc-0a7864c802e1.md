
# TwoInitialCapsException.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified two initial-capital exception from the collection of AutoCorrect exceptions.

## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents a  ** [TwoInitialCapsException](48e89297-4137-960b-a92a-2a70929e298a.md)** object.
