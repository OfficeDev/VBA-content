
# Footnote.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Footnote** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Footnote](877340c4-14f9-4560-eaf8-2c6482a1ade8.md)** object.
