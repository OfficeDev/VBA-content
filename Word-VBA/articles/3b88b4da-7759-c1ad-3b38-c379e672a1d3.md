
# CustomProperties.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **CustomProperties** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [CustomProperties](8b4248a1-7e1f-dbbd-37ab-f52a2d1ee505.md)** collection.
