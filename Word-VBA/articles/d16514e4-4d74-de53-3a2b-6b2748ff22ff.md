
# OMathMatRows.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an  **OMathMatRow** object that represents the specified item in the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns an  ** [OMathMatRows](beeb74aa-23ba-b9da-1f24-65c91fb3dc2c.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|Specifies the ordinal position of the object within the collection.|

### Return Value

OMathMatRow
