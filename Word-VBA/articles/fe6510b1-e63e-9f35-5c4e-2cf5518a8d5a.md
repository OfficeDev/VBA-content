
# Reviewers.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of reviewers in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Reviewers](ae1bec96-e6dc-39f0-421a-dfeeb95c9049.md)** collection.
