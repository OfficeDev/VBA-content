
# Templates.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **Template** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [Templates](de62f768-011a-7446-48c3-1c4512da5f7c.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The individual object to be returned. Can be a  **Long** indicating the ordinal position or a **String** representing the name of the individual object.|

### Return Value

Template
