
# KeyBindings.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **KeyBinding** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [KeyBindings](d2e38b04-b7e1-b35c-e511-5988d132b074.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

KeyBinding
