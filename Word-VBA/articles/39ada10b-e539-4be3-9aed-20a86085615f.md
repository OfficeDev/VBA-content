
# TickLabels.NumberFormatLocal Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the format code for the object as a string in the language of the user. Read/write  **Variant**.

## Syntax

 _expression_. **NumberFormatLocal**

 _expression_A variable that represents a  ** [TickLabels](d94e90dc-0b0e-f4af-078e-6f2b97729db5.md)** object.


## Remarks

The  **Format** function uses different format code strings than do the ** [NumberFormat](ce7d9eb9-83d4-2c7e-e8ff-a08b4978c70d.md)** and **NumberFormatLocal** properties.
