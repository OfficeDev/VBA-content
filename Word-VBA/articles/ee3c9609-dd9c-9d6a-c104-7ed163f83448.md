
# OMathFunction.Frac Property (Word)

 **Last modified:** July 28, 2015

Returns an  ** [OMathFrac](31221b8f-9edc-9684-3b4e-867c23cf1c26.md)** object that represents a fraction. Read-only.

## Syntax

 _expression_. **Frac**

 _expression_An expression that returns an  ** [OMathFunction](2347ec7b-5e1a-8039-5fd0-195c08860cb5.md)** object.
