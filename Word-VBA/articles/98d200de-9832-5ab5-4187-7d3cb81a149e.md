
# TextInput.Width Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the width, in points, of the specified text input field. Read/write  **Long**.

## Syntax

 _expression_. **Width**

 _expression_A variable that represents a  ** [TextInput](d7f6531a-4da2-ccc4-29b3-ad79ca7b18de.md)** object.
