
# AxisTitle.Shadow Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a value that determines whether the object has a shadow. Read/write  **Boolean**.

## Syntax

 _expression_. **Shadow**

 _expression_A variable that represents an  ** [AxisTitle](ec746a05-40df-95cc-c017-40ef150504cf.md)** object.
