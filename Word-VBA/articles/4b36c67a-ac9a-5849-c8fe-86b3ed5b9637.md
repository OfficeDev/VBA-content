
# ChartFormat.Glow Property (Word)

 **Last modified:** July 28, 2015

Returns the glow formatting properties for the chart element. Read-only  ** [GlowFormat](d177e399-cafc-132f-b135-b8b578e76889.md)**.

## Syntax

 _expression_. **Glow**

 _expression_A variable that represents a  ** [ChartFormat](5f6546e8-c2fd-eec5-27a9-f2fd2c058f16.md)** object.


## Remarks

The glow effect adds a vibrant colored edge to graphics.
