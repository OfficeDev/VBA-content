
# Font.TextColor Property (Word)

 **Last modified:** July 28, 2015

Returns a  [ColorFormat](5f12793f-d847-ecf2-6cf6-39387f7f0b28.md) object that represents the color for the specified font. Read-only.

## Syntax

 _expression_. **TextColor**

 _expression_An expression that returns a  ** [Font](bc97f4df-fc81-d6c8-e99a-d50dc793b7ae.md)** object.
