
# GroupShapes.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **GroupShapes** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [GroupShapes](de29d571-476b-fa8b-619e-f7d0181d9756.md)** collection.
