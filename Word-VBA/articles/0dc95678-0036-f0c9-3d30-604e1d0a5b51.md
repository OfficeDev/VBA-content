
# HangulHanjaConversionDictionaries.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of dictionaries in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [HangulHanjaConversionDictionaries](b6ed1c54-428b-c160-a2bd-642978660f44.md)** collection.
