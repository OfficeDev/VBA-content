
# TickLabels.Font Property (Word)

 **Last modified:** July 28, 2015

## Syntax

 _expression_. **Font**

 _expression_A variable that represents a  ** [TickLabels](d94e90dc-0b0e-f4af-078e-6f2b97729db5.md)** object.
