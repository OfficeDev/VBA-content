
# Indexes.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Indexes** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [Indexes](0441446a-c1b5-d333-5950-906fe463b61d.md)** collection.
