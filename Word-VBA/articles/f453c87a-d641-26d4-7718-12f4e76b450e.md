
# Replacement.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Replacement** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Replacement](5d9615e4-f6ef-af5f-6e45-c382a88395c9.md)** object.
