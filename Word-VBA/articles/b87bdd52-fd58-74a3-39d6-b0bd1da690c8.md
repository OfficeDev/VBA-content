
# TextColumns.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **TextColumn** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [TextColumns](00b62c93-db7d-00b9-cc84-9a21e427d0cd.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

TextColumn
