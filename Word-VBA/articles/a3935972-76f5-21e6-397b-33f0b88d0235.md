
# Style.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Style** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Style](473f8f41-2cba-769e-c0da-441d9d85b009.md)** object.
