
# Categories.Item Method (Word)

 **Last modified:** July 28, 2015

Returns a  **Category** object that represents the specified item in the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns a  ** [Categories](f5f5081d-4309-6617-28da-c369c1fe690c.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|Specifies a  **String** or **Integer** that represents the name or ordinal position of the object in the collection.|

### Return Value

Category
