
# ParagraphFormat.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ParagraphFormat** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ParagraphFormat](712d754a-dc92-f1a3-531d-dfae74a42c23.md)** object.
