
# Frames.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of frames in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **Frames** object.


## Property value

 **INT32**
