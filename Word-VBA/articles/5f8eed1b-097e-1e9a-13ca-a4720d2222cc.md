
# Email.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Email** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents an  ** [Email](ee23a74e-556b-04d8-f0b9-fb95f7aa8cfc.md)** object.
