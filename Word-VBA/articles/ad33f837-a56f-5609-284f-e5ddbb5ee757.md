
# XMLSchemaReference.Reload Method (Word)

 **Last modified:** July 28, 2015

Reloads the XML schemas that are referenced in a document.

## Syntax

 _expression_. **Reload**

 _expression_An expression that returns an  ** [XMLSchemaReference](54142ef1-f731-3f82-2dc0-809d8a041b73.md)** object.


### Return Value

Nothing
