
# OMathAutoCorrectEntry.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified equation auto correct entry.

## Syntax

 _expression_. **Delete**

 _expression_An expression that returns an  **OMathAutoCorrectEntry** object.
