
# PageNumbers.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **PageNumbers** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [PageNumbers](9090f96e-d898-ace6-35fa-f6e59c527ea2.md)** collection.
