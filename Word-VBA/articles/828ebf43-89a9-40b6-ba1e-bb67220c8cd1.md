
# TableOfContents Properties (Word)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](8082d901-cb50-e4e1-6fc2-f6b69908e633.md)|Returns an  ** [Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)** object that represents the Microsoft Word application.|
| [Creator](1918bed7-47ca-b2d6-ad1e-a4b4e9e8d4a9.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
| [HeadingStyles](05cf7783-6b5d-bfbb-a417-1ae12d13f78e.md)|Returns a  ** [HeadingStyles](be882a12-1264-8f7e-415b-b8bcbf28e703.md)** object that represents additional styles used to compile a table of contents or table of figures (styles other than the Heading 1 - Heading 9 styles). Read-only.|
| [HidePageNumbersInWeb](81d77980-099e-e048-b219-d10b64cd6a38.md)|Returns or sets whether page numbers in a table of contents or a table of figures should be hidden when publishing to the Web. Read/write  **Boolean**.|
| [IncludePageNumbers](2097f009-ae18-70c3-3f3b-dbabcee06fa5.md)| **True** if page numbers are included in the table of contents. Read/write **Boolean**.|
| [LowerHeadingLevel](02bd1965-b3a1-e09a-fb08-62862e87536b.md)|Returns or sets the ending heading level for a table of contents or table of figures. Read/write  **Long**.|
| [Parent](1df4a5e7-7aed-e843-6d20-c0e2b09aef9d.md)|Returns an  **Object** that represents the parent object of the specified **TableOfContents** object.|
| [Range](1e85e099-4997-56cc-7b10-b3a0080760b2.md)|Returns a  **Range** object that represents the portion of a document that is contained within the specified table of contents.|
| [RightAlignPageNumbers](f14e4b13-a6d4-0085-af31-ef4077b5104f.md)| **True** if page numbers are aligned with the right margin in a table of contents. Read/write **Boolean**.|
| [TabLeader](aba91b67-33c6-fe8c-0a13-4d5692256091.md)|Returns or sets the character between entries and their page numbers in an index, table of authorities, table of contents, or table of figures. Read/write  ** [WdTabLeader](055e8d2d-86ec-cca6-6d73-6c65559974b8.md)**.|
| [TableID](d95186f5-b6ee-20cd-840e-e55ec3f06d04.md)|Returns or sets a one-letter identifier that's used to build a table of contents from TOC fields. Read/write  **String**.|
| [UpperHeadingLevel](3b360b6b-a422-4af5-9121-200105b0ad19.md)|Returns or sets the starting heading level for a table of contents. Read/write  **Long**.|
| [UseFields](36d01961-ba9a-fe8d-d791-f892bea8b994.md)| **True** if Table of Contents Entry (TC) fields are used to create a table of contents or a table of figures. Read/write **Boolean**.|
| [UseHeadingStyles](c026c00b-f3ec-b350-d046-0761b6e70851.md)| **True** if built-in heading styles are used to create a table of contents. Read/write **Boolean**.|
| [UseHyperlinks](2ff74d58-6411-eb10-1ce4-86d0b8e37490.md)|Returns or sets whether entries in a table of contents should be formatted as hyperlinks when publishing to the Web. Read/write  **Boolean**.|
