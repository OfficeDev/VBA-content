
# XSLTransforms.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of XSLTransforms in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [XSLTransforms](489a3391-fbd1-a326-9c4d-7fa8d4b64d02.md)** collection.
