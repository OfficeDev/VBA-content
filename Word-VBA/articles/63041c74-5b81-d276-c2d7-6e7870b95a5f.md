
# Sections.First Property (Word)

 **Last modified:** July 28, 2015

Returns a  ** [Section](3fe563d8-fc05-c17a-e67b-c50eea7e7f13.md)** object that represents the first item in the **Sections** collection.

## Syntax

 _expression_. **First**

 _expression_Required. A variable that represents a  ** [Sections](cf6f77ba-9eee-5614-e697-bc031c4c6dcd.md)** collection.
