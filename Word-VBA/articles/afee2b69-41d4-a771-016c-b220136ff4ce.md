
# CustomLabels.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **CustomLabels** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [CustomLabels](407e75b5-4116-fdc7-f0c1-dfd3809cdb41.md)** collection.
