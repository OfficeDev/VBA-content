
# Footnotes.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of footnotes in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Footnotes](d46a0972-2784-4814-d547-30122a35cdc1.md)** collection.
