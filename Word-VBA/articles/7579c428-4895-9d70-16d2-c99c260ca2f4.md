
# LegendKey.PictureType Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the way pictures are displayed on a legend key. Read/write  ** [XlChartPictureType](d28ab916-34dc-344e-503f-53102f2fd8c2.md)**.

## Syntax

 _expression_. **PictureType**

 _expression_A variable that represents a  ** [LegendKey](07578528-3e73-7898-47dc-296aefb854f0.md)** object.
