
# Panes.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of panes in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Panes](6ed6353c-9134-f47d-a108-13e84eced8ff.md)** collection.
