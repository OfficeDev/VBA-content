
# OMathDelim.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathDelim** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathDelim](6c203045-a0e0-6bf7-d8c7-01e991a3a168.md)** object.
