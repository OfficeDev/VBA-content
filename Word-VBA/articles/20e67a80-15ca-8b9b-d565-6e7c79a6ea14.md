
# ParagraphFormat.Shading Property (Word)

 **Last modified:** July 28, 2015

Returns a  ** [Shading](e136509a-1be1-29e4-7b37-1faf659e37ba.md)** object that refers to the shading formatting for the specified object.

## Syntax

 _expression_. **Shading**

 _expression_Required. A variable that represents a  ** [ParagraphFormat](712d754a-dc92-f1a3-531d-dfae74a42c23.md)** object.
