
# Subdocuments.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of subdocuments in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Subdocuments](8e14fced-19b3-2794-39a3-9e5ec15dd0ad.md)** collection.
