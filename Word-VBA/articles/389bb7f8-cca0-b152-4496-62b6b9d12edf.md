
# AutoCorrectEntries.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **AutoCorrectEntry** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents an  ** [AutoCorrectEntries](3823f96c-f600-d279-2592-253025ad63ff.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The individual object to be returned. Can be a  **Long** indicating the ordinal position or a **String** representing the name of the individual object.|

### Return Value

AutoCorrectEntry
