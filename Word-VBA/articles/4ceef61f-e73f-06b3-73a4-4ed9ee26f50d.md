
# OMathNary.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathNary** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathNary](a1e89e56-2881-7ec3-915d-6a0dff45eab2.md)** object.
