
# CustomLabel.Width Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the width of a custom mailing label, in points. Read/write  **Long**.

## Syntax

 _expression_. **Width**

 _expression_A variable that represents a  ** [CustomLabel](a89ff4e1-ff8a-8a8f-afa2-6071bb49355b.md)** object.
