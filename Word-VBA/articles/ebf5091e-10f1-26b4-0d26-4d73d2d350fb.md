
# Revisions.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of revisions in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Revisions](7f267a64-885a-cb4c-008a-e8545cea94d2.md)** collection.
