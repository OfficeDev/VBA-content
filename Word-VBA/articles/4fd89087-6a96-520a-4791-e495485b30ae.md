
# Columns.Last Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Column** object that represents the last column in a table.

## Syntax

 _expression_. **Last**

 _expression_Required. A variable that represents a  ** [Columns](7c2d1353-cbc4-a162-83a1-6cac1300266f.md)** collection.
