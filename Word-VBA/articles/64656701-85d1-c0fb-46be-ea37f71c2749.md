
# ShapeRange.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ShapeRange** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ShapeRange](7112acc0-e241-16ef-77bc-101b72d05af0.md)** object.
