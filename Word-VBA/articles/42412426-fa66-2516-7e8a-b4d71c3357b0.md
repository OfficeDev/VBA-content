
# TabStop.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **TabStop** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [TabStop](5290ae79-f728-24a8-6bb0-267072cd0288.md)** object.
