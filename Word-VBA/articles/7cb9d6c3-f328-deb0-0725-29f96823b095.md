
# SpellingSuggestions.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of spelling suggestions in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [SpellingSuggestions](7e0fb008-e43c-c4cb-b7d2-9436d039a070.md)** collection.
