
# OMathFunction.OMath Property (Word)

 **Last modified:** July 28, 2015

Returns an  ** [OMath](82f2f81b-e2d5-140f-bdcc-8b52b821b24d.md)** object that represents the equation. Read-only.

## Syntax

 _expression_. **OMath**

 _expression_An expression that returns an  ** [OMathFunction](2347ec7b-5e1a-8039-5fd0-195c08860cb5.md)** object.
