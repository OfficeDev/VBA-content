
# Legend.Shadow Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean**.

## Syntax

 _expression_. **Shadow**

 _expression_A variable that represents a  ** [Legend](f0122074-87b7-0225-3c6c-406103fa4c29.md)** object.
