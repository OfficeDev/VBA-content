
# OMathLimUpp.ToLimLow Method (Word)

 **Last modified:** July 28, 2015

Converts an equation from the upper limit to the lower limit.

## Syntax

 _expression_. **ToLimLow**

 _expression_An expression that returns a  **OMathLimUpp** object.


### Return Value

OMathFunction
