
# TaskPanes.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of task panes in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [TaskPanes](a560a41b-a1d7-175a-b475-af742c9fa1f8.md)** collection.
