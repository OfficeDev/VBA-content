
# Rectangles.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **Rectangle** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [Rectangles](c1de5e7f-13b1-e35a-d9f1-9a8f1246e2e5.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

Rectangle
