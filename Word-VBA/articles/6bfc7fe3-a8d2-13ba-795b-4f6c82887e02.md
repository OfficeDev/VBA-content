
# FirstLetterExceptions.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **FirstLetterExceptions** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [FirstLetterExceptions](5dc5cc43-a696-d80f-58f9-0f74dfcad0ed.md)** collection.
