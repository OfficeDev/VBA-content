
# OMathRad.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **OMathRad** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [OMathRad](2179cda9-b1dc-9593-c4f9-99496081e191.md)** object.
