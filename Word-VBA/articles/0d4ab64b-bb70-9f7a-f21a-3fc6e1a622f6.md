
# Paragraph.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Paragraph** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Paragraph](0a704079-a082-4ab1-841b-fc0d49dd26d4.md)** object.
