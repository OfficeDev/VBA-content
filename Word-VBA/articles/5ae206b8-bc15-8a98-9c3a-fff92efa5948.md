
# CanvasShapes.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **CanvasShapes** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [CanvasShapes](f4b37915-7fde-2a21-0df0-fc3c97983900.md)** collection.
