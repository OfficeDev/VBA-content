
# Styles.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Styles** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Styles](bc4688ce-5055-c135-a656-e58e31d8be42.md)** collection.
