
# OMathNary.Sup Property (Word)

 **Last modified:** July 28, 2015

Returns an  ** [OMath](82f2f81b-e2d5-140f-bdcc-8b52b821b24d.md)** object that represents the upper limit of an n-ary operator. Read-only.

## Syntax

 _expression_. **Sup**

 _expression_An expression that returns an  ** [OMathNary](a1e89e56-2881-7ec3-915d-6a0dff45eab2.md)** object.
