
# Browser.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Browser** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents an  ** [Browser](447bcab6-cfb2-77b0-9bbd-35e774417a60.md)** object.
