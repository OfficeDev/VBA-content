
# Line.Height Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the height of a line.

## Syntax

 _expression_. **Height**

 _expression_An expression that represents a  ** [Line](1fbc9a15-c677-0f79-4311-9e6de6fc1b27.md)** object.
