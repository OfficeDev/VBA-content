
# ShapeNode.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ShapeNode** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ShapeNode](d5afb71a-a218-57f3-87f0-171094ba6610.md)** object.
