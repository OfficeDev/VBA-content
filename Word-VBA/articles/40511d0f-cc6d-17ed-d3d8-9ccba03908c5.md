
# ListParagraphs.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ListParagraphs** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ListParagraphs](759c510b-bca1-0b4b-005c-5a3783dd8e96.md)** collection.
