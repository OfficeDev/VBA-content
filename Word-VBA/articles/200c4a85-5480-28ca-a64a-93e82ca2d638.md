
# ChartTitle.Shadow Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean**.

## Syntax

 _expression_. **Shadow**

 _expression_A variable that represents a  ** [ChartTitle](fc8ca540-0a29-123b-2fdf-b16aaa1f940c.md)** object.
