
# Selection.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified ** [Selection](7b574a91-c33e-ecfd-6783-6b7528b2ed8f.md)** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [Selection](7b574a91-c33e-ecfd-6783-6b7528b2ed8f.md)** object.
