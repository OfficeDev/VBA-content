
# Source.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Source** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [Source](f90108a8-6432-a700-86ce-7b8f9e9c034b.md)** object.
