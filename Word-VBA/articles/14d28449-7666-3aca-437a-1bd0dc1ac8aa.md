
# OMath.Functions Property (Word)

 **Last modified:** July 28, 2015

Returns an  ** [OMathFunctions](163c6af9-5cb9-93bb-f1c2-b9ebeca28f6a.md)** collection that represents the functions contained within an equation. Read-only.

## Syntax

 _expression_. **Functions**

 _expression_An expression that returns an  ** [OMath](82f2f81b-e2d5-140f-bdcc-8b52b821b24d.md)** object.
