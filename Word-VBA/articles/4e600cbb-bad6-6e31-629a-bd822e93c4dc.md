
# Comments.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Comments** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Comments](e384b37a-50e3-a214-52a8-6fda2acc4991.md)** collection.
