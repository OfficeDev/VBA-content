
# InlineShape.Height Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the height of an inline shape. Read/write  **Single**.

## Syntax

 _expression_. **Height**

 _expression_An expression that represents a  ** [InlineShape](a8fd110a-4aa7-c4b9-1559-32022787d955.md)** object.
