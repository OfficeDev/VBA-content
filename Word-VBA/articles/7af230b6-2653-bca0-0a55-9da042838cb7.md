
# Dialog.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Dialog** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Dialog](f90f6e6d-aaa0-c127-ab37-ca074144eff1.md)** object.
