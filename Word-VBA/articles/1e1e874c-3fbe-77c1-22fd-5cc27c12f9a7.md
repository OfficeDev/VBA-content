
# Column.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified column.

## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents a  ** [Column](49d68571-2a57-6795-34b9-eb09aeb43043.md)** object.
