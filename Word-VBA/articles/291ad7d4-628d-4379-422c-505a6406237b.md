
# FirstLetterExceptions.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of first letter exceptions in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [FirstLetterExceptions](5dc5cc43-a696-d80f-58f9-0f74dfcad0ed.md)** collection.
