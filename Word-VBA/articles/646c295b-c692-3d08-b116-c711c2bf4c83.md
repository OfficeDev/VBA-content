
# XMLNamespaces.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of XML namespaces in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [XMLNamespaces](f11a6cc1-f33d-e1ab-870c-aa5857d66797.md)** collection.
