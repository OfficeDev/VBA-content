
# PivotItem.LabelRange Property (Excel)

 **Last modified:** July 28, 2015

Returns a  ** [Range](b8207778-0dcc-4570-1234-f130532cc8cd.md)** object that represents all the cells in the PivotTable report that contain the item. Read-only.

## Syntax

 _expression_. **LabelRange**

 _expression_A variable that represents a  **PivotItem** object.
