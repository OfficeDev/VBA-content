
# FillFormat.Patterned Method (Excel)

 **Last modified:** July 28, 2015

Sets the specified fill to a pattern.

## Syntax

 _expression_. **Patterned**( **_Pattern_**)

 _expression_A variable that represents a  **FillFormat** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Pattern|Required| ** [MsoPatternType](http://msdn.microsoft.com/library/b95a7e43-329f-b93b-3664-04d8f570c747%28Office.15%29.aspx)**|The type of pattern.|
