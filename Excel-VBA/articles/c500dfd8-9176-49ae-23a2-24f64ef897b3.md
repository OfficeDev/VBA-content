
# OLEObject.SendToBack Method (Excel)

 **Last modified:** July 28, 2015

Sends the object to the back of the z-order.

## Syntax

 _expression_. **SendToBack**

 _expression_A variable that represents an  **OLEObject** object.


### Return Value

Variant
