
# TickLabels.Depth Property (Excel)

 **Last modified:** July 28, 2015

Returns a  **Long** value that represents the number of levels of category tick labels.

## Syntax

 _expression_. **Depth**

 _expression_A variable that represents a  **TickLabels** object.
