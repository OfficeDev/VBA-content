
# Window.DisplayWhitespace Property (Excel)

 **Last modified:** July 28, 2015

 **True** if whitespace is displayed. Read/write **Boolean**.

## Syntax

 _expression_. **DisplayWhitespace**

 _expression_A variable that represents a  **Window** object.
