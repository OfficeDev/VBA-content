
# SpellingOptions.SpanishModes Property (Excel)

 **Last modified:** July 28, 2015

## Syntax

 _expression_A variable that represents a  ** [SpellingOptions](3ba7d0b4-bebb-0cc9-cb50-066d1c19d876.md)** object.
