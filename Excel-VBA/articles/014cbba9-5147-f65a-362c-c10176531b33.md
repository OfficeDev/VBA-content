
# LegendEntry.Width Property (Excel)

 **Last modified:** July 28, 2015

Returns a  **Double** value that represents the width, in points, of the object.

## Syntax

 _expression_. **Width**

 _expression_A variable that represents a  **LegendEntry** object.
