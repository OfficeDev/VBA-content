
# AutoFilter.ShowAllData Method (Excel)

 **Last modified:** July 28, 2015

Displays all the data returned by the  **AutoFilter** object.

## Syntax

 _expression_. **ShowAllData**

 _expression_A variable that represents an  **AutoFilter** object.
