
# Slicer.SlicerCache Property (Excel)

 **Last modified:** July 28, 2015

Returns the  ** [SlicerCache](6e6533e3-0503-a1d3-9ecd-f7997233565f.md)** object associated with the slicer. Read-only.

## Syntax

 _expression_. **SlicerCache**

 _expression_A variable that represents a  ** [Slicer](577be0f6-4eda-0093-8899-097f3c900383.md)** object.


### Return Value

SlicerCache
