
# AddIn.CLSID Property (Excel)

 **Last modified:** July 28, 2015

Returns a read-only unique identifier, or CLSID, identifying an object, as a  **String**.

## Syntax

 _expression_. **CLSID**

 _expression_A variable that represents an  **AddIn** object.
