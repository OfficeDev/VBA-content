
# ShapeNodes.Count Property (Excel)

 **Last modified:** July 28, 2015

Returns an  **Integer** value that represents the number of objects in the collection.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **ShapeNodes** object.
