
# Range.CopyPicture Method (Excel)

 **Last modified:** July 28, 2015

Copies the selected object to the Clipboard as a picture.  **Variant**.

## Syntax

 _expression_. **CopyPicture**( **_Appearance_**,  **_Format_**)

 _expression_A variable that represents a  **Range** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Appearance|Optional| ** [XlPictureAppearance](5fd97feb-a976-2c1c-4051-f2c99d0fa793.md)**|. Specifies how the picture should be copied.|
|Format|Optional| ** [XlCopyPictureFormat](a764bd52-1c2b-9395-7774-1daceb6a8415.md)**|. The format of the picture.|

### Return Value

Variant
