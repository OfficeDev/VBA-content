
# Window.NewWindow Method (Excel)

 **Last modified:** July 28, 2015

Creates a new window or a copy of the specified window.

## Syntax

 _expression_. **NewWindow**

 _expression_A variable that represents a  **Window** object.


### Return Value

Window
