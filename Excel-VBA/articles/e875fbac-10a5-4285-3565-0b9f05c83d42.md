
# LineFormat.Weight Property (Excel)

 **Last modified:** July 28, 2015

Returns or sets a  **Single** value that represents the weight of the line.

## Syntax

 _expression_. **Weight**

 _expression_A variable that represents a  **LineFormat** object.
