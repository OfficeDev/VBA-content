
# LegendKey.Smooth Property (Excel)

 **Last modified:** July 28, 2015

 **True** if curve smoothing is turned on for the legend key. Read/write.

## Syntax

 _expression_. **Smooth**

 _expression_A variable that represents a  **LegendKey** object.
