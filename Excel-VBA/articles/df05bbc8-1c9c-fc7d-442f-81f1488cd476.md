
# PageSetup.Pages Property (Excel)

 **Last modified:** July 28, 2015

Returns or sets the count or item number of the pages in  **Pages** collection.

## Syntax

 _expression_. **Pages**

 _expression_A variable that represents a  **PageSetup** object.
