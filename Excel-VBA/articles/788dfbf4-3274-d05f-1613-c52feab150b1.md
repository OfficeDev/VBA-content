
# ColorStop.Position Property (Excel)

 **Last modified:** July 28, 2015

Returns or sets the position of the  **ColorStop**. Read/write

## Syntax

 _expression_. **Position**

 _expression_A variable that represents a  **ColorStop** object.


### Return Value

Double
