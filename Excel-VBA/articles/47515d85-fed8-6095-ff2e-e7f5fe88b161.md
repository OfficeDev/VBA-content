
# OLEObject.Height Property (Excel)

 **Last modified:** July 28, 2015

Returns or sets a  **Double** value that represents the height, in points, of the object.

## Syntax

 _expression_. **Height**

 _expression_A variable that represents an  **OLEObject** object.
