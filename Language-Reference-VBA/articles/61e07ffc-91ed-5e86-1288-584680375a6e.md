
# CreateFolder Method

 **Last modified:** July 28, 2015


 **Description**
Creates a folder.
 **Syntax**
 _object_. **CreateFolder(**_foldername_**)**
The  **CreateFolder** method has these parts:


|**Part**|**Description**|
|:-----|:-----|
| _object_|Required. Always the name of a  **FileSystemObject**.|
| _foldername_|Required.  [String expression](b8bdf64f-5920-1ae9-16d0-b26d09524a30.md) that identifies the folder to create.|
 **Remarks**
An error occurs if the specified folder already exists.
