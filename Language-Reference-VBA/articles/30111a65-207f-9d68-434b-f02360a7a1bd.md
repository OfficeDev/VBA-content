
# VBProjects Property

 **Last modified:** July 28, 2015


Returns the VBProjects collection, which represents all of the projects currently open in the Visual Basic IDE.
 **Syntax**
 _object_. **VBProjects**
The  _object_ placeholder represents an [object expression](b8bdf64f-5920-1ae9-16d0-b26d09524a30.md) that evaluates to an object in the Applies To list.
