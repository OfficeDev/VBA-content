
# Point.Format Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns the line, fill, and effect formatting for the object. Read-only  ** [ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)**.

## Syntax

 _expression_A variable that represents a  ** [Point](e0137fdd-5632-88d7-a6c0-57a76717e736.md)** object.
