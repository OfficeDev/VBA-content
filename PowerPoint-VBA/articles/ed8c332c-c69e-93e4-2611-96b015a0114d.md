
# CustomLayout.Shapes Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns a  ** [Shapes](eb208855-254e-1a0f-884b-4a5edcfd584d.md)** collection representing all the layout elements included in the specified custom layout. Read-only.

## Syntax

 _expression_. **Shapes**

 _expression_A variable that represents a  **CustomLayout** object.


### Return Value

Shapes


## Remarks

The  **Shapes** collection of layout elements can include drawings, shapes, OLE objects, pictures, text objects, titles, headers, footers, slide numbers, and date and time objects.
