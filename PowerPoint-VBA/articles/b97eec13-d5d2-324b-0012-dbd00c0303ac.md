
# Chart.Shapes Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns a collection that represents all the shapes on the chart sheet. Read-only  ** [Shapes](eb208855-254e-1a0f-884b-4a5edcfd584d.md)**.

## Syntax

 _expression_. **Shapes**

 _expression_A variable that represents a  ** [Chart](3fcf082f-9f58-f67d-1061-e7f37e30fbcd.md)** object.
