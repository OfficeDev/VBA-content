
# Font Properties (PowerPoint)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](70e38091-9f12-74c7-18b9-13474ac26644.md)|Returns an  ** [Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)**object that represents the creator of the specified object.|
| [AutoRotateNumbers](621ccc86-d5cb-d2c1-262f-5652eff5800a.md)|Returns or sets lateral compression. Read/write.|
| [BaselineOffset](aa948e2e-957c-ff4c-16b9-480d7f5f2d24.md)|Returns or sets the baseline offset for the specified superscript or subscript characters. Read/write.|
| [Bold](13e81c46-5ae7-21ee-58e1-5ab23de552d5.md)|Determines whether the character format is bold. Read/write.|
| [Color](461d3fdc-5097-ceca-76f6-81d924f8a7b7.md)|Returns the color of the current  **Font** object. Read-only.|
| [Embeddable](50824587-0371-e7eb-8885-370f97b8bf0c.md)|Determines whether the specified font can be embedded in the presentation. Read-only.|
| [Embedded](3fd7fe50-19a9-9944-f7c8-0ba54bc07c93.md)|Determines whether the specified font is embedded in the presentation. Read-only.|
| [Emboss](734b5bd7-4b1f-d3b3-d8bd-f73d0bc86f67.md)|Determines whether the character format is embossed. Read/write.|
| [Italic](5fc7e3fe-e103-72ea-42cb-c178b411312a.md)| **True** if the font or range is formatted as italic. Read/write **Long**.|
| [Name](6798b75b-7fb8-a046-1532-a8cc41b76af8.md)|Returns or sets the name of the specified object. Read/write.|
| [NameAscii](06db0f5b-71ac-704d-eef2-1be8a96fb7a8.md)|Returns or sets the font used for ASCII characters (characters with character set numbers within the range of 0 to 127). Read/write.|
| [NameComplexScript](ef1e44d6-ff5d-aaa9-4eaa-643cb2ebc2bf.md)|Returns or sets the complex script font name. Used for mixed language text. Read/write.|
| [NameFarEast](0b3f7d98-bda5-eec3-f570-20d8b575c0a3.md)|Returns or sets the Asian font name. Read/write.|
| [NameOther](64f62838-635c-9b6d-082a-06fe698685e1.md)|Returns or sets the font used for characters whose character set numbers are greater than 127. Read/write.|
| [Parent](5cf96dc7-aa6a-e3f6-d8f3-c0b92d6b1a6a.md)|Returns the parent object for the specified object.|
| [Shadow](37d23e3a-26a7-ba20-1e23-13861090ae79.md)|Determines whether the specified text has a shadow. Read/write.|
| [Size](dd56a4e9-20c7-b38d-0d0e-82e5326d51c4.md)|Returns or sets the character size, in points. Read/write.|
| [Subscript](ad23433b-b14b-9b2a-3bf6-772de41995f7.md)|Determines whether the specified text is subscript. Read/write.|
| [Superscript](6f0bba73-f375-d715-3ddb-f1ab6041336c.md)|Determines whether the specified text is superscript. Read/write.|
| [Underline](ee21ab18-b131-7e4d-de19-93c9b7549d3b.md)|Determines whether the specified text (for the  **Font** object) or the font style (for the **FontInfo** object) is underlined. Read/write.|
