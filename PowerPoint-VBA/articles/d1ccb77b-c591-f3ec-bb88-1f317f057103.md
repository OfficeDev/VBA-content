
# SlideShowView.EraseDrawing Method (PowerPoint)

 **Last modified:** July 28, 2015

Removes lines drawn during a slide show by using either the  ** [DrawLine](d4c3c1c9-cd12-67ba-b1b9-4d7e924bd084.md)**method or the pen tool.

## Syntax

 _expression_. **EraseDrawing**

 _expression_A variable that represents a  **SlideShowView** object.


## Example


```
SlideShowWindows(1).View.EraseDrawing
```
