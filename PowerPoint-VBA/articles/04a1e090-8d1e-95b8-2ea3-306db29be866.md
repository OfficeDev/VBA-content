
# Table.Style Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns a  ** [TableStyle](2191b68d-2dae-3453-6904-3689f34eeaf9.md)** object that contains information about the specified table's current table style. Read-only.

## Syntax

 _expression_. **Style**

 _expression_A variable that represents a  **Table** object.


### Return Value

TableStyle
