
# TextStyle.Levels Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns a  ** [TextStyleLevels](dc61e97f-e92e-d214-47af-5830c14b1b09.md)** object that represents outline text formatting. Read-only.

## Syntax

 _expression_. **Levels**

 _expression_A variable that represents a  **TextStyle** object.
