
# CustomLayout.Parent Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns a reference to the slide master object that is the parent of the specified  ** [CustomLayout](67829704-0314-aed2-5415-6736cefc197e.md)** object. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **CustomLayout** object.


### Return Value

Object
