
# Comment.Text Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns a  **String** that represents the text in a comment. Read-only.

## Syntax

 _expression_. **Text**

 _expression_A variable that represents a  **Comment** object.


### Return Value

String
