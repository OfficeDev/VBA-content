
# DocumentWindows.Item Method (PowerPoint)

 **Last modified:** July 28, 2015

Returns a single  **DocumentWindow** object from the specified **DocumentWindows** collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **DocumentWindows** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The index number of the single  **DocumentWindow** object in the collection to be returned.|

### Return Value

DocumentWindow
