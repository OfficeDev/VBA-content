
# ThreeDFormat.ContourColor Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns a  ** [ColorFormat](3bfcd08d-65f4-25a3-2d05-77111fbd13e5.md)** object that represents the color of the specified **ThreeDFormat** object's contour. Read-only.

## Syntax

 _expression_. **ContourColor**

 _expression_A variable that represents a  **ThreeDFormat** object.


### Return Value

ColorFormat
