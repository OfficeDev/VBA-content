
# ShadowFormat.Blur Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns or sets the blur radius of the specified shadow, in points. Read/write.

## Syntax

 _expression_. **Blur**

 _expression_A variable that represents a  **ShadowFormat** object.


### Return Value

Single
