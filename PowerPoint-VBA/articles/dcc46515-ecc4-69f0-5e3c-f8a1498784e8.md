
# AnimationBehavior.Delete Method (PowerPoint)

 **Last modified:** July 28, 2015

Deletes the specified  **AnimationBehavior** object.

## Syntax

 _expression_. **Delete**

 _expression_A variable that represents an  **AnimationBehavior** object.
