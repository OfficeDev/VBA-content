
# CustomLayouts.Count Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns the number of objects in the specified collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **CustomLayouts** object.


### Return Value

Long
