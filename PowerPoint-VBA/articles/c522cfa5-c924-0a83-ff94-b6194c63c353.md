
# ChartGroups Members (PowerPoint)
Represents one or more series plotted in a chart with the same format.

 **Last modified:** July 28, 2015


## Methods



|**Name**|**Description**|
|:-----|:-----|
| [Item](0b04a471-d726-f400-062c-8d4a7dc9c752.md)|Returns a single object from a collection.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](cd827f9c-5695-0b9d-9404-bbed1aba8322.md)|When used without an object qualifier, returns an  ** [Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)**object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an  **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
| [Count](184188d8-37b8-d057-5378-a1d649d528da.md)|Returns the number of objects in the collection. Read-only  **Long**.|
| [Creator](6c3f50b0-9877-220a-26cc-3fb94765535a.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
| [Parent](4dcddb95-7af2-bc28-0dbc-affb4464ced9.md)|Returns the parent for the specified object. Read-only  **Object**.|
