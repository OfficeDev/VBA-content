
# HeadersFooters.Clear Method (PowerPoint)

 **Last modified:** July 28, 2015

Clears the header and footer from the specified slide or slides.

## Syntax

 _expression_. **Clear**

 _expression_A variable that represents a  **HeadersFooters** object.
