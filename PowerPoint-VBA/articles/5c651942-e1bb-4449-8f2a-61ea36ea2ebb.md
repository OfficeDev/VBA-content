
# UpBars Properties (PowerPoint)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](40d5f349-bd76-18e3-a791-fdd2fd6f9efb.md)|When used without an object qualifier, returns an  ** [Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)**object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an  **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
| [Creator](e7f69c86-a8c4-e4c6-83fc-fc7b3e32d143.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
| [Fill](56609f59-253a-c1ba-c133-6a4be91dd837.md)|Returns a  ** [FillFormat](5bd4e2cb-4466-b468-d494-bec30ed5c9d8.md)**object for the parent chart element that contains fill formatting properties for the chart element. Read-only.|
| [Format](05f7e68f-51ba-e03e-836a-3f3d6c8259c3.md)|Returns the line, fill, and effect formatting for the object. Read-only  ** [ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)**.|
| [Name](756dc379-ce00-7417-9ab0-1ca44a82f5a1.md)|Returns the name of the object. Read-only  **String**.|
| [Parent](65b7baf1-88dd-cf8e-93a8-8c0d3980fc18.md)|Returns the parent for the specified object. Read-only  **Object**.|
