
# Table.Title Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns or sets the title of the  **Table** object. Read/write.

## Syntax

 _expression_. **Title**

 _expression_A variable that represents a  **Table** object.


### Return Value

String
