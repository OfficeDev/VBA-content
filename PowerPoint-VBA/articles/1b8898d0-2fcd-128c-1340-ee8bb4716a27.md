
# Tags.Delete Method (PowerPoint)

 **Last modified:** July 28, 2015

Deletes a tag.

## Syntax

 _expression_. **Delete**( **_Name_**)

 _expression_A variable that represents a  **Tags** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Name|Required| **String**|Tthe name of the tag to be deleted.|
