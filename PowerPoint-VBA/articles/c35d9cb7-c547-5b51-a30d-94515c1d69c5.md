
# TextFrame2.Parent Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns the parent object of the specified test frame. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_An expression that returns a  **TextFrame2** object.


### Return Value

Object
