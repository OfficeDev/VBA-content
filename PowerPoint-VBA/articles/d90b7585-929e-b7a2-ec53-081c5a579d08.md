
# SectionProperties.Name Method (PowerPoint)

 **Last modified:** July 28, 2015

Returns the name of the specified section as a  **String**.

## Syntax

 _expression_. **Name**( **_sectionIndex_**)

 _expression_A variable that represents a  **SectionProperties** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|sectionIndex|Required| **Integer**|The index of the section in the  **SectionProperties** colleciton.|

### Return Value

String
