
# LegendKey.Smooth Property (PowerPoint)

 **Last modified:** July 28, 2015

 **True** if curve smoothing is turned on for the legend key. Read/write **Boolean**.

## Syntax

 _expression_. **Smooth**

 _expression_A variable that represents a  ** [LegendKey](98e8b9c3-b53e-9595-9389-6f92a6d730f4.md)** object.
