
# ColorFormat Properties (PowerPoint)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](b911e512-fae0-990d-36b1-85e8f9d74e08.md)|Returns an  ** [Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)**object that represents the creator of the specified object.|
| [Brightness](5140244e-d70b-f764-c127-917506b4074d.md)|Returns or sets the brightness of the specified picture or OLE object. The value for this property must be a number from 0.0 (dimmest) to 1.0 (brightest). Read/write  **Single**.|
| [Creator](403c2188-999b-07e8-ff60-16147dd9cb55.md)|Returns a  **Long** that represents the four-character creator code for the application in which the specified object was created. For example, if the object was created in Microsoft PowerPoint, this property returns the hexadecimal number 50575054. Read-only.|
| [ObjectThemeColor](40264b94-b16d-2a52-9adc-8e8510ec581d.md)|Returns or sets the theme color of the specified  **ColorFormat** object. Read/Write.|
| [Parent](aa54269d-ee71-0a4e-7063-8631577d6e76.md)|Returns the parent object for the specified object.|
| [RGB](5bb68052-5931-2096-277c-fb44c76b37eb.md)|Returns or sets the red-green-blue (RGB) value of the specified color. Read/write.|
| [SchemeColor](4c62e7a7-ce53-c93e-9ec5-b299a18f5bf7.md)|Returns or sets the color in the applied color scheme that's associated with the specified object. Read/write.|
| [TintAndShade](01f3581c-eccf-d627-5ba0-f560f58e6e0f.md)|Sets or returns the lightening or darkening of the the color of a specified shape. Read/write.|
| [Type](289f0d88-2b8c-ea7c-38bb-875a9fc1a3dc.md)|Represents the type of color. Read-only.|
