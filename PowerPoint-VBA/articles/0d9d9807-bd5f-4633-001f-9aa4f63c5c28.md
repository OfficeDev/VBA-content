
# SlideShowView.MediaControlsVisible Property (PowerPoint)

 **Last modified:** July 28, 2015

Indicates whether the media controls are visible. Read-only.

## Syntax

 _expression_. **MediaControlsVisible**

 _expression_A variable that represents a  **SlideShowView** object.


### Return Value

 ** [MsoTriState](http://msdn.microsoft.com/library/2036cfc9-be7d-e05c-bec7-af05e3c3c515%28Office.15%29.aspx)**
