
# LegendKey.PictureType Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns or sets the way pictures are displayed on a legend key. Read/write  ** [XlChartPictureType](ed2a241c-1362-16c3-660a-781d7915257e.md)**.

## Syntax

 _expression_. **PictureType**

 _expression_A variable that represents a  ** [LegendKey](98e8b9c3-b53e-9595-9389-6f92a6d730f4.md)** object.
