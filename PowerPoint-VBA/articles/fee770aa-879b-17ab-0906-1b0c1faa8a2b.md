
# Axis.Border Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns the border of the object. Read-only  ** [ChartBorder](fd651a9a-4068-9a9b-f605-9228da5e6183.md)**.

## Syntax

 _expression_. **Border**

 _expression_A variable that represents an  ** [Axis](38d5e006-ac32-7bdb-f9f0-e8a858dcbf49.md)** object.
