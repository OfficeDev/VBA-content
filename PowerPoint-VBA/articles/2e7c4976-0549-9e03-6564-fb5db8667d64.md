
# DataLabel.Orientation Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns or sets the text orientation. Read/write  **Long**.

## Syntax

 _expression_. **Orientation**

 _expression_A variable that represents a  ** [DataLabel](a17d23c5-0361-9129-28e5-b892f6966bda.md)** object.


## Remarks

The value of this property can be an integer from -90 through 90 degrees or one of the  ** [XlOrientation](df963ba0-dc15-689a-4922-8b0b0ec6f147.md)** constants.
