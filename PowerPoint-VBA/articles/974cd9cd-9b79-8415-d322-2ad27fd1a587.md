
# MotionEffect.Path Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns a  **String** that represents the path followed by a ** [MotionEffect](77a34f68-8806-22b8-149f-c28e0457e7e9.md)**object. Read-only.

## Syntax

 _expression_. **Path**

 _expression_A variable that represents a  **MotionEffect** object.


### Return Value

String


## Remarks

The  **String** returned for a **MotionEffect** object is a specific path that the motion effect follows between From and To, using the same syntax as the VML path description.
