
# SlideShowView.FirstAnimationIsAutomatic Method (PowerPoint)

 **Last modified:** July 28, 2015

Returns  **True** if the current slide has an initial animation that runs automically.

## Syntax

 _expression_. **FirstAnimationIsAutomatic**

 _expression_A variable that represents a  **SlideShowView** object.


### Return Value

Boolean
