
# LegendKey Properties (PowerPoint)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](7d1d0aab-32b0-b65e-53ef-be2043f4cd21.md)|When used without an object qualifier, returns an  ** [Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)**object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an  **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
| [Creator](eebe01a6-a5ad-c0b7-7799-7ce5659349c8.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
| [Format](9b91b871-d6cc-f149-9268-d12cf236694c.md)|Returns the line, fill, and effect formatting for the object. Read-only  ** [ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)**.|
| [Height](f12c9287-99a8-3d68-1c22-79c8e84b02e9.md)|Returns the height, in points, of the object. Read-only  **Double**.|
| [InvertIfNegative](2a865359-2436-0d8b-553c-970f28da8125.md)| **True** if Microsoft Word inverts the pattern in the object when it corresponds to a negative number. Read/write **Variant**.|
| [Left](79cb8efe-47a1-7d2d-c2b7-5496072ce5f7.md)|Returns the distance, in points, from the left edge of the object to the left edge of the chart area. Read-only  **Double**.|
| [MarkerBackgroundColor](46553e06-6468-1b6c-03ae-6930c0700565.md)|Sets the marker background color as an RGB value or returns the corresponding color index value. Read/write  **Long**.|
| [MarkerBackgroundColorIndex](882d5408-9d75-d452-ea53-1f7f2d575752.md)|Returns or sets the marker background color as an index into the current color palette, or as one of the following  ** [XlColorIndex](e9d26c5b-26f8-7baf-5f53-4beac789add3.md)** constants: **xlColorIndexAutomatic** or **xlColorIndexNone**. Read/write  **Long**.|
| [MarkerForegroundColor](6b8020ba-ccd1-fb3a-6ea9-8a59eaa64d3b.md)|Sets the marker foreground color as an RGB value or returns the corresponding color index value. Read/write  **Long**.|
| [MarkerForegroundColorIndex](47760c8c-a791-fac5-a5cc-d91c59221026.md)|Returns or sets the marker foreground color as an index into the current color palette, or as one of the following  ** [XlColorIndex](e9d26c5b-26f8-7baf-5f53-4beac789add3.md)** constants: **xlColorIndexAutomatic** or **xlColorIndexNone**. Read/write  **Long**.|
| [MarkerSize](712765a9-5a9a-5712-fc12-e1a477d1080c.md)|Returns or sets the data-marker size, in points. Read/write  **Long**.|
| [MarkerStyle](146c9146-063e-8456-c61e-e1286eae14c1.md)|Returns or sets the marker style for a point or series in a line chart, scatter chart, or radar chart. Read/write  ** [XlMarkerStyle](ec213196-0993-526a-48ca-80f92385a0bf.md)**.|
| [Parent](aaf91d7a-ccff-8517-1afb-5ca47bb5ac42.md)|Returns the parent for the specified object. Read-only  **Object**.|
| [PictureType](9f605c49-eb0f-2e04-304a-34fbf15143b1.md)|Returns or sets the way pictures are displayed on a legend key. Read/write  ** [XlChartPictureType](ed2a241c-1362-16c3-660a-781d7915257e.md)**.|
| [PictureUnit2](018b1a1e-2d00-abee-eeed-8886baec9f4a.md)|Returns or sets the unit for each picture on the chart if the  ** [PictureType](9f605c49-eb0f-2e04-304a-34fbf15143b1.md)** property is set to **xlStackScale**; otherwise, this property is ignored. Read/write  **Double**.|
| [Shadow](67153c37-8b97-56c8-7ea6-cd90e1dd51cc.md)|Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean**.|
| [Smooth](7ba0f917-8e65-047d-513c-004ecef82aac.md)| **True** if curve smoothing is turned on for the legend key. Read/write **Boolean**.|
| [Top](bcce6f18-39c3-9445-2a5a-dea0b242a816.md)|Returns the distance, in points, from the top edge of the object to the top of the first row (on a worksheet) or the top of the chart area (on a chart). Read-only  **Double**.|
| [Width](8ff18557-1079-f1fd-96a7-02097ccb574b.md)|Returns the width, in points, of the object. Read-only  **Double**.|
