
# PrintRanges.Item Method (PowerPoint)

 **Last modified:** July 28, 2015

Returns a single  **PrintRange** object from the specified **PrintRanges** collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **PrintRanges** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The index number of the single  **PrintRange** object in the collection to be returned.|

### Return Value

PrintRange
