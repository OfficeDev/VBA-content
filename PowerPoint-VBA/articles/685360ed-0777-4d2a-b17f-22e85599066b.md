
# ShapeNodes Properties (PowerPoint)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](592142a3-ced3-17df-ab13-5dae22ddf08b.md)|Returns an  ** [Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)**object that represents the creator of the specified object.|
| [Count](63f8a4da-1b0a-b72c-06ed-27477fb74809.md)|Returns the number of objects in the specified collection. Read-only.|
| [Creator](d0cff272-8da0-05c0-49ac-ec27bd5de6d4.md)|Returns a  **Long** that represents the four-character creator code for the application in which the specified object was created. For example, if the object was created in Microsoft PowerPoint, this property returns the hexadecimal number 50575054. Read-only.|
| [Parent](726e80d7-c8f4-918f-cb01-96b26c0afb59.md)|Returns the parent object for the specified object.|
