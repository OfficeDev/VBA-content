
# CellRange Members (PowerPoint)
A collection of  ** [Cell](e89e5d69-33b1-d7b1-0a6c-4dfd8b676977.md)**objects in a table column or row. The  **CellRange** collection represents all the cells in the specified column or row. To use the **CellRange** collection, use the **Cells** keyword.

 **Last modified:** July 28, 2015


## Methods



|**Name**|**Description**|
|:-----|:-----|
| [Item](69960691-7853-3068-19c5-b757e22024d9.md)|Returns a single  **Cell** object from the specified **CellRange** collection.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](8f3aaa0e-05ee-71d5-df33-6b449fa893ba.md)|Returns an  ** [Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)**object that represents the creator of the specified object.|
| [Borders](06bd16b9-8d3e-d818-cdf4-44e0dfbaca5c.md)|Returns a  ** [Borders](af3b8d8b-9214-b1ac-f12e-0be244b60b08.md)**collection that represents the borders and diagonal lines for the specified  **Cell** object or **CellRange** collection. Read-only.|
| [Count](9f81da2d-1b5d-9650-0631-19319dcc4bc0.md)|Returns the number of objects in the specified collection. Read-only.|
| [Parent](9e19b9d0-eff7-f820-fee0-57b8b024eba2.md)|Returns the parent object for the specified object.|
