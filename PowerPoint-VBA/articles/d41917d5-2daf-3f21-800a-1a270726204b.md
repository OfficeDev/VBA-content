
# Theme Properties (PowerPoint)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](d33d9d1b-642b-a0da-4439-281e4a014f55.md)|When used without an object qualifier, returns an  ** [Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
| [Parent](a4332e6e-cbed-42bc-ef24-cabcde111c57.md)|Returns the parent for the specified object. Read-only  **Object**.|
| [ThemeVariants](f88d8c2c-4964-7392-5457-25216ece92d0.md)|Returns a  ** [ThemeVariants](de00374f-05fd-4cae-08f8-ef417cd944b5.md)** collection that represents the variations in the theme.|
