
# CustomLayout.Index Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns the index position of the specified custom layout in the  ** [CustomLayouts](9ce682fb-545c-55cb-e9ac-3475f7556af1.md)** collection. Read-only.

## Syntax

 _expression_. **Index**

 _expression_A variable that represents a  **CustomLayout** object.


### Return Value

Long
