
# DataLabels.NumberFormatLocal Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns or sets the format code for the object as a string in the language of the user. Read/write  **Variant**.

## Syntax

 _expression_. **NumberFormatLocal**

 _expression_A variable that represents a  ** [DataLabels](a0d0b0ec-6a12-9a5c-1026-1e1d85e488fa.md)** object.


## Remarks

The  **Format** function uses different format code strings than do the ** [NumberFormat](5007f141-9ed1-1337-d8b9-9d28903db7ce.md)** and **NumberFormatLocal** properties.
