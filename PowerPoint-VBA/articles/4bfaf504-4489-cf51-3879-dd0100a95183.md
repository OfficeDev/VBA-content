
# ChartCharacters.Caption Property (PowerPoint)

 **Last modified:** July 28, 2015

## Syntax

 _expression_A variable that represents a  ** [ChartCharacters](2f659f71-f277-dab4-f2bd-631c7a2424de.md)** object.
