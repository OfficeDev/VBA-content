
# TickLabels.NumberFormat Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns or sets the format code for the object. Read/write  **String**.

## Syntax

 _expression_. **NumberFormat**

 _expression_A variable that represents a  ** [TickLabels](2ba878bf-3a76-1350-2bd4-615c2520f042.md)** object.


## Remarks

The format code is the same string as the  **Format Codes** option in the **Format Cells** dialog box. The **Format** function uses different format code strings than do the **NumberFormat** and ** [NumberFormatLocal](e9415f73-32aa-2043-06e5-7a796749e823.md)** properties.
