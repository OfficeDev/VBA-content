
# ExtraColors.Item Method (PowerPoint)

 **Last modified:** July 28, 2015

Returns a single color from the specified  **ExtraColors** collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents an  **ExtraColors** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The index number of the single object in the collection to be returned.|

### Return Value

MsoRGBType
