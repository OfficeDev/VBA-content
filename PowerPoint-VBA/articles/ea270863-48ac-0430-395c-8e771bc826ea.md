
# CustomerData.Application Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns an  ** [Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)** object that represents the creator of the specified **CustomerData** object. Read-only.

## Syntax

 _expression_. **Application**

 _expression_An expression that returns a  **CustomerData** object.


### Return Value

Application
