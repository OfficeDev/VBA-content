
# MediaFormat.Muted Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns whether audio playback of the media is muted. Read/write

## Syntax

 _expression_. **Muted**

 _expression_An expression that returns a  **MediaFormat** object.


### Return Value

Boolean
