
# TickLabels.NumberFormatLocal Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns or sets the format code for the object as a string in the language of the user. Read/write  **Variant**.

## Syntax

 _expression_. **NumberFormatLocal**

 _expression_A variable that represents a  ** [TickLabels](2ba878bf-3a76-1350-2bd4-615c2520f042.md)** object.


## Remarks

The  **Format** function uses different format code strings than do the ** [NumberFormat](aeb2a6fb-b62f-bd74-dd71-aaf527969925.md)** and **NumberFormatLocal** properties.
