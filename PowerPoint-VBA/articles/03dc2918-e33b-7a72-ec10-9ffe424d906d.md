
# AxisTitle.VerticalAlignment Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns or sets the vertical alignment of the specified object. Read/write  **Variant**.

## Syntax

 _expression_. **VerticalAlignment**

 _expression_A variable that represents an  ** [AxisTitle](8eddc95c-2353-43fa-c055-ee76de28009d.md)** object.


## Remarks

The value of this property can be set to one of the following  ** [XlConstants](fe30391f-83ee-3259-620f-d1224ab92d5f.md)** constants:


-  **xlBottom**
    
-  **xlCenter**
    
-  **xlDistributed**
    
-  **xlJustify**
    
-  **xlTop**
