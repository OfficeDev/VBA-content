
# ShadowFormat.Type Property (PowerPoint)

 **Last modified:** July 28, 2015

Represents the type of shadow. Read/write.

## Syntax

 _expression_. **Type**

 _expression_A variable that represents a  **ShadowFormat** object.


### Return Value

MsoShadowType
