
# PrintRanges.Add Method (PowerPoint)

 **Last modified:** July 28, 2015

Returns a  ** [PrintRange](62f098b3-5e67-8fa4-3af9-4507160fa1ad.md)** object that represents a new print range to be added to the **PrintRanges** collection.

## Syntax

 _expression_. **Add**( **_Start_**,  **_End_**)

 _expression_A variable that represents a  **PrintRanges** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Start|Required| **Long**|The number of the slide that is the starting slide in the range.|
|End|Required| **Long**|The number of the slide that is the ending slide in the range.|

### Return Value

PrintRange
