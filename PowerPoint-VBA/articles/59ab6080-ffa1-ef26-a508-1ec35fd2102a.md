
# SlideRange.CustomLayout Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns a  ** [CustomLayout](67829704-0314-aed2-5415-6736cefc197e.md)** object that represents the custom layout associated with the specified range of slides. Read-only.

## Syntax

 _expression_. **CustomLayout**

 _expression_A variable that represents a  **SlideRange** object.


### Return Value

CustomLayout
