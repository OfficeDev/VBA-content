
# ChartFormat.ThreeD Property (PowerPoint)

 **Last modified:** July 28, 2015

## Syntax

 _expression_A variable that represents a  ** [ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)** object.
