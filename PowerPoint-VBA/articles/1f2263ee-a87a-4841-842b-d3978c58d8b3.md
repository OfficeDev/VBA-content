
# MotionEffect Properties (PowerPoint)

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](c85f081d-d6b7-d628-7cff-95c1da8eb74e.md)|Returns an  ** [Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)**object that represents the creator of the specified object.|
| [ByX](d25b2fda-1ca1-535f-82c9-aed0f59f090c.md)|Sets or returns a  **Single** that represents scaling or moving an object horizontally by a specified percentage of the screen width, depending on whether it used in conjunction with a ** [ScaleEffect](cb7c296e-a9ea-4ed6-87e0-a5d603da4f9f.md)**or  ** [MotionEffect](77a34f68-8806-22b8-149f-c28e0457e7e9.md)**object, respectively. For example, a value of 50 for a motion effect means to move the object half the screen width to the right. Read/write.|
| [ByY](6797b2bd-edab-9bd2-e234-ffb9da57352a.md)|Sets or returns a  **Single** that represents scaling or moving an object vertically by a specified percentage of the screen width, depending on whether it is used in conjunction with a ** [ScaleEffect](cb7c296e-a9ea-4ed6-87e0-a5d603da4f9f.md)**or  ** [MotionEffect](77a34f68-8806-22b8-149f-c28e0457e7e9.md)**object, respectively. Read/write.|
| [FromX](1e849f47-aab0-3e54-6cfe-e7fe3021dcf7.md)|Sets or returns a  **Single** that represents the starting width or horizontal position of a ** [MotionEffect](77a34f68-8806-22b8-149f-c28e0457e7e9.md)**object, specified as a percent of the screen width. Read/write.|
| [FromY](e9754587-f8bc-4af5-bd2b-ec6b1e3c4bc4.md)|Returns or sets a  **Single** that represents the vertical position of a ** [MotionEffect](77a34f68-8806-22b8-149f-c28e0457e7e9.md)**object, specified as a percentage of the screen width. Read/write.|
| [Parent](6c46a46c-14f3-b61e-e381-87ec0eff8974.md)|Returns the parent object for the specified object.|
| [Path](974cd9cd-9b79-8415-d322-2ad27fd1a587.md)|Returns a  **String** that represents the path followed by a ** [MotionEffect](77a34f68-8806-22b8-149f-c28e0457e7e9.md)**object. Read-only.|
| [ToX](fd597d99-0d33-f9e2-a179-b3d5efd54236.md)|Sets or returns a  **Single** that represents the horizontal position of a ** [MotionEffect](77a34f68-8806-22b8-149f-c28e0457e7e9.md)**object, specified as a percent of the screen width. Read/write.|
| [ToY](e6f7a6ca-0eca-4698-512e-2e0339513bc1.md)|Returns or sets a  **Single** that represents the vertical position of a ** [MotionEffect](77a34f68-8806-22b8-149f-c28e0457e7e9.md)**object, specified as a percentage of the screen width. Read/write.|
