
# ResampleMediaTasks.Pause Method (PowerPoint)

 **Last modified:** July 28, 2015

Pauses the media represented by the specified object.

## Syntax

 _expression_. **Pause**

 _expression_A variable that represents a  **ResampleMediaTasks** object.
