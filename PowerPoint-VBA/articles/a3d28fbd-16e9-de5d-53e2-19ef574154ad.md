
# Chart.Title Property (PowerPoint)

 **Last modified:** July 28, 2015

Gets or sets a  **String** that represents the title of the chart. Read/write.

## Syntax

 _expression_. **Title**

 _expression_A variable that represents a  **Chart** object.
