
# BulletFormat.Type Property (PowerPoint)

 **Last modified:** July 28, 2015

Represents the type of bullet. Read/write.

## Syntax

 _expression_. **Type**

 _expression_A variable that represents a  **BulletFormat** object.


### Return Value

PpBulletType


## Remarks

The value of the  **Type** property can be one of these **PpBulletType** constants.



| **ppBulletMixed**|
| **ppBulletNone**|
| **ppBulletNumbered**|
| **ppBulletPicture**|
| **ppBulletUnnumbered**|
