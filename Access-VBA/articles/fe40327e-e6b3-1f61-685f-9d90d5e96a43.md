
# WorksheetFunction.Phi Method (Excel)

 **Last modified:** July 28, 2015

Returns the value of the density function for a standard normal distribution.

## Syntax

 _expression_. **Phi**(Arg1)

 _expression_A variable that represents a  [WorksheetFunction Object (Excel)](7b1d5639-363d-632c-2cf0-2232562646b6.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
|Arg1|Required|DOUBLE|Number for which you want the density of the standard normal distribution.|

### Return value

 **DOUBLE**
