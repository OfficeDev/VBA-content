
# ProtectedViewWindows.Parent Property (Excel)

 **Last modified:** July 28, 2015

Returns the parent object for the specified object. Read-only

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **ProtectedViewWindows** object.
