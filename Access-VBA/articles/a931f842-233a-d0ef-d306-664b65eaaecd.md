
# PivotTable.Value Property (Excel)

 **Last modified:** July 28, 2015

Returns or sets a  **String** value that represents the name of the PivotTable report.

## Syntax

 _expression_. **Value**

 _expression_A variable that represents a  **PivotTable** object.
