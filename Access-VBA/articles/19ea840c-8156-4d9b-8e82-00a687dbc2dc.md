
# Worksheet.BeforeDelete Event (Excel)

 **Last modified:** July 28, 2015

Occurs before the worksheet is deleted.

## Syntax

 _expression_. **BeforeDelete**

 _expression_A variable that represents a  **Worksheet** object.


### Return Value

None
