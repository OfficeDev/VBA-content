
# TextEffectFormat.Text Property (Excel)

 **Last modified:** July 28, 2015

Returns or sets the text for the specified object. Read/write  **String**.

## Syntax

 _expression_. **Text**

 _expression_A variable that represents a  **TextEffectFormat** object.
