
# CommandButton.VerticalAnchor Property (Access)

 **Last modified:** July 28, 2015

Gets or sets an  ** [AcVerticalAnchor](08f16c8b-1566-cfad-795a-cb65a91c4e52.md)** constant that indicates how the specified button is anchored vertically within its layout. Read/write.

## Syntax

 _expression_. **VerticalAnchor**

 _expression_A variable that represents a  **CommandButton** object.


## Remarks

The default value is  **acVerticalAnchorTop**.
