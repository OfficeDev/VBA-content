
# Application.Forms Property (Access)

 **Last modified:** July 28, 2015

You can use the  **Forms** property to return a read-only reference to the ** [Forms](a41af7be-873c-ef8b-20cd-24b78a25b5ca.md)**collection and its related properties.

## Syntax

 _expression_. **Forms**

 _expression_A variable that represents an  **Application** object.


## Remarks

The properties of the  **Forms** collection in Visual Basic refer to forms that are currently open.
