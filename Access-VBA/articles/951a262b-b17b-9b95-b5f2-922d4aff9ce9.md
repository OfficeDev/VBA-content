
# Report.PSet Method (Access)

 **Last modified:** July 28, 2015

 **In this article**
 [Syntax](#sectionSection0)
 [Remarks](#sectionSection1)
 [Example](#sectionSection2)


The  **PSet** method sets a point on a ** [Report](6f77c1b4-a9ce-7caa-204c-fe0755c6f9df.md)**object to a specified color when the  **Print**event occurs.


## Syntax
<a name="sectionSection0"> </a>

 _expression_. **PSet**( **_flags_**,  **_X_**,  **_Y_**,  **_color_**)

 _expression_A variable that represents a  **Report** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|flags|Required| **Integer**| A keyword that indicates the coordinates are relative to the current graphics position given by the settings for the ** [CurrentX](3b5e7c50-ecb4-606a-6715-4d54ed912c45.md)**and  ** [CurrentY](040c0b5d-f7d6-2fa1-e34d-f69799f0b273.md)**properties of the object argument.|
|X|Required| **Single**|The horizontal coordinate of the point to set.|
|Y|Required| **Single**|The vertical coordinate of the point to set.|
|color|Required| **Long**|the RGB (red-green-blue) color to set the point to. If this argument is omitted, the value of the  **ForeColor**property is used. You can also use the  **RGB** function or **QBColor** function to specify the color.|

### Return Value

Nothing


## Remarks
<a name="sectionSection1"> </a>

The size of the point depends on the  ** [DrawWidth](1bda5387-9244-f150-2165-8dba1684ca25.md)**property setting. When the  **DrawWidth** property is set to 1, the **PSet** method sets a single pixel to the specified color. When the **DrawWidth** property is greater than 1, the point is centered on the specified coordinates.

The way the point is drawn depends on the settings of the  ** [DrawMode](773a3c7f-fb59-9614-3363-b417607fbe28.md)**and  ** [DrawStyle](0dd2afb9-d310-3637-6ed7-e66c9ad3460d.md)**properties.

When you apply the  **PSet** method, the **CurrentX** and **CurrentY** properties are set to the point specified by thex andy arguments.

To clear a single pixel with the  **PSet** method, specify the coordinates of the pixel and use &amp;HFFFFFF (white) as thecolor argument.


## Example
<a name="sectionSection2"> </a>

The following example uses the  **PSet** method to draw a line through the horizontal axis of a report.

To try this example in Microsoft Access, create a new report. Set the  **OnPrint** property of the Detail section to [Event Procedure]. Enter the following code in the report's module, then switch to Print Preview.




```
Sub Detail_Print(Cancel As Integer, PrintCount As Integer) 
 Dim sngMidPt As Single, intI As Integer 
 ' Set scale to pixels. 
 Me.ScaleMode = 3 
 ' Calculate midpoint. 
 sngMidPt = Me.ScaleHeight / 2 
 ' Loop to draw line down horizontal axis pixel by pixel. 
 For intI = 1 To Me.ScaleWidth 
 Me.PSet(intI, sngMidPt) 
 Next intI 
End Sub
```
