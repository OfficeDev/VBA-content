
# Series.Shadow Property (Excel)

 **Last modified:** July 28, 2015

Returns or sets a  **Boolean** value that determines if the object has a shadow.

## Syntax

 _expression_. **Shadow**

 _expression_A variable that represents a  **Series** object.
