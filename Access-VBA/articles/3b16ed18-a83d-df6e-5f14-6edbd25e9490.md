
# CommandButton.HorizontalAnchor Property (Access)

 **Last modified:** July 28, 2015

Gets or sets an  ** [AcHorizontalAnchor](2b9f0574-252d-7957-d25d-cb382d2cee73.md)** constant that indicates how the button is anchored horizontally within its layout. Read/write.

## Syntax

 _expression_. **HorizontalAnchor**

 _expression_A variable that represents a  **CommandButton** object.


## Remarks

The default value is  **acHorizontalAnchorLeft**.
