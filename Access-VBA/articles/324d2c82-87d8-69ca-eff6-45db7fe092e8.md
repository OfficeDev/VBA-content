
# TableStyleElements.Count Property (Excel)

 **Last modified:** July 28, 2015

Returns the number of objects in the collection. Read-only  **Long**.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **TableStyleElements** object.
