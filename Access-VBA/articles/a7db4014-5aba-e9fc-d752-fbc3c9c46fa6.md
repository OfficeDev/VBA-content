
# PivotFilters.Count Property (Excel)

 **Last modified:** July 28, 2015

Returns the number of items in the  **PivotFilters** collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **PivotFilters** object.
