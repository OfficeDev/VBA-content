
# IconSet.Count Property (Excel)

 **Last modified:** July 28, 2015

Returns a  **Long** value that specifies the number of icons in the icon set. Read-only.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents an  **IconSet** object.
