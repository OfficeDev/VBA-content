
# WorksheetFunction.Subtotal Method (Excel)

 **Last modified:** July 28, 2015

Creates subtotals. 

## Syntax

 _expression_. **Subtotal**( **_Arg1_**,  **_Arg2_**,  **_Arg3_**,  **_Arg4_**,  **_Arg5_**,  **_Arg6_**,  **_Arg7_**,  **_Arg8_**,  **_Arg9_**,  **_Arg10_**,  **_Arg11_**,  **_Arg12_**,  **_Arg13_**,  **_Arg14_**,  **_Arg15_**,  **_Arg16_**,  **_Arg17_**,  **_Arg18_**,  **_Arg19_**,  **_Arg20_**,  **_Arg21_**,  **_Arg22_**,  **_Arg23_**,  **_Arg24_**,  **_Arg25_**,  **_Arg26_**,  **_Arg27_**,  **_Arg28_**,  **_Arg29_**,  **_Arg30_**)

 _expression_A variable that represents a  **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Arg1|Required| **Double**|A number indicating the aggregation function to be used by the method.|
|Arg2|Required| **Range**|The first  ** [Range](b8207778-0dcc-4570-1234-f130532cc8cd.md)** object for which a subtotal is to be calculated.|
|Arg3-Arg30|Optional| **Variant**|Subsequent  ** [Range](b8207778-0dcc-4570-1234-f130532cc8cd.md)** objects for which a subtotal is to be calculated.|

### Return Value

A Double value that represents the subtotal.
