
# FormatCondition.Expression1 Property (Access)

 **Last modified:** July 28, 2015

You can use the  **Expression1** property to return the values of a conditional format within a ** [FormatCondition](a31deaae-b32d-c45b-b3b2-113a9e62cc7a.md)**object. Read-only  **String**.

## Syntax

 _expression_. **Expression1**

 _expression_A variable that represents a  **FormatCondition** object.


## Remarks

The  **Expression1** property returns a **Variant** value or expression associated with the first part of the conditional format.
