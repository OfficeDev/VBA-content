
# DisplayFormat Members (Excel)
Represents the display settings for an associated  ** [Range](b8207778-0dcc-4570-1234-f130532cc8cd.md)** object. Read only.

 **Last modified:** July 28, 2015


## Properties



|**Name**|**Description**|

Read-only.|
