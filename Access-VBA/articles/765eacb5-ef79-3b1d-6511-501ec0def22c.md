
# Form.SplitFormDatasheet Property (Access)

 **Last modified:** July 28, 2015

Gets or sets whether the user can edit records in the datasheet when a form is displayed in Split Form view. Read/write  ** [AcSplitFormDatasheet](cb90ac7e-af5e-7a99-de3a-283601a3e180.md)**.

## Syntax

 _expression_. **SplitFormDatasheet**

 _expression_A variable that represents a  **Form** object.
