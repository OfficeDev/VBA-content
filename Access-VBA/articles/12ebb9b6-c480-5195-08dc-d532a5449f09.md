
# Point.Copy Method (Excel)

 **Last modified:** July 28, 2015

If the point has a picture fill, then this method copies the picture to the Clipboard.

## Syntax

 _expression_. **Copy**

 _expression_A variable that represents a  **Point** object.


### Return Value

Variant
