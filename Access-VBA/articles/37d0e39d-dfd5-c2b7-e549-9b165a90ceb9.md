
# Report.FilterOnLoad Property (Access)

 **Last modified:** July 28, 2015

Gets or sets whether the filter specified by the  ** [Filter](ce57e05d-c3a8-012a-205e-7dfb2e4dd78d.md)** property is applied when the report is loaded. Read/write **Boolean**.

## Syntax

 _expression_. **FilterOnLoad**

 _expression_A variable that represents a  **Report** object.
