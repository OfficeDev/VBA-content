
# SparkPoints.Lowpoint Property (Excel)

 **Last modified:** July 28, 2015

Returns a  ** [SparkColor](3de82c5c-eb0a-ab39-64a8-00f4c005c6af.md)** object that represents the color and visibility of the marker for the lowest point of data on a sparkline. Read-only

## Syntax

 _expression_. **Lowpoint**

 _expression_A variable that represents a  ** [SparkPoints](382bf292-7824-179f-e254-1b72dfb557b2.md)** object.


### Return Value

 **SparkColor**


## Remarks

 The **Lowpoint** property corresponds to the setting of the **Low Point** check box in the **Show** section, and of the **Low Point** item on the **Marker Color** drop-down list in the **Style** section of the ** Sparkline Tools Design** tab of the ribbon.
