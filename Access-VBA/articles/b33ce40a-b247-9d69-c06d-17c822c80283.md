
# CommandButton.PictureCaptionArrangement Property (Access)

 **Last modified:** July 28, 2015

Gets or sets how the specified button's caption is arranged in relation to its picture. Read/write  ** [AcPictureCaptionArrangement](e77a758f-4139-343f-9f1c-5e29db3fe5a3.md)**.

## Syntax

 _expression_. **PictureCaptionArrangement**

 _expression_A variable that represents a  **CommandButton** object.
