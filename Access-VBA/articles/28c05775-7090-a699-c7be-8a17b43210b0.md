
# Report.OrderByOnLoad Property (Access)

 **Last modified:** July 28, 2015

Gets or sets whether the sorting specified by the  ** [OrderBy](1939157c-12ad-2e58-bf4c-22c04a6c4366.md)** property is applied when the report is loaded. Read/write **Boolean**.

## Syntax

 _expression_. **OrderByOnLoad**

 _expression_A variable that represents a  **Report** object.
