
# CheckBox.Layout Property (Access)

 **Last modified:** July 28, 2015

Returns the type of layout for the specified check box. Read-only  ** [AcLayoutType](ee963ed0-9293-8ad8-5694-4b93a5e4d89a.md)**.

## Syntax

 _expression_. **Layout**

 _expression_A variable that represents a  **CheckBox** object.
