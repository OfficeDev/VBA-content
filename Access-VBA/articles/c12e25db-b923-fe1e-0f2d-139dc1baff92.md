
# SlicerItem.Name Property (Excel)

 **Last modified:** July 28, 2015

Returns the name of the slicer item. Read-only.

## Syntax

 _expression_. **Name**

 _expression_A variable that represents a  ** [SlicerItem](cb93cd82-fc3a-f6b7-ae64-db6312db649d.md)** object.


### Return Value

String
