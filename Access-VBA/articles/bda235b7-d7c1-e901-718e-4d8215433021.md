
# Chart.Tab Property (Excel)

 **Last modified:** July 28, 2015

Returns a  ** [Tab](c6555e96-b96e-54d8-b8c6-5ab13c256d97.md)** object for a chart.

## Syntax

 _expression_. **Tab**

 _expression_A variable that represents a  **Chart** object.
