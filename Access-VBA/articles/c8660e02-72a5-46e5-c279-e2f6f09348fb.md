
# ChartArea.Select Method (Excel)

 **Last modified:** July 28, 2015

Selects the object.

## Syntax

 _expression_. **Select**

 _expression_A variable that represents a  **ChartArea** object.


### Return Value

A Variant value that represents the selected object.
